### Dinamização de dados

Após o estudo anterior, preparamos os dados e os transformamos no formato que desejamos. Em seguida vem a etapa mais importante, a dinamização de dados. Quando obtemos uma grande quantidade de dados, como podemos ler rapidamente informações valiosas deles, transformar dados complicados em gráficos estatísticos fáceis de ler e, então, gerar insights de negócios com base nisso? Este é exatamente o problema central que a análise de dados precisa resolver.

#### Obtendo estatísticas descritivas

Primeiro, podemos obter estatísticas descritivas dos dados. Através da estatística descritiva, podemos compreender a tendência central e a tendência de dispersão dos dados.

Por exemplo, suponha que temos a tabela de pontuação dos alunos abaixo.

```python
scores = np.random.randint(50, 101, (5, 3))
names = ('Guan Yu', 'Zhang Fei', 'Zhao Yun', 'Ma Chao', 'Huang Zhong')
courses = ('Chinese', 'Math', 'English')
df = pd.DataFrame(data=scores, columns=courses, index=names)
df
```

Saída:

```
             Chinese  Math  English
Guan Yu           96    72       73
Zhang Fei         72    70       97
Zhao Yun          74    51       79
Ma Chao          100    54       54
Huang Zhong       89   100       88
```

Podemos usar métodos como `mean`, `max`, `min`, `std` e `var` em um objeto `DataFrame` para obter a pontuação média, pontuação mais alta, pontuação mais baixa, desvio padrão, variância e outras informações para cada aluno ou cada curso. Também podemos usar `describe` diretamente para obter estatísticas descritivas.

Calcule a pontuação média de cada curso:

```python
df.mean()
```

Saída:

```
Chinese    86.2
Math       69.4
English    78.2
dtype: float64
```

Calcule a pontuação média de cada aluno:

```python
df.mean(axis=1)
```

Saída:

```
Guan Yu        80.333333
Zhang Fei      79.666667
Zhao Yun       68.000000
Ma Chao        69.333333
Huang Zhong    92.333333
dtype: float64
```

Calcule a variância de cada curso:

```python
df.var()
```

> **Nota**: Pela variância, podemos ver que Math tem a maior flutuação, portanto sua polarização pode ser mais séria.

Saída:

```
Chinese    161.2
Math       379.8
English    265.7
dtype: float64
```

Obtenha estatísticas descritivas para cada curso:

```python
df.describe()
```

Saída:

```
          Chinese       Math    English
count    5.000000   5.000000   5.000000
mean    86.200000  69.400000  78.200000
std     12.696456  19.488458  16.300307
min     72.000000  51.000000  54.000000
25%     74.000000  54.000000  73.000000
50%     89.000000  70.000000  79.000000
75%     96.000000  72.000000  88.000000
max    100.000000 100.000000  97.000000
```

#### Classificando e obtendo valores principais

Se precisarmos classificar os dados, podemos usar o método `sort_values` de `DataFrame`. O parâmetro `by` especifica quais colunas são usadas para classificação e o parâmetro `ascending` especifica a ordem crescente ou decrescente. Por exemplo, o código a seguir classifica a tabela de alunos em ordem decrescente pela pontuação em chinês.

```python
df.sort_values(by='Chinese', ascending=False)
```

Saída:

```
             Chinese  Math  English
Ma Chao          100    54       54
Guan Yu           96    72       73
Huang Zhong       89   100       88
Zhao Yun          74    51       79
Zhang Fei         72    70       97
```

Se a quantidade de dados em um `DataFrame` for muito grande, a classificação poderá ser uma operação demorada. Às vezes, precisamos apenas dos N valores superiores ou N inferiores e, nesse caso, não há necessidade de classificar a tabela inteira. Em vez disso, podemos encontrar diretamente os dados Top-N por meio de uma estrutura heap. Os métodos `nlargest` e `nsmallest` de `DataFrame` suportam operações Top-N.

Encontre os 3 melhores alunos em chinês:

```python
df.nlargest(3, 'Chinese')
```

Saída:

```
             Chinese  Math  English
Ma Chao          100    54       54
Guan Yu           96    72       73
Huang Zhong       89   100       88
```

Encontre os 3 alunos com as notas mais baixas em matemática:

```python
df.nsmallest(3, 'Math')
```

Saída:

```
             Chinese  Math  English
Zhao Yun          74    51       79
Ma Chao          100    54       54
Zhang Fei         72    70       97
```

#### Agregação agrupada

Vamos primeiro ler os dados de vendas de 2020 do arquivo Excel usado anteriormente e, em seguida, demonstrar como fazer agregação agrupada.

```python
df = pd.read_excel('data/2020-sales-data.xlsx')
df.head()
```

Saída:

```
  Sales Date Sales Area Sales Channel Sales Order   Brand  Price  Sales Quantity
0 2020-01-01     Shanghai     Pinduoduo 182894-455  BPM      99              83
1 2020-01-01     Shanghai         Douyin 205635-402  BPM     219              29
2 2020-01-01     Shanghai          Tmall 205654-021  BPM     169              85
3 2020-01-01     Shanghai          Tmall 205654-519  BPM     169              14
4 2020-01-01     Shanghai          Tmall 377781-010  PPX     249              61
```

Se quisermos contar o valor total das vendas de cada área de vendas, podemos primeiro calcular o valor das vendas de `price` e `sales quantity` e adicionar uma nova coluna ao `DataFrame`.

```python
df['Sales Amount'] = df['Price'] * df['Sales Quantity']
df.head()
```

Saída:

```
Head of the table with the new `Sales Amount` column added.
```

Em seguida, agrupamos os dados de acordo com a coluna `Sales Area` usando o método `groupby` de um `DataFrame`. Após o agrupamento, pegamos a coluna `Sales Amount` e somamos dentro de cada grupo.

```python
df.groupby('Sales Area')['Sales Amount'].sum()
```

Saída:

```
Total sales amount grouped by sales area.
```

Se quisermos contar o valor total das vendas de cada mês, podemos usar a data da venda como base de agrupamento, mas aqui primeiro precisamos transformar a data da venda em valores do mês.

```python
df.groupby(df['Sales Date'].dt.month)['Sales Amount'].sum()
```

Saída:

```
Total sales amount grouped by month.
```

Agora vamos aumentar a dificuldade e contar o valor total das vendas de cada área de vendas em cada mês. Na verdade, o primeiro argumento de `groupby` pode ser uma lista, e a lista pode especificar múltiplas bases de agrupamento.

```python
df.groupby(['Sales Area', df['Sales Date'].dt.month])['Sales Amount'].sum()
```

Saída:

```
Grouped totals by sales area and month in a long narrow table.
```

Se quisermos contar o valor total de vendas de cada área e também o valor máximo e mínimo de um único pedido em cada área, podemos usar o método `agg` em um `DataFrame` ou `Series` e especificar múltiplas funções de agregação.

```python
df.groupby('Sales Area')['Sales Amount'].agg(['sum', 'max', 'min'])
```

Saída:

```
Grouped sum, max, and min of sales amount for each sales area.
```

Se quisermos customizar os nomes das colunas agregadas, podemos fazer assim:

```python
df.groupby('Sales Area')['Sales Amount'].agg(Total_Sales='sum', Highest_Single_Order='max', Lowest_Single_Order='min')
```

Saída:

```
Grouped result with custom column names.
```

Se precisarmos usar diferentes funções de agregação em múltiplas colunas, por exemplo “contar a soma do valor das vendas e o máximo e mínimo da quantidade de vendas para cada área de vendas”, podemos escrever:

```python
df.groupby('Sales Area')[['Sales Amount', 'Sales Quantity']].agg({
    'Sales Amount': 'sum', 'Sales Quantity': ['max', 'min']
})
```

Saída:

```
Grouped aggregation result using different functions on multiple columns.
```

#### Tabelas dinâmicas e tabelas cruzadas

No exemplo acima, “contar o valor total das vendas de cada área de vendas em cada mês” produz um resultado muito longo. No trabalho real, geralmente chamamos uma tabela com muitas linhas e poucas colunas de "tabela estreita". Se não quisermos uma tabela tão estreita, podemos usar o método `pivot_table` de um `DataFrame` ou a função `pivot_table` para gerar uma tabela dinâmica. A essência de uma tabela dinâmica é a agregação agrupada. Significa **contar a coluna B de acordo com a coluna A**. Se você já usou o Excel, esse conceito deve ser familiar.

Por exemplo, se quisermos "contar o valor total de vendas de cada área de vendas", então `Sales Area` é a coluna A e `Sales Amount` é a coluna B. Na função `pivot_table`, eles correspondem aos parâmetros `index` e `values`.

```python
pd.pivot_table(df, index='Sales Area', values='Sales Amount', aggfunc='sum')
```

Saída:

```
Pivot table of total sales amount by sales area.
```

> **Aviso**: O resultado acima é um pouco diferente do resultado do uso de `groupby`. Depois de `groupby`, se agregarmos uma única coluna, o resultado será `Series`, enquanto o resultado acima será `DataFrame`.

Se quisermos contar o valor total de vendas de cada área de vendas em cada mês, também podemos utilizar `pivot_table`.

```python
df['Month'] = df['Sales Date'].dt.month
pd.pivot_table(df, index=['Sales Area', 'Month'], values='Sales Amount', aggfunc='sum')
```

Saída:

```
Long pivot table by sales area and month.
```

O resultado acima ainda é uma tabela longa e estreita. Se quisermos uma "tabela ampla" com menos linhas e mais colunas, podemos mover a coluna em `index` para o parâmetro `columns`.

```python
pd.pivot_table(df, index='Sales Area', columns='Month', values='Sales Amount', aggfunc='sum', fill_value=0)
```

Saída:

```
Wide pivot table by sales area and month, with null values filled as 0.
```

> **Observação**: `fill_value=0` em `pivot_table` tratará valores nulos como `0`.

Ao usar `pivot_table`, também podemos adicionar `margins` e `margins_name` para gerar totais resumidos para o resultado agrupado.

```python
pd.pivot_table(df, index='Sales Area', columns='Month', values='Sales Amount', aggfunc='sum', fill_value=0, margins=True, margins_name='Total')
```

Saída:

```
Wide pivot table with totals added.
```

Uma tabela cruzada é um tipo especial de tabela dinâmica. Não é necessário que primeiro construamos um objeto `DataFrame`. Em vez disso, ele usa matrizes ou objetos `Series` diretamente para especificar dois ou mais fatores e calcula um resultado estatístico. Por exemplo, se quisermos contar o valor total das vendas de cada área de vendas, também podemos fazê-lo da seguinte forma.

Primeiro prepare três grupos de dados:

```python
sales_area, sales_month, sales_amount = df['Sales Area'], df['Month'], df['Sales Amount']
```

Use a função `crosstab` para gerar uma tabela cruzada:

```python
pd.crosstab(index=sales_area, columns=sales_month, values=sales_amount, aggfunc='sum').fillna(0).astype('i8')
```

> **Observação**: No código acima, `fillna` é usado para tratar valores nulos como 0 e `astype` é usado para converter o tipo de dados em números inteiros.

Saída:

```
Cross table of sales amount by sales area and sales month.
```

### Apresentação de dados

Uma imagem vale mais que mil palavras. O resultado da nossa dinamização de dados deve finalmente ser apresentado em gráficos, porque os gráficos têm um poder expressivo muito forte e nos permitem compreender rapidamente o valor escondido nos dados. Assim como `Series`, `DataFrame` também fornece um método `plot` para suportar plotagem. Nos bastidores, a renderização do gráfico ainda é feita por meio do matplotlib.

Por exemplo, se quisermos utilizar um gráfico de barras para comparar o valor total de vendas de cada área de vendas, podemos utilizar diretamente o método `plot` na tabela dinâmica. Primeiro importe `matplotlib.pyplot` e modifique os parâmetros de plotagem para que o chinês possa ser exibido corretamente, se necessário.

```python
import matplotlib.pyplot as plt

plt.rcParams['font.sans-serif'] = 'FZJKai-Z03S'
```

Use um comando mágico para configurar a saída de gráficos vetoriais:

```python
%config InlineBackend.figure_format = 'svg'
```

Desenhe um gráfico de barras do valor total de vendas de cada área de vendas:

```python
temp = pd.pivot_table(df, index='Sales Area', values='Sales Amount', aggfunc='sum')
temp.plot(figsize=(8, 4), kind='bar')
plt.xticks(rotation=0)
plt.show()
```

Se quisermos desenhar um gráfico de pizza, podemos alterar o parâmetro `kind` de `plot` para `pie` e, em seguida, usar os parâmetros do gráfico de pizza para personalizar o gráfico.

```python
temp.sort_values(by='Sales Amount', ascending=False).plot(
    figsize=(6, 6),
    kind='pie',
    y='Sales Amount',
    ylabel='',
    autopct='%.2f%%',
    pctdistance=0.8,
    wedgeprops=dict(linewidth=1, width=0.35),
    legend=False
)
plt.show()
```
