## Reading and Writing Excel Files in Python, Part 1

### An Introduction to Excel

Excel is a spreadsheet software developed by Microsoft for Windows and macOS. Because of its intuitive interface, excellent calculation ability and chart tools, together with successful marketing, Excel has always been the most popular data processing software on personal computers. Of course, Excel also has many competing products, such as Google Sheets, LibreOffice Calc, and Numbers. These competing products can basically also be compatible with Excel, at least they can read and write newer Excel files. Of course, these are not the focus of our discussion. Mastering how to operate Excel files with Python programs can make daily office automation work easier and more pleasant. Also, in many commercial projects, importing and exporting Excel files are very common functions.

Python needs third-party library support to work with Excel. If you want compatibility with Excel versions before 2007, in other words `.xls` files, you can use the third-party libraries `xlrd` and `xlwt`, where the former reads Excel files and the latter writes them. If you are using newer Excel files, namely `.xlsx`, you can use the `openpyxl` library. Of course, that library can do more than just handle Excel, because it can also operate on other spreadsheet files based on Office Open XML.

In this lesson, we first cover Excel file operations based on `xlwt` and `xlrd`. You can install those two libraries together with the helper module `xlutils` using the command below.


### Reading Excel Files

For example, if there is an Excel file named `Alibaba 2020 Equity Data.xls` in the current folder, if we want to read and display the content of this file, we can do it with the code shown below.

```python
import xlrd

# Use xlrd.open_workbook to open the specified Excel file and get a Book object.
wb = xlrd.open_workbook('Alibaba 2020 Equity Data.xls')
# Get the names of all worksheets.
sheetnames = wb.sheet_names()
print(sheetnames)
# Get a Sheet object by worksheet name.
sheet = wb.sheet_by_name(sheetnames[0])
# Get the row count and column count of the worksheet.
print(sheet.nrows, sheet.ncols)
for row in range(sheet.nrows):
    for col in range(sheet.ncols):
        # Get the specified Cell object with sheet.cell.
        # Then get the value stored in the cell.
        value = sheet.cell(row, col).value
        # Format all rows other than the header row.
        if row > 0:
            # Convert the xldate value in the first column to a tuple,
            # then format it as year-month-day.
            if col == 0:
                # The second argument to xldate_as_tuple can only be 0 or 1.
                # 0 means dates are based on 1900-01-01, and 1 means 1904-01-01.
                value = xlrd.xldate_as_tuple(value, 0)
                value = f'{value[0]}Year{value[1]:>02d}moon{value[2]:>02d}day'
            # Format number values in the remaining columns with two decimal places.
            else:
                value = f'{value:.2f}'
        print(value, end='\t')
    print()
# Get the data type of the last cell.
# 0 - empty, 1 - string, 2 - number, 3 - date, 4 - boolean, 5 - error
last_cell_type = sheet.cell_type(sheet.nrows - 1, sheet.ncols - 1)
print(last_cell_type)
# Get the values in the first row as a list.
print(sheet.row_values(0))
# Get a range of columns from a specified row as a list.
# The first argument is the row index, the second and third are the start
# and end column indices, with the end index excluded.
print(sheet.row_slice(3, 0, 5))
```

### Writing Excel Files

To write an Excel file, we can create a workbook object through the `Workbook` class of the `xlwt` module, add a worksheet through the `add_sheet` method of the workbook object, write data into the specified cell through the `write` method of the worksheet object, and finally write the workbook into the specified file or memory through the `save` method of the workbook object. The code below completes writing the exam scores of 5 students in 3 courses into an Excel file.

```python
import random

import xlwt

student_names = ['Guan Yu', 'Zhang Fei', 'Zhao Yun', 'Ma Chao', 'Huang Zhong']
scores = [[random.randrange(50, 101) for _ in range(3)] for _ in range(5)]
# Create a Workbook object.
wb = xlwt.Workbook()
# Create a Worksheet object.
sheet = wb.add_sheet('Grade 1 Class 2')
# Add header cells.
titles = ('name', 'Chinese', 'Mathematics', 'English')
for index, title in enumerate(titles):
    sheet.write(0, index, title)
# Write student names and scores into the worksheet.
for row in range(len(scores)):
    sheet.write(row + 1, 0, student_names[row])
    for col in range(len(scores[row])):
        sheet.write(row + 1, col + 1, scores[row][col])
# Save the Excel workbook.
wb.save('Exam Score Sheet.xls')
```

### Adjusting Cell Styles

When writing an Excel file, we can also set styles for cells, mainly including the font, alignment, border, and background. `xlwt` provides corresponding classes for all of these. To set a cell style, we first create an `XFStyle` object, and then configure its font, alignment, border, and other properties. For example, if we want to change the header cells to have a yellow background, we can do it like this.

```python
header_style = xlwt.XFStyle()
pattern = xlwt.Pattern()
pattern.pattern = xlwt.Pattern.SOLID_PATTERN
# 0 - black, 1 - white, 2 - red, 3 - green, 4 - blue, 5 - yellow, 6 - pink, 7 - cyan
pattern.pattern_fore_colour = 5
header_style.pattern = pattern
titles = ('name', 'Chinese', 'Mathematics', 'English')
for index, title in enumerate(titles):
    sheet.write(0, index, title, header_style)
```

If we want to assign a specific font to the header, we can use the `Font` class and add the code below.

```python
font = xlwt.Font()
# Font name
font.name = 'Chinese Italic'
# Font size (20 is the base unit, 18 means 18px)
font.height = 20 * 18
# Use bold or not
font.bold = True
# Use italic or not
font.italic = False
# Font color
font.colour_index = 1
header_style.font = font
```

> **Note**: The font name assigned above through `font.name` should be a font available on your local system.

If we want the header to be vertically and horizontally centered, we can use the following code.

```python
align = xlwt.Alignment()
# Vertical alignment
align.vert = xlwt.Alignment.VERT_CENTER
# Horizontal alignment
align.horz = xlwt.Alignment.HORZ_CENTER
header_style.alignment = align
```

If we want the header to have a yellow dashed border, we can configure it like this.

```python
borders = xlwt.Borders()
props = (
    ('top', 'top_colour'), ('right', 'right_colour'),
    ('bottom', 'bottom_colour'), ('left', 'left_colour')
)
# Use a loop to configure style and color for borders on all four sides.
for position, color in props:
    # Use setattr to assign attribute values dynamically.
    setattr(borders, position, xlwt.Borders.DASHED)
    setattr(borders, color, 5)
header_style.borders = borders
```

If we want to adjust the column width and the header row height, we can do it as shown below.

```python
# Set the row height to 40px.
sheet.row(0).set_style(xlwt.easyxf(f'font:height {20 * 40}'))
titles = ('name', 'Chinese', 'Mathematics', 'English')
for index, title in enumerate(titles):
    # Set the column width to 200px.
    sheet.col(index).width = 20 * 200
    # Set the cell value and style.
    sheet.write(0, index, title, header_style)
```

### Formula Calculation

For the `Alibaba 2020 Equity Data.xls` file that we opened earlier, if we want to calculate the yearly average of the closing price, the `Close` field, and the yearly total of the trading volume, the `Volume` field, we can do it with Excel formulas. We can first use `xlrd` to read the Excel file, then use the `copy` function provided by `xlutils` to convert the workbook into a writable `Workbook` object, and when calling `write`, we can write an `xlwt.Formula` object into a cell.

The code for formula calculation is shown below.

```python
import xlrd
import xlwt
from xlutils.copy import copy

wb_for_read = xlrd.open_workbook('Alibaba 2020 Equity Data.xls')
sheet1 = wb_for_read.sheet_by_index(0)
nrows, ncols = sheet1.nrows, sheet1.ncols
wb_for_write = copy(wb_for_read)
sheet2 = wb_for_write.get_sheet(0)
sheet2.write(nrows, 4, xlwt.Formula(f'average(E2:E{nrows})'))
sheet2.write(nrows, 6, xlwt.Formula(f'sum(G2:G{nrows})'))
wb_for_write.save('Alibaba 2020 Stock Data Summary.xls')
```
