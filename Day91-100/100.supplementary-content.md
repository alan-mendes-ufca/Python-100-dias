## Conteúdo Suplementar

Estas são partes do conteúdo que não foram organizadas anteriormente e estão planejadas para serem adicionadas posteriormente:

1. Para recursos relacionados a **entrevistas**, acesse o projeto [Bíblia da Entrevista Python](https://github.com/jackfrued/Python-Interview-Bible).
   - Bíblia da Entrevista Python
   - Bíblia de entrevista de análise de negócios
   - Bíblia da entrevista SQL com analista de dados
   - Bíblia de entrevista de aprendizado de máquina
2. Para recursos relacionados a **fundamentos de matemática**, acesse o projeto [Matemática para aprendizado de máquina](https://github.com/jackfrued/Math_for_ML).
   - Álgebra linear
   - Teoria da probabilidade e estatística
   - Matemática avançada
   - Teoria da informação
3. Para conteúdo relacionado a **aprendizado profundo**, acesse o projeto [Aprendizado profundo não é nada](https://github.com/jackfrued/Deep-Learning-Is-Nothing).
