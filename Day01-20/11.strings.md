### Definindo strings

Uma **string** é uma sequência finita composta de zero ou mais caracteres:

$$ s = a_1a_2 \cdots a_n\,\,\,\,\,(0 \le n \le \infty) $$

Em um programa Python, se colocarmos um ou mais caracteres entre aspas simples ou duplas, podemos representar uma string. Os caracteres em uma string podem ser símbolos especiais, letras inglesas, caracteres chineses, hiragana ou katakana japoneses, letras gregas, caracteres Emoji e assim por diante.

```python
s1 = 'hello, world!'
s2 = "Greetings, world!❤️"
s3 = '''hello,
wonderful
world!'''
print(s1)
print(s2)
print(s3)
```

#### Personagens de fuga

Podemos usar `\` (barra invertida) em uma string para significar escape. Isso significa que o caractere após `\` não mantém mais seu significado original. Por exemplo, `\n` não significa o caractere `\` e o caractere `n`; significa uma nova linha. `\t` não significa o caractere `\` e o caractere `t`; isso significa uma guia. Portanto, se a própria string contiver caracteres especiais como `'`, `"` e `\`, devemos escapá-los com `\`. Por exemplo, se quisermos gerar uma string que contenha aspas simples ou barras invertidas, precisamos escrevê-la assim.

```python
s1 = '\'hello, world!\''
s2 = '\\hello, world!\\'
print(s1)
print(s2)
```

#### Cordas brutas

Python também possui um tipo de string que começa com `r` ou `R`. Esse tipo de string é chamado de string bruta. Isso significa que cada caractere da string mantém seu significado original e não há caracteres de escape. Por exemplo, na string `'hello\n'`, `\n` significa uma nova linha; mas em `r'hello\n'`, `\n` é apenas o caractere `\` e o caractere `n`. Execute o código abaixo e veja o que ele imprime.

```python
s1 = '\it \is \time \to \read \now'
s2 = r'\it \is \time \to \read \now'
print(s1)
print(s2)
```

> **Observação**: Na variável `s1` acima, `\t`, `\r` e `\n` são todos caracteres de escape. `\t` é uma guia, `\n` é uma nova linha e `\r` é um retorno de carro, que é como mover a saída de volta para o início da linha. Compare as saídas das duas funções `print` e veja qual é a diferença.

#### Notação de caracteres especiais

Em Python, depois de `\` também podemos colocar um número octal ou hexadecimal para representar um caractere. Por exemplo, `\141` e `\x61` representam a letra minúscula `a`. A primeira é a notação octal e a última é a notação hexadecimal. Outra forma de representar um caractere é colocar um código de caractere Unicode após `\u`. Por exemplo, `\u9a86\u660a` significa o nome chinês "Luo Hao". Execute o código abaixo e veja o que ele imprime.

```python
s1 = '\141\142\143\x61\x62\x63'
s2 = '\u9a86\u660a'
print(s1)
print(s2)
```

### Operações de cordas

Python fornece muitos operadores para strings, e muitos deles funcionam como operadores para listas. Por exemplo, podemos usar `+` para unir strings, `*` para repetir o conteúdo de uma string, `in` e `not in` para determinar se uma string contém outra string, e `[]` e `[:]` para pegar extrair um caractere ou vários caracteres de uma string.

#### Concatenação e Repetição

O exemplo abaixo mostra como usar `+` e `*` para unir e repetir strings.

```python
s1 = 'hello' + ', ' + 'world'
print(s1)    # hello, world
s2 = '!' * 3
print(s2)    # !!!
s1 += s2
print(s1)    # hello, world!!!
s1 *= 2
print(s1)    # hello, world!!!hello, world!!!
```

Usar `*` para repetir uma string é um recurso muito interessante. Em muitas linguagens de programação, se você deseja representar uma string com 10 caracteres `a`, você só pode escrever `'aaaaaaaaaa'`; mas em Python, você pode escrever `'a' * 10`. Você pode achar que `'aaaaaaaaaa'` não é inconveniente, mas pense no que acontece se o caractere `a` precisar ser repetido 100 ou 1000 vezes.

#### Comparação

Para duas variáveis ​​de string, podemos usar diretamente operadores de comparação para verificar se as duas strings são iguais ou para comparar seu tamanho. Deve ser explicado que as strings também existem em formato binário na memória do computador, portanto, comparar o tamanho das strings realmente significa comparar o valor do código de cada caractere. Por exemplo, o código de `A` é `65`, enquanto o código de `a` é `97`, então o resultado de `'A' < 'a'` é realmente o resultado de `65 < 97`, e isso é claramente `True`. Para `'boy' < 'bad'`, como os primeiros caracteres são ambos `'b'`, ainda não podemos dizer sua ordem, então a comparação real é entre os segundos caracteres. Claramente, o resultado de `'o' < 'a'` é `False`, então `'boy' < 'bad'` também é `False`. Se você não souber o valor do código de um caractere, poderá usar a função `ord` para obtê-lo. Mencionamos essa função antes. Por exemplo, o valor de `ord('A')` é `65` e o valor de `ord('Hao')` é `26122`. Observe o código abaixo com atenção.

```python
s1 = 'a whole new world'
s2 = 'hello world'
print(s1 == s2)             # False
print(s1 < s2)              # True
print(s1 == 'hello world')  # False
print(s2 == 'hello world')  # True
print(s2 != 'Hello world')  # True
s3 = 'Luo Hao'
print(ord('Luo'))            # 39558
print(ord('Hao'))            # 26122
s4 = 'Wang Dahuang'
print(ord('king'))            # 29579
print(ord('big'))            # 22823
print(ord('hammer'))            # 38180
print(s3 >= s4)             # True
print(s3 != s4)             # True
```

#### Associação

Em Python, podemos usar `in` e `not in` para julgar se uma string contém outro caractere ou string. Assim como nas listas, `in` e `not in` são chamados de operadores de associação e produzem o valor booleano `True` ou `False`, conforme mostrado abaixo.

```python
s1 = 'hello, world'
s2 = 'goodbye, world'
print('wo' in s1)      # True
print('wo' not in s2)  # False
print(s2 in s1)        # False
```

#### Comprimento

Obter o comprimento de uma string é o mesmo que obter o número de elementos em uma lista: use a função integrada `len`, conforme mostrado abaixo.

```python
s = 'hello, world'
print(len(s))                 # 12
print(len('goodbye, world'))  # 14
```

#### Indexação e fatiamento

Strings suportam indexação e fatiamento assim como listas e tuplas, porque strings também são sequências ordenadas. Mas há uma coisa a ser observada: como **strings são imutáveis**, **você não pode usar indexação para modificar caracteres em uma string**.

```python
s = 'abc123456'
n = len(s)
print(s[0], s[-n])    # a a
print(s[n-1], s[-1])  # 6 6
print(s[2], s[-7])    # c c
print(s[5], s[-4])    # 3 3
print(s[2:5])         # c12
print(s[-7:-4])       # c12
print(s[2:])          # c123456
print(s[:2])          # ab
print(s[::2])         # ac246
print(s[::-1])        # 654321cba
```

Mais um lembrete: ao fazer a indexação, se o índice sair do intervalo, o Python gera `IndexError`, e a mensagem de erro é `string index out of range`.

### Iterando sobre personagens

Se quisermos percorrer cada caractere de uma string, podemos usar um loop `for-in`. Existem duas maneiras comuns.

Método 1:

```python
s = 'hello'
for i in range(len(s)):
    print(s[i])
```

Método 2:

```python
s = 'hello'
for elem in s:
    print(elem)
```

### Métodos de string

Em Python, podemos trabalhar com strings através de métodos que pertencem ao próprio tipo de string. Suponha que tenhamos uma string chamada `foo`, e a string tenha um método chamado `bar`, então a sintaxe para usar o método de string é `foo.bar()`. Esta é a mesma sintaxe que usamos antes para métodos de lista.

#### Conversão de Caso

O código abaixo mostra métodos relacionados à conversão de maiúsculas e minúsculas de string.

```python
s1 = 'hello, world!'
# Capitalize the first letter of the string.
print(s1.capitalize())  # Hello, world!
# Capitalize the first letter of every word.
print(s1.title())       # Hello, World!
# Convert the string to uppercase.
print(s1.upper())       # HELLO, WORLD!
s2 = 'GOODBYE'
# Convert the string to lowercase.
print(s2.lower())       # goodbye
# Check the values of s1 and s2.
print(s1)
print(s2)
```

> **Nota**: Como as strings são imutáveis, esses métodos retornam novas strings em vez de alterar a original. Portanto, quando verificamos os valores de `s1` e `s2` no final, seus valores não foram alterados.

#### Procurando

Se quisermos pesquisar uma string dentro de outra string da esquerda para a direita, podemos usar o método `find` ou `index`. Ao usar `find` e `index`, também podemos especificar o intervalo de pesquisa por meio dos argumentos do método, o que significa que a pesquisa não precisa começar no índice `0`.

```python
s = 'hello, world!'
print(s.find('or'))      # 8
print(s.find('or', 9))   # -1
print(s.find('of'))      # -1
print(s.index('or'))     # 8
print(s.index('or', 9))  # ValueError: substring not found
```

> **Observação**: Se `find` não conseguir encontrar a string especificada, ele retornará `-1`; se `index` não puder encontrar a sequência especificada, ele gerará `ValueError`.

`find` e `index` também possuem versões de pesquisa reversa, que pesquisam da direita para a esquerda. Eles são `rfind` e `rindex`, conforme mostrado abaixo.

```python
s = 'hello world!'
print(s.find('o'))       # 4
print(s.rfind('o'))      # 7
print(s.rindex('o'))     # 7
```

#### Verificações de propriedade

Podemos usar `startswith` e `endswith` para julgar se uma string começa ou termina com outra string. Também podemos usar métodos que começam com `is` para julgar as características de uma string. Todos esses métodos retornam valores booleanos, conforme mostrado abaixo.

```python
s1 = 'hello, world!'
print(s1.startswith('He'))   # False
print(s1.startswith('hel'))  # True
print(s1.endswith('!'))      # True
s2 = 'abc123456'
print(s2.isdigit())  # False
print(s2.isalpha())  # False
print(s2.isalnum())  # True
```

> **Nota**: `isdigit` é usado para julgar se uma string é composta apenas de dígitos. `isalpha` é usado para julgar se uma string é composta apenas por letras. As letras aqui significam letras Unicode, mas não incluem caracteres Emoji. `isalnum` é usado para julgar se uma string é composta de letras e dígitos.

#### Formatação

Em Python, as strings podem ser centralizadas, alinhadas à esquerda ou à direita por meio dos métodos `center`, `ljust` e `rjust`. Se quisermos preencher o lado esquerdo de uma string com zeros, também podemos usar o método `zfill`.

```python
s = 'hello, world'
print(s.center(20, '*'))  # ****hello, world****
print(s.rjust(20))        #         hello, world
print(s.ljust(20, '~'))   # hello, world~~~~~~~~
print('33'.zfill(5))      # 00033
print('-33'.zfill(5))     # -0033
```

#### Formatação

Já vimos o seguinte estilo de formatação ao usar a função `print`.

```python
a = 321
b = 123
print('%d * %d = %d' % (a, b, a * b))
```

Claro, também podemos usar o método string `format` para formatar uma string, conforme mostrado abaixo.

```python
a = 321
b = 123
print('{0} * {1} = {2}'.format(a, b, a * b))
```

A partir do Python 3.6, existe uma maneira mais simples de escrever strings formatadas: adicione `f` antes da string. Em uma string `f`, `{variable_name}` é um espaço reservado e será substituído pelo valor da variável, conforme mostrado abaixo.

```python
a = 321
b = 123
print(f'{a} * {b} = {a * b}')
```

Se precisarmos de mais controle sobre como os valores das variáveis ​​são mostrados, podemos usar padrões de formatação como os da tabela abaixo.

| Valor | Espaço reservado | Resultado | Significado |
| --- | --- | --- | --- |
| `3.1415926` | `{:.2f}` | `'3.14'` | mantenha 2 casas decimais |
| `3.1415926` | `{:+.2f}` | `'+3.14'` | mostre o sinal e mantenha 2 casas decimais |
| `-1` | `{:+.2f}` | `'-1.00'` | mostre o sinal e mantenha 2 casas decimais |
| `3.1415926` | `{:.0f}` | `'3'` | sem decimais |
| `123` | `{:0>10d}` | `'0000000123'` | bloco à esquerda com `0` com 10 caracteres |
| `123` | `{:x<10d}` | `'123xxxxxxx'` | bloco à direita com `x` para 10 caracteres |
| `123` | `{:>10d}` | `'       123'` | bloco à esquerda com espaços de até 10 caracteres |
| `123` | `{:<10d}` | `'123       '` | bloco à direita com espaços de até 10 caracteres |
| `123456789` | `{:,}` | `'123,456,789'` | formato separado por vírgula |
| `0.123` | `{:.2%}` | `'12.30%'` | formato percentual |
| `123456789` | `{:.2e}` | `'1.23e+08'` | notação científica |

#### Aparar

O método `strip` de uma string pode nos ajudar a obter uma nova string após remover caracteres especificados de ambas as extremidades da string original. Por padrão, remove espaços. Este método é muito prático. Ele pode ser usado para remover espaços iniciais e finais que um usuário digitou por engano. O método `strip` também possui versões `lstrip` e `rstrip`, e você provavelmente pode adivinhar o que eles fazem pelos nomes.

```python
s1 = '   jackfrued@126.com  '
print(s1.strip())      # jackfrued@126.com
s2 = '~Hello, world.~'
print(s2.lstrip('~'))  # Hello, world.~
print(s2.rstrip('~'))  # ~Hello, world.
```

#### Substituindo

Se quisermos substituir o conteúdo especificado em uma string por um novo conteúdo, podemos usar o método `replace`, conforme mostrado abaixo. O primeiro parâmetro de `replace` é o conteúdo a ser substituído, o segundo parâmetro é o novo conteúdo e o terceiro parâmetro pode ser usado para especificar o número de substituições.

```python
s = 'hello, good world'
print(s.replace('o', '@'))     # hell@, g@@d w@rld
print(s.replace('o', '@', 1))  # hell@, good world
```

#### Dividindo e Unindo

Podemos usar o método de string `split` para dividir uma string em várias strings e colocá-las em uma lista. Também podemos usar o método de string `join` para juntar muitas strings de uma lista em uma string, conforme mostrado abaixo.

```python
s = 'I love you'
words = s.split()
print(words)            # ['I', 'love', 'you']
print('~'.join(words))  # I~love~you
```

Deve ser explicado que `split` utiliza espaços por padrão. Também podemos usar outros caracteres para dividir a string, e também podemos definir o número máximo de divisões para controlar o resultado, conforme mostrado abaixo.

```python
s = 'I#love#you#so#much'
words = s.split('#')
print(words)
words = s.split('#', 2)
print(words)
```

#### Codificação e decodificação

Além do tipo string `str`, Python também possui um tipo string de bytes para dados binários, que é `bytes`. Uma string de bytes é uma sequência finita composta de zero ou mais bytes. Através do método `encode` de uma string, podemos codificar uma string em uma string de bytes em alguma codificação. Também podemos usar o método `decode` de uma string de bytes para decodificar uma string de bytes em uma string, conforme mostrado abaixo.

```python
a = 'Luo Hao'
b = a.encode('utf-8')
c = a.encode('gbk')
print(b)                  # b'\xe9\xaa\x86\xe6\x98\x8a'
print(c)                  # b'\xc2\xe6\xea\xbb'
print(b.decode('utf-8'))  # Luo Hao
print(c.decode('gbk'))    # Luo Hao
```

Observe que se os métodos de codificação e decodificação não forem os mesmos, isso causará texto ilegível, o que significa que o conteúdo original não pode ser restaurado ou pode aumentar `UnicodeDecodeError` e fazer o programa travar.

A ferramenta usada para correspondência de padrões é chamada de expressão regular. Python oferece suporte a expressões regulares por meio do módulo `re` na biblioteca padrão.
