### Gráficos de bolhas

Um gráfico de bolhas pode ser usado para compreender a relação entre três variáveis. Ao comparar a posição e o tamanho das bolhas, podemos analisar as relações entre as dimensões dos dados. Por exemplo, no gráfico de dispersão que traçamos anteriormente para a renda mensal e as despesas com compras online, já havíamos encontrado uma correlação positiva entre os dois. Se adicionarmos uma terceira variável, a frequência de compras, precisaremos de um gráfico de bolhas.

```python
income = np.array([5550, 7500, 10500, 15000, 20000, 25000, 30000, 40000])
outcome = np.array([800, 1800, 1250, 2000, 1800, 2100, 2500, 3500])
nums = np.array([5, 3, 10, 5, 12, 20, 8, 10])

plt.scatter(income, outcome, s=nums * 30, c=nums, cmap='Reds')
plt.colorbar()
plt.show()
```

### Gráficos de área

Um gráfico de área também é chamado de gráfico de linhas empilhadas. É baseado em um gráfico de linhas, mas a área abaixo da linha é preenchida com cores. É usado para mostrar valores em intervalos contínuos ou intervalos de tempo e geralmente é usado para mostrar tendências e comparar valores. O exemplo abaixo usa um gráfico de área para mostrar o tempo gasto dormindo, comendo, trabalhando e jogando de segunda a domingo.

```python
plt.figure(figsize=(8, 4))
days = np.arange(7)
sleeping = [7, 8, 6, 6, 7, 8, 10]
eating = [2, 3, 2, 1, 2, 3, 2]
working = [7, 8, 7, 8, 6, 2, 3]
playing = [8, 5, 9, 9, 9, 11, 9]
plt.stackplot(days, sleeping, eating, working, playing)
plt.xticks(days, labels=[f'Weekday {x}' for x in ['Mon', 'Tue', 'Wed', 'Thu', 'Fri', 'Sat', 'Sun']])
plt.legend(['Sleeping', 'Eating', 'Working', 'Playing'], fontsize=10)
plt.show()
```

### Gráficos de radar

Os gráficos de radar são geralmente usados ​​para comparar vários indicadores quantitativos e são bons para ver quais variáveis ​​têm valores semelhantes ou quais valores são baixos e quais são altos. Os leitores que assistem frequentemente a transmissões de basquete ou futebol devem estar familiarizados com gráficos de radar. A essência de um gráfico de radar ainda é um gráfico de linhas, mas é mapeado em coordenadas polares. Ao desenhar um gráfico de radar, precisamos fechar a linha, o que significa conectar o final ao início.

```python
labels = np.array(['Speed', 'Strength', 'Experience', 'Defense', 'Serve', 'Technique'])
malong_values = np.array([93, 95, 98, 92, 96, 97])
shuigu_values = np.array([30, 40, 65, 80, 45, 60])
angles = np.linspace(0, 2 * np.pi, labels.size, endpoint=False)
malong_values = np.append(malong_values, malong_values[0])
shuigu_values = np.append(shuigu_values, shuigu_values[0])
angles = np.append(angles, angles[0])
plt.figure(figsize=(4, 4), dpi=120)
ax = plt.subplot(projection='polar')
plt.plot(angles, malong_values, color='r', linewidth=2, label='Ma Long')
plt.fill(angles, malong_values, color='r', alpha=0.3)
plt.plot(angles, shuigu_values, color='g', linewidth=2, label='Jun Mizutani')
plt.fill(angles, shuigu_values, color='g', alpha=0.2)
ax.legend()
plt.show()
```

### Gráficos de rosas

Um gráfico rosa é um gráfico de barras mapeado em coordenadas polares. Foi inventado por Florence Nightingale e usado para apresentar a mortalidade sazonal em hospitais militares. Como a relação entre raio e área é quadrática, o gráfico rosa de Nightingale amplia as diferenças proporcionais, especialmente para valores próximos uns dos outros. Ao mesmo tempo, como o círculo tem natureza periódica, o gráfico rosa de Nightingale também é adequado para representar conceitos de tempo em um ciclo, como dias da semana ou meses.

```python
group1 = np.random.randint(20, 50, 4)
group2 = np.random.randint(10, 60, 4)
x = np.array([f'Group A-Q{i}' for i in range(1, 5)] + [f'Group B-Q{i}' for i in range(1, 5)])
y = np.array(group1.tolist() + group2.tolist())
theta = np.linspace(0, 2 * np.pi, x.size, endpoint=False)
width = 2 * np.pi / x.size
colors = np.random.rand(8, 3)
ax = plt.subplot(projection='polar')
plt.bar(theta, y, width=width, color=colors, bottom=0)
ax.set_thetagrids(theta * 180 / np.pi, x, fontsize=10)
plt.show()
```

### Gráficos 3D

Matplotlib também pode ser usado para desenhar gráficos 3D. Você pode verificar a documentação oficial para mais detalhes. Aqui usamos um código simples para mostrar como desenhar um gráfico 3D.

```python
from mpl_toolkits.mplot3d import Axes3D

fig = plt.figure(figsize=(8, 4), dpi=120)
ax = Axes3D(fig, auto_add_to_figure=False)
fig.add_axes(ax)
x = np.arange(-2, 2, 0.1)
y = np.arange(-2, 2, 0.1)
x, y = np.meshgrid(x, y)
z = (1 - y ** 5 + x ** 5) * np.exp(-x ** 2 - y ** 2)
ax.plot_surface(x, y, z)
plt.show()
```
