## Leitura e gravação de arquivos CSV em Python

### Uma introdução aos arquivos CSV

CSV, o nome completo é Valores Separados por Vírgula, é um formato de arquivo simples e comum. É amplamente utilizado na importação e exportação de dados de aplicações como bancos de dados e planilhas, e na troca de dados entre sistemas heterogêneos. Como o CSV é um arquivo de texto simples, independentemente do sistema operacional e da linguagem de programação, todos eles podem lidar com texto simples. Além disso, muitas linguagens de programação oferecem suporte para leitura e gravação de arquivos CSV, portanto, o formato CSV é amplamente utilizado em processamento e ciência de dados.

Os arquivos CSV têm as seguintes características:

1. Eles são texto simples e usam algum conjunto de caracteres, como [ASCII](https://en.wikipedia.org/wiki/ASCII), [Unicode](https://en.wikipedia.org/wiki/Unicode) ou [GB2312](https://en.wikipedia.org/wiki/GB_2312).
2. Eles são compostos de registros, normalmente um registro por linha.
3. Cada registro é dividido em campos ou colunas por um delimitador como vírgula, ponto e vírgula ou tabulação.
4. Cada registro segue a mesma sequência de campos.

Os arquivos CSV podem ser abertos e editados com um editor de texto ou ferramentas semelhantes ao Excel. Ao abrir um arquivo CSV com um software de planilha como o Excel, você quase não notará qualquer diferença entre um arquivo CSV e uma pasta de trabalho do Excel. Muitos sistemas de banco de dados suportam a exportação de dados para arquivos CSV e também suportam a importação de dados CSV de volta para o banco de dados, mas esse não é o foco da nossa discussão aqui.

### Gravando dados em um arquivo CSV

Suponha que precisemos salvar as notas dos exames de cinco alunos em três cursos em um arquivo CSV. Para fazer isso, podemos usar o módulo `csv` da biblioteca padrão do Python. A função `writer` do módulo retorna um objeto `csvwriter`, e através dos métodos `writerow` ou `writerows` desse objeto podemos gravar dados em um arquivo CSV. O código é mostrado abaixo.

```python
import csv
import random

with open('scores.csv', 'w') as file:
    writer = csv.writer(file)
    writer.writerow(['name', 'Chinese', 'Mathematics', 'English'])
    names = ['Guan Yu', 'Zhang Fei', 'Zhao Yun', 'Ma Chao', 'Huang Zhong']
    for name in names:
        scores = [random.randrange(50, 101) for _ in range(3)]
        scores.insert(0, name)
        writer.writerow(scores)
```

O arquivo CSV gerado tem esta aparência.

```text
name,Chinese,Mathematics,English
Guan Yu,98,86,61
Zhang Fei,86,58,80
Zhao Yun,95,73,70
Ma Chao,83,97,55
Huang Zhong,61,54,87
```

Deve ser explicado que para a função `writer` acima, além de passar o objeto arquivo para gravar os dados, também podemos passar o parâmetro `dialect`, que significa o dialeto do arquivo CSV, e seu valor padrão é `excel`. Além disso, também podemos utilizar os parâmetros `delimiter`, `quotechar` e `quoting` para especificar o delimitador, o caractere dos valores circundantes e a forma de circundar. Entre eles, o caractere para valores circundantes é usado principalmente quando há símbolos especiais em um campo. Ao adicionar o caráter circundante, a ambiguidade pode ser evitada. Você pode tentar alterar o código acima para o código a seguir e, em seguida, observar o arquivo CSV gerado.

```python
writer = csv.writer(file, delimiter='|', quoting=csv.QUOTE_ALL)
```

O arquivo CSV gerado ficará assim.

```text
"name"|"Chinese"|"Mathematics"|"English"
"Guan Yu"|"88"|"64"|"65"
"Zhang Fei"|"76"|"93"|"79"
"Zhao Yun"|"78"|"55"|"76"
"Ma Chao"|"72"|"77"|"68"
"Huang Zhong"|"70"|"72"|"51"
```

### Lendo dados de um arquivo CSV

Se quisermos ler o arquivo CSV que acabamos de criar, podemos usar o código abaixo. A função `reader` no módulo `csv` cria um objeto `csvreader`. Esse objeto é um iterador, portanto podemos ler os dados do arquivo com a função `next` ou um loop `for-in`.

```python
import csv

with open('scores.csv', 'r') as file:
    reader = csv.reader(file, delimiter='|')
    for data_list in reader:
        print(reader.line_num, end='\t')
        for elem in data_list:
            print(elem, end='\t')
        print()
```

> **Observação**: No loop `for` acima, cada iteração recupera um objeto de lista de `csvreader`, e essa lista contém todos os campos em uma linha.

### Resumo

Se todos usarem Python para análise de dados no futuro, é muito provável que você use a biblioteca de terceiros chamada `pandas`. É uma das ferramentas mágicas de análise de dados Python. Em `pandas`, as funções denominadas `read_csv` e `to_csv` são agrupadas para leitura e gravação de arquivos CSV. `read_csv` transforma os dados lidos em um objeto `DataFrame`, e `DataFrame` é o tipo mais importante na biblioteca `pandas`. Ele envolve uma série de métodos usados para processamento de dados, como limpeza, transformação e agregação. `to_csv` grava os dados em um objeto `DataFrame` em um arquivo CSV e conclui a persistência dos dados. A função `read_csv` e a função `to_csv` são muito mais poderosas do que as nativas `csvreader` e `csvwriter`.
