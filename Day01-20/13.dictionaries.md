## Estruturas de dados comuns: dicionários

Até agora, já introduzimos três tipos de dados contêineres em Python: listas, tuplas e conjuntos. Mas estes tipos de dados ainda não são suficientes para nos ajudar a resolver todos os problemas. Por exemplo, se precisarmos de uma variável para salvar muitas informações sobre uma pessoa, incluindo nome, idade, altura, peso, endereço residencial, número de celular pessoal e número de contato de emergência, você descobrirá que a lista, a tupla e os tipos de conjuntos que aprendemos antes não são bons o suficiente.

```python
person1 = ['Wang Dahuang', 55, 168, 60, '101, Building 1, No. 62, Kehua North Road, Wuhou District, Chengdu', '13122334455', '13800998877']
person2 = ('Wang Dahuang', 55, 168, 60, '101, Building 1, No. 62, Kehua North Road, Wuhou District, Chengdu', '13122334455', '13800998877')
person3 = {'Wang Dahuang', 55, 168, 60, '101, Building 1, No. 62, Kehua North Road, Wuhou District, Chengdu', '13122334455', '13800998877'}
```

Os conjuntos são definitivamente os menos adequados, porque os conjuntos não podem ter elementos duplicados. Se a idade e o peso de uma pessoa forem iguais, o aparelho perderá uma informação. Da mesma forma, se o telefone e o número do contato de emergência da pessoa forem iguais, o aparelho perderá outra informação. Por outro lado, embora listas e tuplas possam salvar todas as informações de uma pessoa, quando você deseja obter o número de telefone ou endereço residencial dessa pessoa, primeiro você precisa saber qual posição do elemento contém essas informações. Resumindo, em cenas como a acima, listas, tuplas e conjuntos não são as escolhas mais adequadas. Neste momento, precisamos do tipo **dictionary**. Esse tipo de dados é mais adequado para reunir informações relacionadas e pode nos ajudar a resolver o problema de modelagem de coisas reais em programas Python.

A palavra “dicionário” já deve ser familiar. Quando estávamos no ensino fundamental, quase todo mundo tinha um exemplar do *Dicionário Xinhua*, conforme mostrado abaixo.

<img src="res/day13/xinhua_dictionary.jpg" style="zoom:80%;">

O dicionário em Python é muito parecido com um dicionário da vida real. Ele organiza os dados na forma de pares de valores-chave, e podemos encontrar o valor correspondente por meio da chave e trabalhar com ele. Assim como no *Dicionário Xinhua*, cada caractere chinês tem uma explicação correspondente. Cada caractere e sua explicação juntos formam uma entrada no dicionário, e um dicionário geralmente contém muitas dessas entradas.

### Criando e usando dicionários

Em Python, podemos criar um dicionário com a sintaxe literal `{}`. Este é o mesmo conjunto sobre o qual falamos na lição anterior. Mas os elementos dentro de `{}` em um dicionário são pares de valores-chave. Cada elemento é composto por dois valores separados por `:`. A parte anterior a `:` é a chave e a parte posterior a `:` é o valor, conforme mostrado abaixo.

```python
xinhua = {
    'Lu': 'foot of the mountain',
    'road': 'Road, a place for traffic; aspect, region: south ~ goods, outside ~ goods; type: they are one person ~',
    'lotus root': 'Alias of Licorice',
    'Lu': 'Lu Shui, water name, that is, the Zhuozhang River in Shanxi Province; Lu River, water name, that is, the Nu River in Yunnan Province'
}
print(xinhua)
person = {
    'name': 'Wang Dahuang',
    'age': 55,
    'height': 168,
    'weight': 60,
    'addr': '101, Building 1, No. 62, Kehua North Road, Wuhou District, Chengdu',
    'tel': '13122334455',
    'emergence contact': '13800998877'
}
print(person)
```

Pelo código acima, você provavelmente já pode ver que usar um dicionário para salvar as informações de uma pessoa é muito melhor do que usar uma lista ou tupla, porque podemos usar a chave antes de `:` para mostrar o significado da entrada, e o valor após `:` é o valor que pertence a essa entrada.

Claro, se quisermos, também podemos usar a função integrada `dict` ou a sintaxe de compreensão de dicionário para criar dicionários, conforme mostrado abaixo.

```python
person = dict(name='Wang Dahuang', age=55, height=168, weight=60, addr='101, Building 1, No. 62, Kehua North Road, Wuhou District, Chengdu')
print(person)  # {'name': 'Wang Dahuang', 'age': 55, 'height': 168, 'weight': 60, 'addr': '101, Building 1, No. 62, Kehua North Road, Wuhou District, Chengdu'}

items1 = dict(zip('ABCDE', '12345'))
print(items1)  # {'A': '1', 'B': '2', 'C': '3', 'D': '4', 'E': '5'}
items2 = dict(zip('ABCDE', range(1, 10)))
print(items2)  # {'A': 1, 'B': 2, 'C': 3, 'D': 4, 'E': 5}

items3 = {x: x ** 3 for x in range(1, 6)}
print(items3)
```

Se quisermos saber quantos pares de valores-chave existem em um dicionário, ainda usaremos a função `len`. Se quisermos iterar em um dicionário, podemos usar um loop `for`, mas deve-se observar que o loop `for` itera apenas nas chaves do dicionário. Não importa. Depois de aprender as operações de indexação de dicionário, podemos usar as chaves para acessar os valores correspondentes.

```python
person = {
    'name': 'Wang Dahuang',
    'age': 55,
    'height': 168,
    'weight': 60,
    'addr': '101, Building 1, No. 62, Kehua North Road, Wuhou District, Chengdu'
}
print(len(person))  # 5
for key in person:
    print(key)
```

### Operações de dicionário

Para o tipo de dicionário, as operações de associação e de indexação são certamente muito importantes. O primeiro pode determinar se uma chave especificada está no dicionário, e o último pode usar uma chave para acessar o valor correspondente ou adicionar um novo par chave-valor ao dicionário. É importante notar que a indexação de dicionário é diferente da indexação de lista. Os elementos de uma lista possuem suas próprias posições, portanto o índice de uma lista é um número inteiro. Mas como um dicionário salva pares de valores-chave, ele precisa usar chaves para indexar os valores correspondentes. É **especialmente importante** lembrar a todos que **as chaves em um dicionário devem ser tipos imutáveis**, como números inteiros (`int`), números de ponto flutuante (`float`), strings (`str`) e tuplas (`tuple`). Este é o mesmo requisito que os conjuntos impõem aos seus elementos. Claramente, os tipos lista (`list`) e conjunto (`set`) de que falamos antes não podem ser usados como chaves em um dicionário, e o próprio tipo de dicionário também não pode ser usado como chave em outro dicionário, porque os dicionários também são mutáveis. Mas listas, conjuntos e dicionários podem ser usados como valores em um dicionário.

```python
person = {
    'name': 'Wang Dahuang',
    'age': 55,
    'height': 168,
    'weight': 60,
    'addr': ['101, Building 1, No. 62, Kehua North Road, Wuhou District, Chengdu', 'No. 1 Baiwanzhuang Street, Xicheng District, Beijing'],
    'car': {
        'brand': 'BMW X7',
        'maxSpeed': '250',
        'length': 5170,
        'width': 2000,
        'height': 1835,
        'displacement': 3.0
    }
}
print(person)
```

Observe o código abaixo para entender as operações de associação ao dicionário e as operações de indexação.

```python
person = {'name': 'Wang Dahuang', 'age': 55, 'height': 168, 'weight': 60, 'addr': '101, Building 1, No. 62, Kehua North Road, Wuhou District, Chengdu'}

print('name' in person)  # True
print('tel' in person)   # False

print(person['name'])
print(person['addr'])
person['age'] = 25
person['height'] = 178
person['tel'] = '13122334455'
person['signature'] = 'Your boyfriend is a piece of crap, he will step on the multicolored clouds to marry your girlfriend'
print(person)

for key in person:
    print(f'{key}:\t{person[key]}')
```

Se você usar uma chave ausente com indexação, o Python gerará `KeyError`.

### Métodos de dicionário

Dentre os métodos do tipo dicionário, a maioria deles está relacionada a operações em pares chave-valor. O método `get` pode usar uma chave para obter o valor correspondente. Diferente das operações de indexação, quando a chave especificada não está no dicionário, o método `get` não gera uma exceção. Em vez disso, ele retorna `None` ou um valor padrão especificado.

```python
person = {'name': 'Wang Dahuang', 'age': 25, 'height': 178, 'addr': '101, Building 1, No. 62, Kehua North Road, Wuhou District, Chengdu'}
print(person.get('name'))       # Wang Dahuang
print(person.get('sex'))        # None
print(person.get('sex', True))  # True
```

Se precisarmos obter todas as chaves de um dicionário, podemos usar o método `keys`. Se precisarmos obter todos os valores em um dicionário, podemos usar o método `values`. O dicionário também possui um método chamado `items`. Ele reúne chaves e valores em duas tuplas, e usar esse método para iterar os elementos de um dicionário também é muito conveniente.

```python
person = {'name': 'Wang Dahuang', 'age': 25, 'height': 178}
print(person.keys())    # dict_keys(['name', 'age', 'height'])
print(person.values())  # dict_values(['Wang Dahuang', 25, 178])
print(person.items())   # dict_items([('name', 'Wang Dahuang'), ('age', 25), ('height', 178)])
for key, value in person.items():
    print(f'{key}:\t{value}')
```

O método `update` de um dicionário pode mesclar dois dicionários. Por exemplo, se houver dois dicionários `x` e `y`, quando executarmos `x.update(y)`, para chaves iguais em `x` e `y`, os valores em `x` serão atualizados com os valores em `y`; e os pares de valores-chave que existem em `y`, mas não em `x`, serão adicionados diretamente a `x`, conforme mostrado abaixo.

```python
person1 = {'name': 'Wang Dahuang', 'age': 55, 'height': 178}
person2 = {'age': 25, 'addr': '101, Building 1, No. 62, Kehua North Road, Wuhou District, Chengdu'}
person1.update(person2)
print(person1)  # {'name': 'Wang Dahuang', 'age': 25, 'height': 178, 'addr': '101, Building 1, No. 62, Kehua North Road, Wuhou District, Chengdu'}
```

Se você usa Python 3.9 ou posterior, também pode usar o operador `|` para fazer a mesma coisa, conforme mostrado abaixo.

```python
person1 = {'name': 'Wang Dahuang', 'age': 55, 'height': 178}
person2 = {'age': 25, 'addr': '101, Building 1, No. 62, Kehua North Road, Wuhou District, Chengdu'}
person1 |= person2
print(person1)  # {'name': 'Wang Dahuang', 'age': 25, 'height': 178, 'addr': '101, Building 1, No. 62, Kehua North Road, Wuhou District, Chengdu'}
```

Podemos usar o método `pop` ou `popitem` para excluir elementos de um dicionário. O primeiro retorna o valor correspondente à chave, mas se a chave especificada não existir no dicionário, gera `KeyError`. O último retorna uma tupla dupla composta pela chave e pelo valor ao excluir o elemento. O método `clear` de um dicionário limpará todos os pares de valores-chave no dicionário, conforme mostrado abaixo.

```python
person = {'name': 'Wang Dahuang', 'age': 25, 'height': 178, 'addr': '101, Building 1, No. 62, Kehua North Road, Wuhou District, Chengdu'}
print(person.pop('age'))  # 25
print(person)             # {'name': 'Wang Dahuang', 'height': 178, 'addr': '101, Building 1, No. 62, Kehua North Road, Wuhou District, Chengdu'}
print(person.popitem())   # ('addr', '101, Building 1, No. 62, Kehua North Road, Wuhou District, Chengdu')
print(person)             # {'name': 'Wang Dahuang', 'height': 178}
person.clear()
print(person)             # {}
```

Assim como nas listas, também podemos usar a palavra-chave `del` para excluir elementos de um dicionário. Se a chave especificada não conseguir encontrar um valor correspondente ao excluir um elemento, ela também gerará `KeyError`, conforme mostrado abaixo.

```python
person = {'name': 'Wang Dahuang', 'age': 25, 'height': 178, 'addr': '101, Building 1, No. 62, Kehua North Road, Wuhou District, Chengdu'}
del person['age']
del person['addr']
print(person)  # {'name': 'Wang Dahuang', 'height': 178}
```

### Aplicativos de dicionário

Podemos usar alguns exemplos simples para ver como usar o tipo de dicionário para resolver alguns problemas reais.

**Exemplo 1**: Insira uma passagem e conte quantas vezes cada letra em inglês aparece. Produza o resultado de alto para baixo por contagem.

```python
sentence = input('Please enter a sentence: ')
counter = {}
for ch in sentence:
    if 'A' <= ch <= 'Z' or 'a' <= ch <= 'z':
        counter[ch] = counter.get(ch, 0) + 1
sorted_keys = sorted(counter, key=counter.get, reverse=True)
for key in sorted_keys:
    print(f'{key} Appeared {counter[key]} times.')
```

**Exemplo 2**: Em um dicionário, salve códigos de ações e preços. Encontre as ações cujo preço seja superior a 100 yuans e crie um novo dicionário.

> **Nota**: Podemos usar a sintaxe de compreensão de dicionário para criar este novo dicionário.

```python
stocks = {
    'AAPL': 191.88,
    'GOOG': 1186.96,
    'IBM': 149.24,
    'ORCL': 48.44,
    'ACN': 166.89,
    'FB': 208.09,
    'SYMC': 21.29
}
stocks2 = {key: value for key, value in stocks.items() if value > 100}
print(stocks2)
```

### Resumo

Os dicionários em Python são muito semelhantes aos dicionários da vida real. Eles nos permitem **salvar dados na forma de pares de valores-chave** e, em seguida, **usar chaves para acessar os valores correspondentes**. Um dicionário é um tipo de dados muito **conveniente para pesquisa de dados**, mas deve ser enfatizado novamente que **as chaves em um dicionário devem ser tipos imutáveis**. Dados como listas, conjuntos e dicionários não podem ser usados como chaves de dicionário.
