## pandas em profundidade - 6

Vejamos agora o tipo `Index`. Ele fornece serviços de indexação para objetos `Series` e `DataFrame`. Com os índices, podemos classificar os dados com o método `sort_index`, alinhar os dados, o que é muito importante nas operações e mesclagens de dados, e obter recuperação rápida de dados por meio de operações de indexação. Como `DataFrame` representa dados bidimensionais, tanto suas linhas quanto suas colunas possuem índices, ou seja, `index` e `columns`.

Criar um objeto `Index` é relativamente simples. Normalmente, precisamos fornecer apenas três parâmetros: `data`, `dtype` e `name`, que representam os dados usados como índice, o tipo de dados do índice e o nome do índice. Como o próprio `Index` também é um dado unidimensional, seus métodos e propriedades são muito semelhantes aos do `Series`. Você pode tentar criar um objeto `Index` e ver se as propriedades e métodos que aprendemos antes também funcionam em `Index`. A seguir, examinamos principalmente vários subtipos de `Index`.

### Índice de intervalo

Um índice de intervalo é um índice feito de inteiros monotônicos. Podemos criá-lo através do construtor `RangeIndex`, ou através do método de classe `from_range` de `RangeIndex`.

```python
sales_data = np.random.randint(400, 1000, 12)
index = pd.RangeIndex(1, 13, name='Month')
ser = pd.Series(data=sales_data, index=index)
ser
```

Saída:

```
Month
1     703
2     705
3     557
4     943
5     961
6     615
7     788
8     985
9     921
10    951
11    874
12    609
dtype: int64
```

### Índice Categórico

Um índice categórico é um índice construído a partir de dados em escala nominal. Se precisarmos agrupar os dados por índice e depois agregar, um índice categórico pode ser muito útil. Um índice categórico também possui um método chamado `reorder_categories`, que pode especificar uma ordem para o índice. O resultado da agregação agrupada será então apresentado na ordem especificada.

```python
sales_data = [6, 6, 7, 6, 8, 6]
index = pd.CategoricalIndex(
    data=['Apple', 'Banana', 'Apple', 'Apple', 'Peach', 'Banana'],
    categories=['Apple', 'Banana', 'Peach'],
    ordered=True
)
ser = pd.Series(data=sales_data, index=index)
ser
```

Saída:

```
Apple     6
Banana    6
Apple     7
Apple     6
Peach     8
Banana    6
dtype: int64
```

Agrupe os dados pelo índice e depois some-os com `sum`.

```python
ser.groupby(level=0).sum()
```

Saída:

```
Apple     19
Banana    12
Peach      8
dtype: int64
```

Especifique a ordem do índice:

```python
ser.index = index.reorder_categories(['Banana', 'Peach', 'Apple'])
ser.groupby(level=0).sum()
```

Saída:

```
Banana    12
Peach      8
Apple     19
dtype: int64
```

### MultiÍndice

O tipo `MultiIndex` em pandas é usado para representar índices hierárquicos ou multiníveis. Podemos usar métodos de classe como `from_arrays`, `from_product` e `from_tuples` de `MultiIndex` para criar índices de vários níveis. Vejamos alguns exemplos.

```python
tuples = [(1, 'red'), (1, 'blue'), (2, 'red'), (2, 'blue')]
index = pd.MultiIndex.from_tuples(tuples, names=['no', 'color'])
index
```

Saída:

```
MultiIndex([(1,  'red'),
            (1, 'blue'),
            (2,  'red'),
            (2, 'blue')],
           names=['no', 'color'])
```

```python
arrays = [[1, 1, 2, 2], ['red', 'blue', 'red', 'blue']]
index = pd.MultiIndex.from_arrays(arrays, names=['no', 'color'])
index
```

Saída:

```
MultiIndex([(1,  'red'),
            (1, 'blue'),
            (2,  'red'),
            (2, 'blue')],
           names=['no', 'color'])
```

```python
sales_data = np.random.randint(1, 100, 4)
ser = pd.Series(data=sales_data, index=index)
ser
```

Saída:

```

dtype: int64
```

```python
ser.groupby('no').sum()
```

Saída:


### Índice de intervalo

Como o nome sugere, um índice de intervalo usa intervalos de intervalo fixos como índice. Normalmente usamos a função `interval_range` para criar um índice de intervalo.

```python
index = pd.interval_range(start=0, end=5)
index
```

Saída:

```
IntervalIndex([(0, 1], (1, 2], (2, 3], (3, 4], (4, 5]], dtype='interval[int64, right]')
```

`IntervalIndex` possui um método denominado `contains`, que pode verificar se um determinado elemento está dentro do intervalo de intervalo.

```python
index.contains(1.5)
```

Saída:

```
array([False,  True, False, False, False])
```

`IntervalIndex` também possui um método chamado `overlaps`, que pode verificar se um intervalo se sobrepõe aos outros.

```python
index.overlaps(pd.Interval(1.5, 3.5))
```

Saída:

```
array([False,  True,  True,  True, False])
```

Se quisermos que os intervalos sejam fechados à esquerda e abertos à direita, podemos fazer isso ao criar o índice de intervalo por meio de `closed='left'`. Se quisermos que ambos os lados sejam fechados, podemos definir o parâmetro `closed` como `both`.

```python
index = pd.interval_range(start=0, end=5, closed='left')
index
```

Saída:

```
IntervalIndex([[0, 1), [1, 2), [2, 3), [3, 4), [4, 5)], dtype='interval[int64, left]')
```

```python
index = pd.interval_range(start=pd.Timestamp('2022-01-01'), end=pd.Timestamp('2022-01-04'), closed='both')
index
```

Saída:

```
IntervalIndex([[2022-01-01, 2022-01-02], [2022-01-02, 2022-01-03], [2022-01-03, 2022-01-04]], dtype='interval[datetime64[ns], both]')
```

### Índice de data e hora

`DatetimeIndex` é provavelmente um dos tipos de índice mais complexos e importantes. Normalmente usamos a função `date_range()` para criá-lo. Esta função possui vários parâmetros muito importantes: `start`, `end`, `periods`, `freq` e `tz`, que representam data e hora de início, data e hora de término, número de períodos, frequência de amostragem e fuso horário.

Vamos primeiro ver como criar um objeto `DatetimeIndex` e depois falar sobre as operações relacionadas.

```python
pd.date_range('2021-1-1', '2021-6-30', periods=10)
```

Saída:

```
DatetimeIndex(['2021-01-01', '2021-01-21', '2021-02-10', '2021-03-02',
               '2021-03-22', '2021-04-11', '2021-05-01', '2021-05-21',
               '2021-06-10', '2021-06-30'],
              dtype='datetime64[ns]', freq=None)
```

```python
pd.date_range('2021-1-1', '2021-6-30', freq='W')
```

> **Observação**: `freq='W'` significa que a frequência de amostragem é de uma semana e, por padrão, o domingo é tratado como o início de uma semana. Se quiser que segunda-feira seja o início da semana, você pode alterá-lo para `freq='W-MON'`. Você também pode tentar valores como `12H`, `M` ou `Q` e ver o que acontece.

Saída:

```
DatetimeIndex(['2021-01-03', '2021-01-10', '2021-01-17', '2021-01-24',
               '2021-01-31', '2021-02-07', '2021-02-14', '2021-02-21',
               '2021-02-28', '2021-03-07', '2021-03-14', '2021-03-21',
               '2021-03-28', '2021-04-04', '2021-04-11', '2021-04-18',
               '2021-04-25', '2021-05-02', '2021-05-09', '2021-05-16',
               '2021-05-23', '2021-05-30', '2021-06-06', '2021-06-13',
               '2021-06-20', '2021-06-27'],
              dtype='datetime64[ns]', freq='W-SUN')
```

`DatetimeIndex` pode operar com o tipo `DateOffset`. Isso é fácil de entender, porque podemos definir uma diferença horária e avançar ou retroceder a hora.

```python
index = pd.date_range('2021-1-1', '2021-6-30', freq='W')
index - pd.DateOffset(days=2)
```

Se um objeto `Series` ou objeto `DataFrame` usar `DatetimeIndex` como índice, poderemos usar o método `asfreq()` para especificar uma frequência de tempo e amostrar os dados. Ainda usamos os dados de estoque do Baidu anteriores como exemplo.

```python
baidu_df = pd.read_excel('data/2022-stock-data.xlsx', sheet_name='BIDU', index_col='Date')
baidu_df.sort_index(inplace=True)
baidu_df.asfreq('5D')
```

Saída:

<img src="res/baidu_stock_asfreq.png" style="zoom:50%;">

Você deve ter notado que a amostragem de um dia a cada 5 dias pode atingir dias sem negociação e, então, as colunas correspondentes tornam-se valores nulos. Para resolver este problema, ao utilizar o método `asfreq`, podemos utilizar o parâmetro `method` para especificar uma forma de preencher valores nulos, para que os dados dos dias de negociação próximos sejam preenchidos.

```python
baidu_df.asfreq('5D', method='ffill')
```

Saída:

<img src="res/baidu_stock_asfreq_ffill.png" style="zoom:50%;">

Ao usar `DatetimeIndex`, também podemos usar o método `resample()` para reamostrar os dados com base no tempo. É como agrupar dados por períodos de tempo e, após o agrupamento, ainda podemos fazer a agregação.

```python
baidu_df.resample('1M').mean()
```
