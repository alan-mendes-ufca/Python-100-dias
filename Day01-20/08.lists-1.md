## Common Data Structures: Lists, Part 1

Before we begin this lesson, let us first look at a small programming task: roll one die 6000 times and count how many times each face appears. This task should already be very simple for everyone. We can use random numbers uniformly distributed from 1 to 6 to simulate rolling a die, and then use 6 variables to record how many times each face appears. Based on what we learned earlier, I believe everyone can write the following code without much trouble.

```python
"""
Roll a die 6000 times and count the occurrences of each face

Author: Luo Hao
Version: 1.0
"""
import random

f1 = 0
f2 = 0
f3 = 0
f4 = 0
f5 = 0
f6 = 0
for _ in range(6000):
    face = random.randrange(1, 7)
    if face == 1:
        f1 += 1
    elif face == 2:
        f2 += 1
    elif face == 3:
        f3 += 1
    elif face == 4:
        f4 += 1
    elif face == 5:
        f5 += 1
    else:
        f6 += 1
print(f'Face 1 appeared {f1} times')
print(f'Face 2 appeared {f2} times')
print(f'Face 3 appeared {f3} times')
print(f'Face 4 appeared {f4} times')
print(f'Face 5 appeared {f5} times')
print(f'Face 6 appeared {f6} times')
```

I probably do not need to say much about how "ugly" the code above is. What is even worse is that if we wanted to roll two dice, or even more dice, and then count how many times each possible result appears, we would need to define more variables and write more branching code. Just thinking about that already feels unpleasant. At this point, a question may already be in your mind: is there a way to use one variable to store multiple pieces of data, and is there a way to use one kind of code to work on multiple pieces of data? The answer is yes. In Python, we can save and work with multiple pieces of data through container-type variables. The first new data type I will introduce is the **list**.

### Creating Lists

In Python, a **list is a data sequence made up of a series of elements in a specific order**. This means that if we define a variable of list type, **we can use it to store multiple pieces of data**. In Python, we can use the literal syntax `[]` to define a list, and the elements in the list are separated by commas, as shown below.

```python
items1 = [35, 12, 99, 68, 55, 35, 87]
items2 = ['Python', 'Java', 'Go', 'Kotlin']
items3 = [100, 12.3, 'Python', True]
print(items1)  # [35, 12, 99, 68, 55, 35, 87]
print(items2)  # ['Python', 'Java', 'Go', 'Kotlin']
print(items3)  # [100, 12.3, 'Python', True]
```

> **Note**: A list can contain duplicate elements, such as the repeated `35` in `items1`. It can also contain values of different types, as shown in `items3`, which contains `int`, `float`, `str`, and `bool` values. However, we usually do not recommend putting many different types into one list, because it is inconvenient to work with them.

We can use the `type` function to see the type of a variable. Interested readers can check for themselves what type the variable `items1` above really is. Because a list can hold many elements and is a container-type data structure, when we name variables of list type, the variable names usually use plural words.

In addition, we can also use Python's built-in `list` function to turn other sequences into lists. More precisely, `list` is not an ordinary function but a constructor for creating list objects. Later lessons will introduce concepts such as objects and constructors.

```python
items4 = list(range(1, 10))
items5 = list('hello')
print(items4)  # [1, 2, 3, 4, 5, 6, 7, 8, 9]
print(items5)  # ['h', 'e', 'l', 'l', 'o']
```

> **Note**: `range(1, 10)` produces the integers from `1` to `9`. Passing that sequence to `list` creates a list made of the integers from `1` to `9`. A string is also a sequence, so `list('hello')` creates a list whose elements are the characters in the string `hello`.

### Operations on Lists

We can use the `+` operator to join two lists. This operation connects the elements in the two lists and puts them into one list, as shown below.

```python
items5 = [35, 12, 99, 45, 66]
items6 = [45, 58, 29]
items7 = ['Python', 'Java', 'JavaScript']
print(items5 + items6)  # [35, 12, 99, 45, 66, 45, 58, 29]
print(items6 + items7)  # [45, 58, 29, 'Python', 'Java', 'JavaScript']
items5 += items6
print(items5)  # [35, 12, 99, 45, 66, 45, 58, 29]
```

We can use the `*` operator to repeat a list. The `*` operator repeats the list elements a specified number of times. Add the following two lines to the code above.

```python
print(items6 * 3)  # [45, 58, 29, 45, 58, 29, 45, 58, 29]
print(items7 * 2)  # ['Python', 'Java', 'JavaScript', 'Python', 'Java', 'JavaScript']
```

We can use `in` or `not in` to check whether an element is in a list. Add the following lines to the code above.

```python
print(29 in items6)  # True
print(99 in items6)  # False
print('C++' not in items7)     # True
print('Python' not in items7)  # False
```

Because a list has many elements, and these elements are placed in the list in a specific order, when we want to work with one element in the list, we can use the `[]` operator and specify the position of the element inside `[]` to access it. This operation is called indexing. The position inside `[]` can be an integer from `0` to `N - 1`, or an integer from `-1` to `-N`. They are called forward indexes and reverse indexes, where `N` means the number of elements in the list. For forward indexes, `[0]` accesses the first element in the list, and `[N - 1]` accesses the last element. For reverse indexes, `[-1]` accesses the last element in the list, and `[-N]` accesses the first element, as shown below.

```python
items8 = ['apple', 'waxberry', 'pitaya', 'peach', 'watermelon']
print(items8[0])   # apple
print(items8[2])   # pitaya
print(items8[4])   # watermelon
items8[2] = 'durian'
print(items8)      # ['apple', 'waxberry', 'durian', 'peach', 'watermelon']
print(items8[-5])  # apple
print(items8[-4])  # waxberry
print(items8[-1])  # watermelon
items8[-4] = 'strawberry'
print(items8)      # ['apple', 'strawberry', 'durian', 'peach', 'watermelon']
```

When using indexing, avoid going out of range. For `items8` above, if we access `items8[5]` or `items8[-6]`, it will raise an `IndexError` and crash the program. The corresponding error message is `list index out of range`. Because the list `items8` has only five elements, the valid forward indexes are `0` to `4`, and the valid reverse indexes are `-1` to `-5`.

If you want to access many elements in the list at one time, you can use slicing. A slicing operation has the form `[start:end:stride]`. Here, `start` means the starting position, `end` means the ending position, and the element at the ending position itself is not included, while `stride` means the step. Simply speaking, if the first element we access is at position `start`, then the second element is at position `start + stride`, and of course `start + stride` must still stay within the slice range. Add the following statements to the code above to use slicing to access list elements.

```python
print(items8[1:3:1])     # ['strawberry', 'durian']
print(items8[0:3:1])     # ['apple', 'strawberry', 'durian']
print(items8[0:5:2])     # ['apple', 'durian', 'watermelon']
print(items8[-4:-2:1])   # ['strawberry', 'durian']
print(items8[-2:-6:-1])  # ['peach', 'durian', 'strawberry', 'apple']
```

If the value of `start` is `0`, it can be omitted when using slicing. If the value of `end` is `N`, where `N` is the number of elements in the list, it can also be omitted. If the value of `stride` is `1`, it can also be omitted. Therefore, the code below has exactly the same effect as the code above.

```python
print(items8[1:3])     # ['strawberry', 'durian']
print(items8[:3:1])    # ['apple', 'strawberry', 'durian']
print(items8[::2])     # ['apple', 'durian', 'watermelon']
print(items8[-4:-2])   # ['strawberry', 'durian']
print(items8[-2::-1])  # ['peach', 'durian', 'strawberry', 'apple']
```

> **Reminder**: Look at the last line in the code above and think about how slicing accesses elements when the step is a negative number.

In fact, we can also use slicing to modify elements in a list. For example, add one more line to the code above and see what the output is.

```python
items8[1:3] = ['x', 'o']
print(items8)  # ['apple', 'x', 'o', 'peach', 'watermelon']
```

Two lists can also be compared. We can compare whether two lists are equal, and we can also compare their sizes, as shown below.

```python
nums1 = [1, 2, 3, 4]
nums2 = list(range(1, 5))
nums3 = [3, 2, 1]
print(nums1 == nums2)  # True
print(nums1 != nums2)  # False
print(nums1 <= nums3)  # True
print(nums2 >= nums3)  # False
```

> **Note**: The corresponding elements of `nums1` and `nums2` are exactly the same, so the result of `==` is `True`. For the comparison between `nums2` and `nums3`, because the first element of `nums2`, which is `1`, is smaller than the first element of `nums3`, which is `3`, the result of `nums2 >= nums3` is `False`. Comparisons between two lists are not used that often in real work. If you do not understand them, just skip them and do not get stuck here.

### Iterating Over Elements

If we want to take out the elements in a list one by one, we can use a `for-in` loop. There are two common ways.

Method 1: use indexing inside the loop to traverse the list elements.

```python
languages = ['Python', 'Java', 'C++', 'Kotlin']
for index in range(len(languages)):
    print(languages[index])
```

Output:

```
Python
Java
C++
Kotlin
```

> **Note**: The `len` function above gets the number of list elements, which is `N`, and `range(N)` then builds the range from `0` to `N - 1`, which matches the indexes of the list elements exactly.

Method 2: loop over the list directly, so the loop variable itself represents each list element.

```python
languages = ['Python', 'Java', 'C++', 'Kotlin']
for language in languages:
    print(language)
```

Output:

```
Python
Java
C++
Kotlin
```

### Summary

Now we can use our knowledge of lists to rewrite the die-rolling example at the beginning.

```python
"""
Roll a die 6000 times and count the occurrences of each face

Author: Luo Hao
Version: 1.1
"""
import random

counters = [0] * 6
# Simulate rolling the die and record the count of each face.
for _ in range(6000):
    face = random.randrange(1, 7)
    counters[face - 1] += 1
# Print the counts.
for face in range(1, 7):
    print(f'Face {face} appeared {counters[face - 1]} times')
```

In the code above, we use the six elements in the `counters` list to represent how many times faces 1 to 6 appear. At the beginning, the values of all six elements are `0`. Next, we use random numbers uniformly distributed from `1` to `6` to simulate rolling the die. If the result is `1`, the value of `counters[0]` increases by `1`; if the result is `2`, the value of `counters[1]` increases by `1`; and so on. You can feel the difference here. Because we use the list type together with a loop structure, our handling of data becomes batch-style, and this makes the rewritten code much simpler and more elegant than the earlier version.
