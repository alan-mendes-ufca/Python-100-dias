## Trabalhando com arquivos PDF em Python

PDF é a forma abreviada de Portable Document Format. Esse tipo de arquivo geralmente usa `.pdf` como extensão. No trabalho diário de desenvolvimento, as tarefas mais fáceis de realizar são a leitura de conteúdo de texto em PDF e a geração de documentos PDF com conteúdo existente.

### Extraindo texto de PDFs

Em Python, podemos usar a biblioteca de terceiros `PyPDF2` para ler arquivos PDF. Instale-o com o seguinte comando.

```bash
pip install PyPDF2
```

`PyPDF2` não pode extrair imagens, gráficos ou outras mídias de um documento PDF, mas pode extrair texto e retorná-lo como uma string Python.

```python
import PyPDF2

reader = PyPDF2.PdfReader('test.pdf')
for page in reader.pages:
    print(page.extract_text())
```

### Girar e sobrepor páginas

No código acima, lemos um documento PDF criando um objeto `PdfReader`. As páginas obtidas desse objeto são instâncias `PageObject`. Através de um `PageObject`, podemos girar as páginas no sentido horário ou anti-horário. O código é mostrado abaixo.

```python
import PyPDF2

reader = PyPDF2.PdfReader('XGBoost.pdf')
writer = PyPDF2.PdfWriter()

for no, page in enumerate(reader.pages):
    if no % 2 == 0:
        new_page = page.rotate(-90)
    else:
        new_page = page.rotate(90)
    writer.add_page(new_page)

with open('temp.pdf', 'wb') as file_obj:
    writer.write(file_obj)
```

### Criptografando arquivos PDF

Usando um objeto `PdfWriter` de `PyPDF2`, podemos criptografar um documento PDF. Se precisarmos atribuir uma senha comum a uma série de arquivos PDF, lidar com isso com um programa Python é muito conveniente.

```python
import PyPDF2

reader = PyPDF2.PdfReader('XGBoost.pdf')
writer = PyPDF2.PdfWriter()

for page in reader.pages:
    writer.add_page(page)

writer.encrypt('foobared')

with open('temp.pdf', 'wb') as file_obj:
    writer.write(file_obj)
```

### Adicionando marcas d'água em lote

O objeto `PageObject` mencionado acima também possui um método para mesclar páginas. Através desta operação, podemos adicionar facilmente uma marca d'água a um arquivo PDF. Por exemplo, se quisermos adicionar uma marca d'água ao arquivo `XGBoost.pdf` acima, podemos primeiro preparar um arquivo PDF que forneça a página da marca d'água, depois ler o `PageObject` que contém a marca d'água e, em seguida, percorrer cada página do arquivo `XGBoost.pdf`, obter o `PageObject` e, em seguida, usar o método para mesclar o página de marca d'água e a página original. O código é mostrado abaixo.

```python
import PyPDF2

reader1 = PyPDF2.PdfReader('XGBoost.pdf')
reader2 = PyPDF2.PdfReader('watermark.pdf')
writer = PyPDF2.PdfWriter()
watermark_page = reader2.pages[0]

for page in reader1.pages:
    page.merge_page(watermark_page)
    writer.add_page(page)

with open('temp.pdf', 'wb') as file_obj:
    writer.write(file_obj)
```

Se desejar, você pode até usar marcas d'água diferentes para páginas pares e ímpares. Você pode pensar em como fazer isso sozinho.

### Criando arquivos PDF

A criação de documentos PDF requer suporte da biblioteca de terceiros `reportlab`. Instale assim.

```bash
pip install reportlab
```

O exemplo a seguir demonstra o uso básico de `reportlab`.

```python
from reportlab.lib.pagesizes import A4
from reportlab.pdfbase import pdfmetrics
from reportlab.pdfbase.ttfonts import TTFont
from reportlab.pdfgen import canvas

pdf_canvas = canvas.Canvas('resources/demo.pdf', pagesize=A4)
width, height = A4

# Draw an image.
image = canvas.ImageReader('resources/guido.jpg')
pdf_canvas.drawImage(image, 20, height - 395, 250, 375)

# Show the current page.
pdf_canvas.showPage()

# Register font files.
pdfmetrics.registerFont(TTFont('Font1', 'resources/fonts/Vera.ttf'))
pdfmetrics.registerFont(TTFont('Font2', 'resources/fonts/Qingcao Stone Body.ttf'))

# Draw text.
pdf_canvas.setFont('Font2', 40)
pdf_canvas.setFillColorRGB(0.9, 0.5, 0.3, 1)
pdf_canvas.drawString(width // 2 - 120, height // 2, 'Greetings, world!')
pdf_canvas.setFont('Font1', 40)
pdf_canvas.setFillColorRGB(0, 1, 0, 0.5)
pdf_canvas.rotate(18)
pdf_canvas.drawString(250, 250, 'hello, world!')

# Save the PDF.
pdf_canvas.save()
```
