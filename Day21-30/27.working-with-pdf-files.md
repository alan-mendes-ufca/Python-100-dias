## Working with PDF Files in Python

PDF is the short form of Portable Document Format. This kind of file usually uses `.pdf` as the extension. In daily development work, the easiest tasks to meet are reading text content from PDF and generating PDF documents with existing content.

### Extracting Text from PDFs

In Python, we can use the third-party library `PyPDF2` to read PDF files. Install it with the following command.

```bash
pip install PyPDF2
```

`PyPDF2` cannot extract images, charts, or other media from a PDF document, but it can extract text and return it as a Python string.

```python
import PyPDF2

reader = PyPDF2.PdfReader('test.pdf')
for page in reader.pages:
    print(page.extract_text())
```

### Rotating and Overlaying Pages

In the code above, we read a PDF document by creating a `PdfReader` object. The pages obtained from that object are `PageObject` instances. Through a `PageObject`, we can rotate pages clockwise or counterclockwise. The code is shown below.

```python
import PyPDF2

reader = PyPDF2.PdfReader('XGBoost.pdf')
writer = PyPDF2.PdfWriter()

for no, page in enumerate(reader.pages):
    if no % 2 == 0:
        new_page = page.rotate(-90)
    else:
        new_page = page.rotate(90)
    writer.add_page(new_page)

with open('temp.pdf', 'wb') as file_obj:
    writer.write(file_obj)
```

### Encrypting PDF Files

Using a `PdfWriter` object from `PyPDF2`, we can encrypt a PDF document. If we need to assign a common password to a series of PDF files, handling it with a Python program is very convenient.

```python
import PyPDF2

reader = PyPDF2.PdfReader('XGBoost.pdf')
writer = PyPDF2.PdfWriter()

for page in reader.pages:
    writer.add_page(page)

writer.encrypt('foobared')

with open('temp.pdf', 'wb') as file_obj:
    writer.write(file_obj)
```

### Adding Watermarks in Batch

The `PageObject` object mentioned above also has a method for merging pages. Through this operation, we can easily add a watermark to a PDF file. For example, if we want to add a watermark to the `XGBoost.pdf` file above, we can first prepare a PDF file that provides the watermark page, then read out the `PageObject` that contains the watermark, and then loop through each page of the `XGBoost.pdf` file, get the `PageObject`, and then use the method to merge the watermark page and the original page. The code is shown below.

```python
import PyPDF2

reader1 = PyPDF2.PdfReader('XGBoost.pdf')
reader2 = PyPDF2.PdfReader('watermark.pdf')
writer = PyPDF2.PdfWriter()
watermark_page = reader2.pages[0]

for page in reader1.pages:
    page.merge_page(watermark_page)
    writer.add_page(page)

with open('temp.pdf', 'wb') as file_obj:
    writer.write(file_obj)
```

If you like, you can even use different watermarks for odd and even pages. You can think through how to do that on your own.

### Creating PDF Files

Creating PDF documents requires support from the third-party library `reportlab`. Install it like this.

```bash
pip install reportlab
```

The following example demonstrates basic usage of `reportlab`.

```python
from reportlab.lib.pagesizes import A4
from reportlab.pdfbase import pdfmetrics
from reportlab.pdfbase.ttfonts import TTFont
from reportlab.pdfgen import canvas

pdf_canvas = canvas.Canvas('resources/demo.pdf', pagesize=A4)
width, height = A4

# Draw an image.
image = canvas.ImageReader('resources/guido.jpg')
pdf_canvas.drawImage(image, 20, height - 395, 250, 375)

# Show the current page.
pdf_canvas.showPage()

# Register font files.
pdfmetrics.registerFont(TTFont('Font1', 'resources/fonts/Vera.ttf'))
pdfmetrics.registerFont(TTFont('Font2', 'resources/fonts/Qingcao Stone Body.ttf'))

# Draw text.
pdf_canvas.setFont('Font2', 40)
pdf_canvas.setFillColorRGB(0.9, 0.5, 0.3, 1)
pdf_canvas.drawString(width // 2 - 120, height // 2, 'Greetings, world!')
pdf_canvas.setFont('Font1', 40)
pdf_canvas.setFillColorRGB(0, 1, 0, 0.5)
pdf_canvas.rotate(18)
pdf_canvas.drawString(250, 250, 'hello, world!')

# Save the PDF.
pdf_canvas.save()
```
