## Aplicativos NumPy - 3

### Operações em matrizes

O que é mais conveniente no NumPy é que quando precisamos realizar operações em elementos do array, não precisamos escrever código de loop para percorrer cada elemento. Todas as operações são automaticamente **vetorizadas**. Simplificando, as operações matemáticas e funções matemáticas no NumPy são aplicadas automaticamente a cada elemento do array.

#### Operações entre um array e um escalar

Matrizes NumPy podem fazer adição, subtração, multiplicação, divisão, módulo, potência e outras operações com um número. A operação correspondente será aplicada a cada elemento do array, conforme mostrado abaixo.

```python
array1 = np.arange(1, 10)
print(array1 + 10)
print(array1 * 10)
```

Saída:

```
[11 12 13 14 15 16 17 18 19]
[10 20 30 40 50 60 70 80 90]
```

Além das operações acima, as operações de comparação também não são problema. Já os conhecemos antes quando falamos sobre indexação booleana.

```python
print(array1 > 5)
print(array1 % 2 == 0)
```

Saída:

```
[False False False False False  True  True  True  True]
[False  True False  True False  True False  True False]
```

#### Operações entre arrays

Os arrays NumPy também podem realizar operações aritméticas e de comparação com outros arrays. As operações são aplicadas aos elementos correspondentes das duas matrizes. Isso requer que as duas matrizes tenham o mesmo formato, conforme mostrado abaixo.

```python
array2 = np.array([1, 1, 1, 2, 2, 2, 3, 3, 3])
print(array1 + array2)
print(array1 * array2)
print(array1 ** array2)
```

Saída:

```
[ 2  3  4  6  7  8 10 11 12]
[ 1  2  3  8 10 12 21 24 27]
[  1   2   3  16  25  36 343 512 729]
```

```python
print(array1 > array2)
print(array1 % array2 == 0)
```

Saída:

```
[False  True  True  True  True  True  True  True  True]
[ True  True  True  True False  True False False  True]
```

#### Funções unárias universais

O argumento de uma função unária universal em NumPy é um objeto array. A função processará o array elemento por elemento. Por exemplo, a função `sqrt` calcula a raiz quadrada de cada elemento da matriz, e a função `log2` calcula o logaritmo de base 2 de cada elemento da matriz, conforme mostrado abaixo.

```python
print(np.sqrt(array1))
print(np.log2(array1))
```

```

 2.80735492 3.         3.169925  ]
```

#### Funções binárias universais

Os argumentos de uma função binária universal em NumPy são dois objetos array. A função realizará operações nos elementos correspondentes nas duas matrizes. Por exemplo, a função `maximum` encontra o valor maior de cada par de elementos correspondentes, e a função `power` eleva os elementos correspondentes em uma matriz às potências dadas pelos elementos correspondentes na outra matriz, conforme mostrado abaixo.

```python
array3 = np.array([[4, 5, 6], [7, 8, 9]])
array4 = np.array([[1, 2, 3], [3, 2, 1]])
print(np.maximum(array3, array4))
print(np.power(array3, array4))
```

Saída:

```
[[4 5 6]
 [7 8 9]]
[[  4  25 216]
 [343  64   9]]
```

| `floor_divide(x, y)` / `mod(x, y)` | Divisão de piso/módulo |
| `allclose(x, y)` | Verifique se os elementos das matrizes `x` e `y` são quase iguais |

#### Radiodifusão

Nos exemplos de operações de array acima, os dois arrays tinham exatamente o mesmo formato. Agora vamos estudar se duas matrizes com formatos diferentes podem realizar operações binárias diretamente ou usar funções binárias universais. Por favor, veja os exemplos abaixo.

```python
array5 = np.array([[0, 0, 0], [1, 1, 1], [2, 2, 2], [3, 3, 3]])
array6 = np.array([1, 2, 3])
array5 + array6
```

Saída:

```
array([[1, 2, 3],
       [2, 3, 4],
       [3, 4, 5],
       [4, 5, 6]])
```

```python
array7 = np.array([[1], [2], [3], [4]])
array5 + array7
```

Saída:

```
array([[1, 1, 1],
       [3, 3, 3],
       [5, 5, 5],
       [7, 7, 7]])
```

A partir dos exemplos acima, podemos ver que arrays com formatos diferentes ainda têm a chance de realizar operações binárias, mas isso não significa que arrays de qualquer formato possam fazê-lo. Simplificando, a transmissão pode ser acionada apenas quando as dimensões finais dos dois arrays são iguais ou quando são diferentes, mas um deles é `1`. Por meio da transmissão, o NumPy transforma os dois arrays, que originalmente tinham formatos diferentes, em arrays com o mesmo formato, e então as operações binárias podem ser executadas. As chamadas dimensões finais significam as partes correspondentes das formas da matriz quando vistas de trás para frente.

Quando vista de trás para frente, a peça correspondente é `3`, portanto as dimensões finais são as mesmas e a transmissão pode ser usada. A segunda matriz se transmitirá ao longo do eixo ausente para que as duas matrizes finalmente tenham formas consistentes.

Assim, a segunda matriz pode se transmitir ao longo do eixo onde o valor é `1` e, finalmente, as duas matrizes tornam-se consistentes em forma.

| `sort` | Retornar uma cópia classificada do array |

```python

np.vstack((array8, array9))
```

```
       [2, 2, 2],
       [3, 3, 3],
       [4, 4, 4],
```

```python
np.concatenate((array8, array9))
```

```
array([[1, 1, 1],
```

**Extraia e processe elementos**.

```python
np.extract(array1 % 2 != 0, array1)
```

Saída:

```
array([1, 3, 5, 7, 9])
```

> **Nota**: A operação de `extract` acima é equivalente à indexação booleana que mencionamos anteriormente.

```python
np.select([array1 <= 3, array1 >= 7], [array1 * 10, array1 ** 2])
```

Saída:

```
array([10, 20, 30,  0,  0,  0, 49, 64, 81])
```

> **Nota**: Na função `select` acima, o primeiro argumento fornece duas condições. Os elementos que satisfazem a primeira condição são multiplicados por 10 e os elementos que satisfazem a segunda condição são elevados ao quadrado. Os elementos que não satisfazem nenhuma das condições são processados ​​como `0`.

```python
np.where(array1 <= 5, array1 * 10, array1 ** 2)
```

Saída:

```
array([10, 20, 30, 40, 50, 36, 49, 64, 81])
```

> **Nota**: Na função `where` acima, o primeiro argumento fornece a condição. Os elementos que satisfazem a condição são multiplicados por 10 e os elementos que não satisfazem a condição são elevados ao quadrado.

**Redimensionar matrizes**.

```python
np.resize(array1, (5, 3))
```

Saída:

```

       [4, 5, 6],
```

> **Dica**: `array1` era originalmente uma matriz unidimensional com 9 elementos. Com a função `resize`, ele se torna um array bidimensional com 5 linhas e 3 colunas, portanto necessita de 15 elementos. Os elementos ausentes são preenchidos reutilizando elementos do array original.

```python
np.resize(array5, (2, 4))
```

Saída:

```
array([[0, 0, 0, 1],
       [1, 1, 2, 2]])
```

**Substitua os elementos da matriz**.

```python
np.put(array1, [0, 1, -1, 3, 5], [100, 200])
array1
```

Saída:

```
array([100, 200,   3, 200,   5, 100,   7,   8, 100])
```

> **Nota**: Na função `put` acima, o segundo argumento fornece os índices dos elementos que serão substituídos, mas existem apenas dois valores de substituição, `100` e `200`, portanto, esses dois valores são usados ​​repetidamente. Portanto, os elementos nos índices `0`, `1`, `-1`, `3` e `5` são substituídos por `100`, `200`, `100`, `200` e `100`.

```python
np.place(array1, array1 > 5, [1, 2, 3])
array1
```

Saída:

```
array([1, 2, 3, 3, 5, 1, 2, 3, 1])
```

> **Aviso**: `put` e `place` não retornam um novo objeto de matriz. Eles substituem diretamente os valores na matriz original.
