## pandas em profundidade - 2

Se usarmos pandas para análise de dados, então `DataFrame` é definitivamente o tipo usado mais amplamente. Pode ser usado para armazenar e processar dados bidimensionais heterogêneos. O chamado "heterogêneo" aqui significa que os tipos de dados de cada coluna em um `DataFrame` não precisam ser iguais. Isso também é o que o torna diferente de um array NumPy bidimensional. `DataFrame` fornece propriedades e métodos muito ricos para nos ajudar a remodelar, limpar, pré-processar, dinamizar e apresentar dados.

### Criando objetos `DataFrame`

#### Crie um `DataFrame` a partir de uma matriz bidimensional

```python
scores = np.random.randint(60, 101, (5, 3))
courses = ['Chinese', 'Math', 'English']
stu_ids = np.arange(1001, 1006)
df1 = pd.DataFrame(data=scores, columns=courses, index=stu_ids)
df1
```

Saída:

```
      Chinese  Math  English
1001       69    80       79
1002       71    60      100
1003       94    81       93
1004       88    88       67
1005       82    66       60
```

#### Crie um `DataFrame` a partir de um dicionário

```python
scores = {
    'Chinese': [62, 72, 93, 88, 93],
    'Math': [95, 65, 86, 66, 87],
    'English': [66, 75, 82, 69, 82],
}

df2
```

#### Leia um arquivo CSV e crie um `DataFrame`

Podemos usar a função `read_csv` no módulo `pandas` para ler um arquivo CSV. A função `read_csv` possui muitos parâmetros. Abaixo estão vários importantes.

- `sep` / `delimiter`: separador, o padrão é `,`
- `header`: posição da linha do cabeçalho, o padrão é `infer`
- `index_col`: qual coluna é usada como índice de linha
- `usecols`: colunas que precisam ser carregadas, que podem ser especificadas por posição ou por nome
- `true_values`/`false_values`: quais valores são tratados como booleanos `True`/`False`
- `skiprows`: linhas a serem ignoradas, especificadas por número de linha, índice ou função
- `skipfooter`: número de linhas a pular na parte inferior
- `nrows`: número de linhas a serem lidas
- `na_values`: quais valores são tratados como valores nulos
- `iterator`: se definida como `True`, a função retorna um objeto iterador
- `chunksize`: usado em conjunto com o parâmetro anterior para definir a quantidade de dados buscados a cada vez

```python
df3 = pd.read_csv('data/2018-beijing-residence-points-data.csv', index_col='id')
df3
```

> **Dica**: o arquivo CSV no código acima é carregado por meio de um caminho relativo. Em outras palavras, há uma pasta chamada `data` no caminho de trabalho atual e `2018-beijing-residence-points-data.csv` está dentro dessa pasta. Se você usa Windows, também é recomendado usar `/` em vez de `\` para separadores de caminho. Se você quiser usar `\`, é recomendado adicionar `r` antes da string e usar uma string bruta para evitar caracteres de escape.

#### Leia uma tabela de banco de dados relacional e crie um `DataFrame`

A função `read_sql` no módulo `pandas` pode ler dados de um banco de dados através de SQL e criar um `DataFrame`. O segundo parâmetro desta função representa o banco de dados ao qual se conectar. Para MySQL, podemos criar uma conexão de banco de dados por meio de `pymysql` ou `mysqlclient` e então usar essa conexão como segundo parâmetro de `read_sql`.

```python
import pymysql

conn = pymysql.connect(

    user='guest', password='Guest.618',

df5 = pd.read_sql('select * from tb_emp', conn, index_col='eno')
df5
```

### Propriedades e métodos básicos

Antes de falar sobre as propriedades e métodos de `DataFrame`, vamos primeiro ler três tabelas do banco de dados `hrs` mencionado anteriormente e criar três objetos `DataFrame`.

```python
from sqlalchemy import create_engine

engine = create_engine('mysql+pymysql://guest:Guest.618@101.42.16.8:3306/hrs')
dept_df = pd.read_sql_table('tb_dept', engine, index_col='dno')
emp_df = pd.read_sql_table('tb_emp', engine, index_col='eno')
emp2_df = pd.read_sql_table('tb_emp2', engine, index_col='eno')
```

Propriedades importantes de um `DataFrame` são mostradas abaixo.

| Propriedade | Descrição |
| -------------- | ----------- |
| `at` / `iat` | Obtenha um único valor em um `DataFrame` através de rótulos |
| `columns` | O índice da coluna de um `DataFrame` |
| `dtypes` | O tipo de dados de cada coluna em um `DataFrame` |
| `empty` | Se o `DataFrame` está vazio |
| `loc` / `iloc` | Obtenha um grupo de valores em um `DataFrame` através de rótulos |
| `ndim` | O número de dimensões do `DataFrame` |
| `shape` | A forma do `DataFrame` |
| `size` | O número de elementos no `DataFrame` |
| `values` | A matriz bidimensional correspondente aos dados no `DataFrame` |

### Operando com Dados

#### Indexação e fatiamento

Se quisermos obter uma coluna de `DataFrame`, por exemplo, a coluna `ename` de `emp_df`, podemos usar uma das seguintes maneiras:

```python
emp_df.ename
```

ou

```python
emp_df['ename']
```

Ao executar o código acima, podemos ver que o que obtemos é um objeto `Series`. Na verdade, um `DataFrame` é o resultado da união de vários objetos `Series`.

Se quisermos obter uma linha de `DataFrame`, podemos usar a indexação inteira ou o índice que nós mesmos definimos. Por exemplo, para retirar o funcionário cujo número é `2056`, o código é o seguinte:

```python
emp_df.iloc[1]
```

ou

```python
emp_df.loc[2056]
```

Se pegarmos uma única linha ou coluna de um `DataFrame`, o que obteremos é um objeto `Series`. Claro, também podemos usar uma indexação sofisticada para obter várias linhas ou colunas, e o resultado ainda será `DataFrame`.

Obtenha várias colunas:

```python
emp_df[['ename', 'job']]
```

Obtenha várias linhas:

```python
emp_df.loc[[2056, 7800, 3344]]
```

Se quisermos obter ou modificar os dados em uma célula de `DataFrame`, precisamos especificar o índice da linha e o índice da coluna. Por exemplo, se quisermos obter as informações do cargo do funcionário `2056`, podemos escrever:

```python
emp_df['job'][2056]
```

ou

```python
emp_df.loc[2056]['job']
```

ou

```python
emp_df.loc[2056, 'job']
```

Recomendamos a terceira abordagem porque ela realiza a indexação apenas uma vez. Se quisermos modificar o cargo deste funcionário para `Architect`, podemos escrever:

```python
emp_df.loc[2056, 'job'] = 'Architect'
```

Claro, também podemos obter várias linhas e colunas fatiando.

```python
emp_df.loc[2056:3344]
```

#### Filtragem de dados

Tal como acontece com `ndarray` e `Series`, podemos usar a indexação booleana para filtrar dados em um `DataFrame`. Por exemplo, se quisermos filtrar funcionários cujo salário mensal seja superior a `3500`, podemos fazer o seguinte:

```python
emp_df[emp_df.sal > 3500]
```

Saída:

```
          ename         job     mgr   sal    comm  dno
eno                                                   
2056   Qiao Feng     Analyst  7800.0  5000  1500.0   20
3251   Zhang Cuishan Programmer 2056.0 4000    NaN   20
5566   Song Yuanqiao Accountant 7800.0 4000 1000.0 10
7800   Zhang Sanfeng President    NaN  9000 1200.0 20
```

Claro, também podemos combinar múltiplas condições. Por exemplo, para filtrar funcionários cujo salário mensal seja superior a `3500` e cujo número de departamento seja `20`, podemos escrever:

```python
emp_df[(emp_df.sal > 3500) & (emp_df.dno == 20)]
```

Saída:

```
          ename         job     mgr   sal    comm  dno
eno                                                   
2056   Qiao Feng     Analyst  7800.0  5000  1500.0   20
3251   Zhang Cuishan Programmer 2056.0 4000    NaN   20
7800   Zhang Sanfeng President    NaN  9000 1200.0 20
```

Além da indexação booleana, o método `query` de um `DataFrame` também pode fazer filtragem de dados. O argumento de `query` é uma string, representando a expressão usada para filtrar os dados, e isso geralmente está mais de acordo com o que os programadores Python gostam.

```python
emp_df.query('sal > 3500 and dno == 20')
```

Saída:

```

7800   Zhang Sanfeng President    NaN  9000 1200.0 20
```
