## Learning Programming with Canvas

This note explains how an interactive canvas-style coding workspace can help with programming practice.

### Common Use Cases

1. Real-time code editing and refinement
2. Learning syntax and programming concepts from examples
3. Understanding algorithms step by step
4. Experimenting with alternative implementations
5. Combining code and explanation in one place

The core idea is that an interactive editing surface makes it easier to:

- iterate on code quickly
- ask focused questions
- compare versions
- understand why a change helps

For learning, the most useful workflow is usually:

1. write a small example
2. ask for an explanation
3. refine the code
4. repeat until the logic is clear
