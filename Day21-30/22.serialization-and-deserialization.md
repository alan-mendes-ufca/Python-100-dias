## Serialização e desserialização de objetos

### Uma visão geral do JSON

Pela explicação acima, já sabemos como salvar dados de texto e dados binários em arquivos. Então aqui fica outra questão: se quisermos salvar os dados de uma lista ou de um dicionário em um arquivo, o que devemos fazer? Em Python, podemos salvar os dados do programa no formato **JSON**. JSON é a forma abreviada de "JavaScript Object Notation". Era originalmente uma sintaxe literal para criar objetos na linguagem JavaScript e agora tem sido amplamente utilizada na troca de dados entre linguagens e plataformas. O motivo da utilização do JSON é muito simples, pois possui uma estrutura compacta e é texto simples. Qualquer sistema operacional e linguagem de programação pode lidar com texto simples, e este é o pré-requisito para **troca de dados entre linguagens e plataformas**. Atualmente, JSON basicamente substituiu XML, eXtensible Markup Language, como o **padrão de fato para troca de dados entre sistemas heterogêneos**. Você pode encontrar mais informações sobre JSON no [site oficial do JSON](https://www.json.org/json-en.html). Este site também fornece ferramentas ou bibliotecas de terceiros que cada linguagem pode usar para lidar com o formato de dados JSON.

```javascript
{
    name: "Luo Hao",
    age: 40,
    friends: ["Wang Dahuang", "Bai Yuanfang"],
    cars: [
        {"brand": "BMW", "max_speed": 240},
        {"brand": "Benz", "max_speed": 280},
        {"brand": "Audi", "max_speed": 280}
    ]
}
```

O acima é um exemplo simples de JSON. Você já deve ter notado que ele é muito semelhante a um dicionário em Python e também suporta estrutura aninhada, assim como o valor em um dicionário Python pode ser outro dicionário. Podemos tentar digitar o código abaixo no console do navegador. Para o Chrome, por exemplo, você pode encontrar o submenu “Ferramentas do desenvolvedor” através do menu “Mais ferramentas” e, em seguida, abrir o console do navegador. O console do navegador fornece um ambiente interativo para execução de código JavaScript, semelhante ao ambiente interativo do Python. O código abaixo nos ajudará a criar um objeto JavaScript e o atribuímos a uma variável chamada `obj`.

```javascript
let obj = {
    name: "Luo Hao",
    age: 40,
    friends: ["Wang Dahuang", "Bai Yuanfang"],
    cars: [
        {"brand": "BMW", "max_speed": 240},
        {"brand": "Benz", "max_speed": 280},
        {"brand": "Audi", "max_speed": 280}
    ]
}
```

O `obj` acima é um objeto JavaScript. Podemos obter o valor correspondente a `name` de duas formas, `obj.name` ou `obj["name"]`, conforme mostrado abaixo. Você pode notar que a forma como `obj["name"]` obtém os dados é exatamente a mesma que a operação de índice de obtenção de um valor por chave de um dicionário Python. Em Python, o módulo denominado `json` também fornece suporte para conversão bidirecional entre dicionários e JSON.

Também é fácil encontrar o relacionamento correspondente entre os tipos de dados usados em JSON e os tipos de dados usados em Python. Você pode ver as duas tabelas abaixo.

Tabela 1: Tipos de dados (valores) JavaScript e seus tipos de dados (valores) Python correspondentes

| JSON | Pitão |
| ------------ | ------------ |
| `object` | `dict` |
| `array` | `list` |
| `string` | `str` |
| `number` | `int` / `float` |
| `number` (real) | `float` |
| `boolean` (`true` / `false`) | `bool` (`True` / `False`) |
| `null` | `None` |

Tabela 2: Tipos de dados (valores) Python e seus tipos de dados (valores) JavaScript correspondentes

| Pitão | JSON |
| --------------------------- | ---------------------------- |
| `dict` | `object` |
| `list` / `tuple` | `array` |
| `str` | `string` |
| `int` / `float` | `number` |
| `bool` (`True` / `False`) | `boolean` (`true` / `false`) |
| `None` | `null` |

### Lendo e gravando dados JSON

Em Python, se quisermos transformar um dicionário no formato JSON, ou seja, fazê-lo existir como uma string, podemos utilizar a função `dumps` do módulo `json`, conforme mostrado abaixo.

```python
import json

my_dict = {
    'name': 'Luo Hao',
    'age': 40,
    'friends': ['Wang Dahuang', 'Bai Yuanfang'],
    'cars': [
        {'brand': 'BMW', 'max_speed': 240},
        {'brand': 'Audi', 'max_speed': 280},
        {'brand': 'Benz', 'max_speed': 280}
    ]
}
print(json.dumps(my_dict))
```

Quando o código acima é executado, a saída é mostrada abaixo. Observe que os caracteres chineses nos dados originais são exibidos usando escapes Unicode.

```json
{"name": "\u9a86\u660a", "age": 40, "friends": ["\u738b\u5927\u9524", "\u767d\u5143\u82b3"], "cars": [{"brand": "BMW", "max_speed": 240}, {"brand": "Audi", "max_speed": 280}, {"brand": "Benz", "max_speed": 280}]}
```

Se quisermos transformar um dicionário no formato JSON e gravá-lo em um arquivo de texto, precisamos apenas substituir `dumps` por `dump` e passar um objeto de arquivo, conforme mostrado abaixo.

```python
import json

my_dict = {
    'name': 'Luo Hao',
    'age': 40,
    'friends': ['Wang Dahuang', 'Bai Yuanfang'],
    'cars': [
        {'brand': 'BMW', 'max_speed': 240},
        {'brand': 'Audi', 'max_speed': 280},
        {'brand': 'Benz', 'max_speed': 280}
    ]
}
with open('data.json', 'w') as file:
    json.dump(my_dict, file)
```

Após executar o código acima, um arquivo chamado `data.json` é criado e seu conteúdo é igual à saída mostrada anteriormente.

Existem quatro funções especialmente importantes no módulo `json`:

- `dump`: serializar um objeto Python em um arquivo no formato JSON
- `dumps`: transforme um objeto Python em uma string formatada em JSON
- `load`: desserializar dados JSON de um arquivo em um objeto
- `loads`: desserializar o conteúdo de uma string em um objeto Python

Dois conceitos aparecem aqui. Uma é a serialização e a outra é a desserialização. A explicação na Wikipedia é que a serialização é a conversão de uma estrutura de dados ou estado de objeto em um formato que pode ser armazenado ou transmitido, para que possa ser restaurado ao estado original quando necessário. Além disso, quando os bytes são obtidos novamente dos dados serializados, esses bytes podem ser usados para produzir uma cópia do objeto original. A ação oposta a esse processo, ou seja, extrair uma estrutura de dados de uma série de bytes, é a desserialização.

Podemos usar o código a seguir para ler o arquivo `data.json` criado acima e restaurar os dados JSON em um dicionário Python.

```python
import json

with open('data.json', 'r') as file:
    my_dict = json.load(file)
    print(type(my_dict))
    print(my_dict)
```

### A ferramenta de gerenciamento de pacotes `pip`

O módulo `json` na biblioteca padrão Python não é muito ideal em desempenho ao fazer serialização e desserialização de dados. Para resolver este problema, podemos usar a biblioteca de terceiros `ujson` para substituir `json`. A chamada biblioteca de terceiros significa um módulo Python que não é desenvolvido e usado dentro da empresa e também não faz parte da biblioteca padrão oficial. Esses módulos geralmente são desenvolvidos por outras empresas, organizações ou indivíduos, por isso são chamados de bibliotecas de terceiros. Embora a biblioteca padrão do Python já tenha fornecido muitos módulos para tornar nosso desenvolvimento conveniente, para uma linguagem poderosa, seu ecossistema também deve ser muito próspero.

Quando você instalou o interpretador Python anteriormente, a opção de instalar `pip` foi marcada por padrão. Você pode verificar se `pip` está disponível executando `pip --version` em um terminal ou prompt de comando. Pip é a ferramenta de gerenciamento de pacotes do Python. Através do pip, podemos pesquisar, instalar, desinstalar e atualizar bibliotecas ou ferramentas de terceiros. Por exemplo, se quisermos instalar `ujson` como substituto do módulo `json`, podemos executar o seguinte comando.

```bash
pip install ujson
```

Por padrão, pip acessa `https://pypi.org/simple/` para obter dados sobre pacotes de terceiros. Mas para usuários na China, a velocidade de visita a este site geralmente não é muito boa, então os usuários na China podem usar espelhos fornecidos por Douban ou pela Universidade Tsinghua em vez da fonte de download padrão e, em seguida, instalar pacotes de terceiros normalmente.

```bash
pip config set global.index-url https://pypi.doubanio.com/simple
pip install ujson
```

Podemos usar `pip search` para procurar pacotes por nome, `pip list` para ver pacotes que já estão instalados, `pip install -U` ou `pip install --upgrade` para atualizar um pacote e `pip uninstall` para remover um.

### Obtendo dados por meio de APIs de rede

Se quisermos exibir informações como clima, condições das estradas e voos em nosso próprio programa, não temos a capacidade de fornecer essas informações por nós mesmos, portanto, devemos usar serviços de dados de rede. Atualmente, a maioria dos serviços de dados de rede, ou APIs de rede, fornecem dados no formato JSON baseados em HTTP ou HTTPS. Podemos usar um programa Python para enviar uma solicitação HTTP para um URL especificado, e esse URL é a chamada API de rede. Se a solicitação for bem-sucedida, ela retornará uma resposta HTTP e, no corpo da resposta HTTP, haverá os dados no formato JSON de que precisamos. Para conhecimento relacionado ao HTTP, você pode ler o artigo de Ruan Yifeng *Introdução ao protocolo HTTP*.

Existem muitos sites na China que fornecem serviços de API de rede, como Juhe Data e Baidu AI Cloud. Esses sites fornecem interfaces de dados gratuitas e pagas. Fora da China, o site [APIs.io](http://apis.io/) também oferece serviços semelhantes. Os leitores interessados podem estudá-los por conta própria. O exemplo abaixo mostra como usar a biblioteca de terceiros [`requests`](http://docs.python-requests.org/zh_CN/latest/), uma biblioteca de terceiros para acessar recursos de rede por meio de HTTP, para acessar uma API de rede, obter notícias nacionais e exibir o título e o link da notícia. Neste exemplo, usamos a interface de dados de notícias nacionais fornecida por [Tian API](https://www.tianapi.com/). Sua credencial de identidade, a chave API, precisa ser solicitada por meio de cadastro no site. Depois de registrar uma conta no site Tian API, uma chave API será atribuída automaticamente, mas para acessar a interface e obter dados, você ainda precisa vincular e verificar seu e-mail ou telefone celular e, em seguida, solicitar a interface que deseja usar.

<img src="res/20210820151134.png" alt="image-20210820151134034" width="100%">

Para que o Python acesse a rede por meio de uma URL, recomendamos a biblioteca de terceiros `requests`. É simples e poderoso, mas precisa ser instalado por você mesmo.

```bash
pip install requests
```

Busque notícias nacionais e exiba o título e o link.

```python
import requests

resp = requests.get('http://api.tianapi.com/guonei/?key=APIKey&num=10')
if resp.status_code == 200:
    data_model = resp.json()
    for news in data_model['newslist']:
        print(news['title'])
        print(news['url'])
        print('-' * 60)
```

O código acima envia uma solicitação para a API de notícias domésticas da Tian API por meio da função `get` do módulo `requests`. Se não houver nenhum problema no processo de solicitação, a função `get` retornará um objeto `Response`. Através do atributo `status_code` deste objeto, podemos ver o código de status da resposta HTTP. Se você não entende, tudo bem. Você só precisa se preocupar com seu valor. Se o valor for igual a `200` ou outro valor começando com `2`, nossa solicitação foi bem-sucedida. Por meio do método `json()` do objeto `Response`, podemos transformar diretamente os dados retornados no formato JSON em um dicionário Python. Isto é muito conveniente. Os dados no formato JSON retornados pela interface de notícias domésticas da API Tian, parcialmente, são mostrados abaixo.

### Resumo

Em Python, além do módulo `json`, também podemos usar módulos como `pickle` e `shelve` para realizar serialização e desserialização. No entanto, esses dois módulos usam protocolos de serialização específicos do Python, portanto, os dados que eles produzem só podem ser reconhecidos pelo Python. Lidar com dados JSON é claramente uma habilidade que todo programador deve dominar, porque quer você esteja chamando uma API de rede ou fornecendo uma para outras pessoas, você precisa entender como trabalhar com JSON.
