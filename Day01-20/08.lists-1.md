## Estruturas de dados comuns: listas, parte 1

Antes de começarmos esta lição, vamos primeiro dar uma olhada em uma pequena tarefa de programação: lançar um dado 6.000 vezes e contar quantas vezes cada face aparece. Essa tarefa já deveria ser bem simples para todos. Podemos usar números aleatórios distribuídos uniformemente de 1 a 6 para simular o lançamento de um dado e, em seguida, usar 6 variáveis para registrar quantas vezes cada face aparece. Com base no que aprendemos anteriormente, acredito que todos podem escrever o código a seguir sem muitos problemas.

```python
"""
Roll a die 6000 times and count the occurrences of each face

Author: Luo Hao
Version: 1.0
"""
import random

f1 = 0
f2 = 0
f3 = 0
f4 = 0
f5 = 0
f6 = 0
for _ in range(6000):
    face = random.randrange(1, 7)
    if face == 1:
        f1 += 1
    elif face == 2:
        f2 += 1
    elif face == 3:
        f3 += 1
    elif face == 4:
        f4 += 1
    elif face == 5:
        f5 += 1
    else:
        f6 += 1
print(f'Face 1 appeared {f1} times')
print(f'Face 2 appeared {f2} times')
print(f'Face 3 appeared {f3} times')
print(f'Face 4 appeared {f4} times')
print(f'Face 5 appeared {f5} times')
print(f'Face 6 appeared {f6} times')
```

Provavelmente não preciso dizer muito sobre o quão "feio" é o código acima. O que é ainda pior é que se quiséssemos lançar dois dados, ou até mais dados, e depois contar quantas vezes cada resultado possível aparece, precisaríamos definir mais variáveis e escrever mais código de ramificação. Só de pensar nisso já parece desagradável. Neste ponto, uma pergunta já pode estar em sua mente: existe uma maneira de usar uma variável para armazenar vários dados e existe uma maneira de usar um tipo de código para trabalhar com vários dados? A resposta é sim. Em Python, podemos salvar e trabalhar com vários dados por meio de variáveis do tipo contêiner. O primeiro novo tipo de dados que apresentarei é a **lista**.

### Criando listas

Em Python, uma **lista é uma sequência de dados composta por uma série de elementos em uma ordem específica**. Isso significa que se definirmos uma variável do tipo lista, **podemos usá-la para armazenar vários dados**. Em Python, podemos usar a sintaxe literal `[]` para definir uma lista, e os elementos da lista são separados por vírgulas, conforme mostrado abaixo.

```python
items1 = [35, 12, 99, 68, 55, 35, 87]
items2 = ['Python', 'Java', 'Go', 'Kotlin']
items3 = [100, 12.3, 'Python', True]
print(items1)  # [35, 12, 99, 68, 55, 35, 87]
print(items2)  # ['Python', 'Java', 'Go', 'Kotlin']
print(items3)  # [100, 12.3, 'Python', True]
```

> **Observação**: Uma lista pode conter elementos duplicados, como `35` repetido em `items1`. Ele também pode conter valores de diferentes tipos, conforme mostrado em `items3`, que contém valores `int`, `float`, `str` e `bool`. No entanto, normalmente não recomendamos colocar muitos tipos diferentes em uma lista, porque é inconveniente trabalhar com eles.

Podemos usar a função `type` para ver o tipo de uma variável. Os leitores interessados podem verificar por si mesmos que tipo realmente é a variável `items1` acima. Como uma lista pode conter muitos elementos e é uma estrutura de dados do tipo contêiner, quando nomeamos variáveis do tipo lista, os nomes das variáveis geralmente usam palavras no plural.

Além disso, também podemos usar a função `list` integrada do Python para transformar outras sequências em listas. Mais precisamente, `list` não é uma função comum, mas um construtor para criar objetos de lista. As lições posteriores apresentarão conceitos como objetos e construtores.

```python
items4 = list(range(1, 10))
items5 = list('hello')
print(items4)  # [1, 2, 3, 4, 5, 6, 7, 8, 9]
print(items5)  # ['h', 'e', 'l', 'l', 'o']
```

> **Observação**: `range(1, 10)` produz os números inteiros de `1` a `9`. Passar essa sequência para `list` cria uma lista composta de números inteiros de `1` a `9`. Uma string também é uma sequência, portanto `list('hello')` cria uma lista cujos elementos são os caracteres da string `hello`.

### Operações em listas

Podemos usar o operador `+` para unir duas listas. Esta operação conecta os elementos das duas listas e os coloca em uma lista, conforme mostrado abaixo.

```python
items5 = [35, 12, 99, 45, 66]
items6 = [45, 58, 29]
items7 = ['Python', 'Java', 'JavaScript']
print(items5 + items6)  # [35, 12, 99, 45, 66, 45, 58, 29]
print(items6 + items7)  # [45, 58, 29, 'Python', 'Java', 'JavaScript']
items5 += items6
print(items5)  # [35, 12, 99, 45, 66, 45, 58, 29]
```

Podemos usar o operador `*` para repetir uma lista. O operador `*` repete os elementos da lista um determinado número de vezes. Adicione as duas linhas a seguir ao código acima.

```python
print(items6 * 3)  # [45, 58, 29, 45, 58, 29, 45, 58, 29]
print(items7 * 2)  # ['Python', 'Java', 'JavaScript', 'Python', 'Java', 'JavaScript']
```

Podemos usar `in` ou `not in` para verificar se um elemento está em uma lista. Adicione as seguintes linhas ao código acima.

```python
print(29 in items6)  # True
print(99 in items6)  # False
print('C++' not in items7)     # True
print('Python' not in items7)  # False
```

Como uma lista possui muitos elementos, e esses elementos são colocados na lista em uma ordem específica, quando queremos trabalhar com um elemento da lista, podemos usar o operador `[]` e especificar a posição do elemento dentro de `[]` para acessá-lo. Esta operação é chamada de indexação. A posição dentro de `[]` pode ser um número inteiro de `0` a `N - 1` ou um número inteiro de `-1` a `-N`. Eles são chamados de índices diretos e índices reversos, onde `N` significa o número de elementos na lista. Para índices diretos, `[0]` acessa o primeiro elemento da lista e `[N - 1]` acessa o último elemento. Para índices reversos, `[-1]` acessa o último elemento da lista e `[-N]` acessa o primeiro elemento, conforme mostrado abaixo.

```python
items8 = ['apple', 'waxberry', 'pitaya', 'peach', 'watermelon']
print(items8[0])   # apple
print(items8[2])   # pitaya
print(items8[4])   # watermelon
items8[2] = 'durian'
print(items8)      # ['apple', 'waxberry', 'durian', 'peach', 'watermelon']
print(items8[-5])  # apple
print(items8[-4])  # waxberry
print(items8[-1])  # watermelon
items8[-4] = 'strawberry'
print(items8)      # ['apple', 'strawberry', 'durian', 'peach', 'watermelon']
```

Ao usar a indexação, evite sair do intervalo. Para `items8` acima, se acessarmos `items8[5]` ou `items8[-6]`, ele gerará um `IndexError` e travará o programa. A mensagem de erro correspondente é `list index out of range`. Como a lista `items8` possui apenas cinco elementos, os índices diretos válidos são `0` a `4` e os índices reversos válidos são `-1` a `-5`.

Se quiser acessar muitos elementos da lista ao mesmo tempo, você pode usar o fatiamento. Uma operação de fatiamento tem o formato `[start:end:stride]`. Aqui, `start` significa a posição inicial, `end` significa a posição final e o elemento na posição final em si não está incluído, enquanto `stride` significa a etapa. Simplificando, se o primeiro elemento que acessamos está na posição `start`, então o segundo elemento está na posição `start + stride` e, claro, `start + stride` ainda deve permanecer dentro do intervalo de fatia. Adicione as seguintes instruções ao código acima para usar o fatiamento para acessar os elementos da lista.

```python
print(items8[1:3:1])     # ['strawberry', 'durian']
print(items8[0:3:1])     # ['apple', 'strawberry', 'durian']
print(items8[0:5:2])     # ['apple', 'durian', 'watermelon']
print(items8[-4:-2:1])   # ['strawberry', 'durian']
print(items8[-2:-6:-1])  # ['peach', 'durian', 'strawberry', 'apple']
```

Se o valor de `start` for `0`, ele poderá ser omitido ao usar o fatiamento. Se o valor de `end` for `N`, onde `N` é o número de elementos da lista, também poderá ser omitido. Se o valor de `stride` for `1`, ele também poderá ser omitido. Portanto, o código abaixo tem exatamente o mesmo efeito do código acima.

```python
print(items8[1:3])     # ['strawberry', 'durian']
print(items8[:3:1])    # ['apple', 'strawberry', 'durian']
print(items8[::2])     # ['apple', 'durian', 'watermelon']
print(items8[-4:-2])   # ['strawberry', 'durian']
print(items8[-2::-1])  # ['peach', 'durian', 'strawberry', 'apple']
```

> **Lembrete**: observe a última linha do código acima e pense em como o fatiamento acessa os elementos quando a etapa é um número negativo.

Na verdade, também podemos usar o fatiamento para modificar elementos de uma lista. Por exemplo, adicione mais uma linha ao código acima e veja qual é o resultado.

```python
items8[1:3] = ['x', 'o']
print(items8)  # ['apple', 'x', 'o', 'peach', 'watermelon']
```

Duas listas também podem ser comparadas. Podemos comparar se duas listas são iguais e também comparar seus tamanhos, conforme mostrado abaixo.

```python
nums1 = [1, 2, 3, 4]
nums2 = list(range(1, 5))
nums3 = [3, 2, 1]
print(nums1 == nums2)  # True
print(nums1 != nums2)  # False
print(nums1 <= nums3)  # True
print(nums2 >= nums3)  # False
```

> **Observação**: Os elementos correspondentes de `nums1` e `nums2` são exatamente iguais, portanto, o resultado de `==` é `True`. Para a comparação entre `nums2` e `nums3`, porque o primeiro elemento de `nums2`, que é `1`, é menor que o primeiro elemento de `nums3`, que é `3`, o resultado de `nums2 >= nums3` é `False`. Comparações entre duas listas não são usadas com frequência em trabalhos reais. Se você não os entende, simplesmente ignore-os e não fique preso aqui.

### Iterando sobre elementos

Se quisermos retirar os elementos de uma lista um por um, podemos usar um loop `for-in`. Existem duas maneiras comuns.

Método 1: use a indexação dentro do loop para percorrer os elementos da lista.

```python
languages = ['Python', 'Java', 'C++', 'Kotlin']
for index in range(len(languages)):
    print(languages[index])
```

Saída:

```
Python
Java
C++
Kotlin
```

> **Observação**: A função `len` acima obtém o número de elementos da lista, que é `N`, e `range(N)` então cria o intervalo de `0` a `N - 1`, que corresponde exatamente aos índices dos elementos da lista.

Método 2: faça um loop direto na lista, de forma que a própria variável do loop represente cada elemento da lista.

```python
languages = ['Python', 'Java', 'C++', 'Kotlin']
for language in languages:
    print(language)
```

Saída:

```
Python
Java
C++
Kotlin
```

### Resumo

Agora podemos usar nosso conhecimento de listas para reescrever o exemplo do lançamento de dados no início.

```python
"""
Roll a die 6000 times and count the occurrences of each face

Author: Luo Hao
Version: 1.1
"""
import random

counters = [0] * 6
# Simulate rolling the die and record the count of each face.
for _ in range(6000):
    face = random.randrange(1, 7)
    counters[face - 1] += 1
# Print the counts.
for face in range(1, 7):
    print(f'Face {face} appeared {counters[face - 1]} times')
```

No código acima, usamos os seis elementos da lista `counters` para representar quantas vezes as faces de 1 a 6 aparecem. No início, os valores de todos os seis elementos são `0`. A seguir, usamos números aleatórios distribuídos uniformemente de `1` a `6` para simular o lançamento do dado. Se o resultado for `1`, o valor de `counters[0]` aumenta em `1`; se o resultado for `2`, o valor de `counters[1]` aumenta em `1`; e assim por diante. Você pode sentir a diferença aqui. Como usamos o tipo lista junto com uma estrutura de loop, nosso tratamento de dados se torna em estilo lote, e isso torna o código reescrito muito mais simples e elegante do que a versão anterior.
