## Estruturas de dados comuns: listas, parte 2

### Listar Métodos

Variáveis ​​do tipo lista possuem muitos métodos que podem nos ajudar a trabalhar com uma lista. Suponha que temos uma lista chamada `foos`, e a lista tem um método chamado `bar`, então a sintaxe para usar esse método de lista é `foos.bar()`. Esta é a sintaxe para chamar o método de um objeto através de uma referência de objeto. Posteriormente, quando falarmos sobre programação orientada a objetos, explicaremos essa sintaxe com mais detalhes. Esta sintaxe também pode ser descrita como o envio de uma mensagem para um objeto.

#### Adicionando e Removendo Elementos

Uma lista é um **contêiner mutável**. Um contêiner mutável significa que podemos adicionar elementos a ele, remover elementos dele e também modificar os elementos que já estão dentro dele.

Podemos usar o método `append` de uma lista para adicionar um elemento ao final da lista e usar o método `insert` para inserir um elemento na lista. Acrescentar significa adicionar o elemento ao final da lista, enquanto inserir significa adicionar um novo elemento em uma posição especificada. Veja o código abaixo.

```python
languages = ['Python', 'Java', 'C++']
languages.append('JavaScript')
print(languages)  # ['Python', 'Java', 'C++', 'JavaScript']
languages.insert(1, 'SQL')
print(languages)  # ['Python', 'SQL', 'Java', 'C++', 'JavaScript']
```

Podemos usar o método `remove` de uma lista para excluir um elemento especificado da lista. Observe que se o elemento a ser excluído não estiver na lista, ele gerará `ValueError` e fará com que o programa trave. Portanto, ao excluir um elemento, é recomendável que você primeiro use a operação de associação que aprendemos anteriormente para verificar. Também podemos usar o método `pop` para remover elementos da lista. Por padrão, `pop` exclui o último elemento da lista, mas é claro que você também pode atribuir uma posição a ele e excluir o elemento na posição especificada. Ao usar `pop`, se o índice estiver fora do intervalo, ele gera `IndexError` e causa falha no programa. Além disso, a lista possui um método `clear`, que limpa todos os elementos da lista, conforme mostrado abaixo.

```python
languages = ['Python', 'SQL', 'Java', 'C++', 'JavaScript']
if 'Java' in languages:
    languages.remove('Java')
if 'Swift' in languages:
    languages.remove('Swift')
print(languages)  # ['Python', 'SQL', 'C++', 'JavaScript']
languages.pop()
temp = languages.pop(1)
print(temp)       # SQL
languages.append(temp)
print(languages)  # ['Python', 'C++', 'SQL']
languages.clear()
print(languages)  # []
```

> **Observação**: `pop` retorna o elemento removido. No exemplo acima, armazenamos o resultado em `temp` e o adicionamos novamente mais tarde. Claro, se quiser, você pode colocar o elemento removido de volta na lista, assim como `languages.append(temp)` faz no código acima.

Aqui está uma pequena pergunta. Se a lista `languages` contém vários valores `'Python'`, quando usamos `languages.remove('Python')`, ele exclui todos os valores `'Python'` ou apenas o primeiro? Você pode adivinhar primeiro e depois tentar você mesmo.

Na verdade, existe outra maneira de excluir elementos de uma lista: use a palavra-chave `del` do Python seguida pelo elemento a ser excluído. Em essência, isso não é diferente de usar `pop` com um índice especificado, mas `pop` retorna o elemento excluído, enquanto `del` tem desempenho um pouco melhor. Isso ocorre porque a instrução de bytecode subjacente correspondente a `del` é `DELETE_SUBSCR`, enquanto as instruções de bytecode correspondentes a `pop` são `CALL_METHOD` e `POP_TOP`. Se você ainda não entendeu isso, simplesmente ignore.

```python
items = ['Python', 'Java', 'C++']
del items[1]
print(items)  # ['Python', 'C++']
```

#### Encontrando posições e contagens

O método `index` de uma lista pode encontrar a posição do índice de um elemento na lista. Se o elemento especificado não puder ser encontrado, o método `index` gerará `ValueError`. O método `count` de uma lista pode contar quantas vezes um elemento aparece na lista, conforme mostrado abaixo.

```python
items = ['Python', 'Java', 'Java', 'C++', 'Kotlin', 'Python']
print(items.index('Python'))     # 0
print(items.index('Python', 1))  # 5
print(items.count('Python'))     # 2
print(items.count('Kotlin'))     # 1
print(items.count('Swift'))      # 0
# print(items.index('Java', 3))  # ValueError: 'Java' is not in list
```

#### Classificando e invertendo

O método `sort` de uma lista pode classificar os elementos da lista, e o método `reverse` pode reverter os elementos, conforme mostrado abaixo.

```python
items = ['Python', 'Java', 'C++', 'Kotlin', 'Swift']
items.sort()
print(items)  # ['C++', 'Java', 'Kotlin', 'Python', 'Swift']
items.reverse()
print(items)  # ['Swift', 'Python', 'Kotlin', 'Java', 'C++']
```

### Compreensões de lista

Em Python, uma lista também pode ser criada por meio de uma sintaxe literal especial. Essa sintaxe é chamada de compreensão. Abaixo, usamos exemplos para explicar quais benefícios a compreensão da lista traz.

Cenário 1: crie uma lista de números de `1` a `99` que são divisíveis por `3` ou `5`.

```python
items = []
for i in range(1, 100):
    if i % 3 == 0 or i % 5 == 0:
        items.append(i)
print(items)
```

Usando uma compreensão de lista para fazer a mesma coisa:

```python
items = [i for i in range(1, 100) if i % 3 == 0 or i % 5 == 0]
print(items)
```

Cenário 2: dada uma lista de inteiros `nums1`, crie uma nova lista `nums2` cujos elementos são os quadrados dos elementos correspondentes em `nums1`.

```python
nums1 = [35, 12, 97, 64, 55]
nums2 = []
for num in nums1:
    nums2.append(num ** 2)
print(nums2)
```

Usando uma compreensão de lista para fazer a mesma coisa:

```python
nums1 = [35, 12, 97, 64, 55]
nums2 = [num ** 2 for num in nums1]
print(nums2)
```

Cenário 3: dada uma lista de inteiros `nums1`, crie uma nova lista `nums2` e coloque nela os elementos em `nums1` que são maiores que `50`.

```python
nums1 = [35, 12, 97, 64, 55]
nums2 = []
for num in nums1:
    if num > 50:
        nums2.append(num)
print(nums2)
```

Usando uma compreensão de lista para fazer a mesma coisa:

```python
nums1 = [35, 12, 97, 64, 55]
nums2 = [num for num in nums1 if num > 50]
print(nums2)
```

Usar compreensões de lista para criar listas não é apenas simples e elegante, mas também mais rápido do que usar um loop `for-in` e o método `append` para continuar adicionando elementos a uma lista vazia. Por que as compreensões funcionam melhor? Isso porque o bytecode Python possui instruções especialmente preparadas para compreensão, como a instrução `LIST_APPEND`. Mas um loop `for` adiciona elementos a uma lista por meio de chamadas de método, representadas por instruções como `LOAD_METHOD` e `CALL_METHOD`, e as chamadas de método em si são relativamente demoradas. Não importa se você não entende esse ponto. Apenas lembre-se da conclusão: **é fortemente recomendado usar sintaxe de compreensão para criar listas**.

### Listas aninhadas

Python não exige que os elementos de uma lista tenham o mesmo tipo de dados. Em outras palavras, os elementos de uma lista podem ser de qualquer tipo de dados e, claro, isso inclui as próprias listas. Se os elementos de uma lista também forem listas, podemos chamá-la de **lista aninhada**. Uma lista aninhada pode ser usada para representar uma tabela ou matriz em matemática. Por exemplo, se quisermos armazenar as notas de 5 alunos em 3 cursos, podemos utilizar a lista abaixo.

```python
scores = [[95, 83, 92], [80, 75, 82], [92, 97, 90], [80, 78, 69], [65, 66, 89]]
print(scores[0])
print(scores[0][1])
```

Para a lista aninhada acima, cada elemento representa as pontuações de um aluno em 3 cursos, como `[95, 83, 92]`. O `83` nessa lista é a pontuação de um curso para aquele aluno. Se quiser acessar esse valor, você precisa usar a indexação duas vezes: `scores[0][1]`. Primeiro, `scores[0]` obtém a lista `[95, 83, 92]` e, em seguida, `[1]` obtém o segundo elemento dessa lista.

Se quiser inserir as partituras pelo teclado e armazená-las em uma lista, você pode usar o código abaixo.

```python
scores = []
for _ in range(5):
    temp = []
    for _ in range(3):
        score = int(input('Enter score: '))
        temp.append(score)
    scores.append(temp)
print(scores)
```

Se quisermos gerar as pontuações aleatoriamente e armazená-las em uma lista, podemos usar uma compreensão de lista, conforme mostrado abaixo.

```python
import random

scores = [[random.randrange(60, 101) for _ in range(3)] for _ in range(5)]
print(scores)
```

> **Nota**: O código `[random.randrange(60, 101) for _ in range(3)]` cria uma lista de 3 números inteiros aleatórios. Em seguida, colocamos esse código dentro de outra compreensão de lista como elementos de lista. Desta forma, são gerados 5 desses elementos e o resultado final é uma lista aninhada.

Agora, vamos usar um programa Python para gerar um conjunto de números aleatórios.

```python
"""
Random number generator for Double Color Ball

Author: Luo Hao
Version: 1.0
"""
import random

red_balls = list(range(1, 34))
selected_balls = []
# Add 6 red balls to the selected list.
for _ in range(6):
    # Generate a random integer representing the index of the selected red ball.
    index = random.randrange(len(red_balls))
    # Remove the selected ball from the red-ball list and add it to the selected list.
    selected_balls.append(red_balls.pop(index))
# Sort the selected red balls.
selected_balls.sort()
# Print the red balls.
for ball in selected_balls:
    print(f'\033[031m{ball:0>2d}\033[0m', end=' ')
# Randomly choose 1 blue ball.
blue_ball = random.randrange(1, 17)
# Print the blue ball.
print(f'\033[034m{blue_ball:0>2d}\033[0m')
```

> **Nota**: No código acima, `print(f'\033[0m...\033[0m')` é usado para controlar a cor de saída, portanto as bolas vermelhas são impressas em vermelho e a bola azul é impressa em azul. O `...` representa o conteúdo que queremos imprimir. `\033[0m` é um código de controle que desativa todos os atributos, portanto o código de controle anterior para de funcionar. Você também pode pensar nisso simplesmente como um separador. O `0` antes de `m` significa o modo de exibição padrão no terminal. Esse `0` pode ser omitido. `1` significa destaque, `5` significa piscar, `7` significa vídeo reverso e assim por diante. Entre `0` e `m`, podemos colocar um número de cor, como `30` para preto, `31` para vermelho, `32` para verde, `33` para amarelo e `34` para azul.

Também podemos usar as funções `sample` e `choice` fornecidas pelo módulo `random` para simplificar o código acima. O primeiro pode fazer amostragem aleatória sem reposição, e o último pode escolher aleatoriamente um elemento. O código modificado é mostrado abaixo.

```python
"""
Random number generator for Double Color Ball

Author: Luo Hao
Version: 1.1
"""
import random

red_balls = [i for i in range(1, 34)]
blue_balls = [i for i in range(1, 17)]
# Randomly choose 6 red balls from the red-ball list (sampling without replacement).
selected_balls = random.sample(red_balls, 6)
# Sort the selected red balls.
selected_balls.sort()
# Print the red balls.
for ball in selected_balls:
    print(f'\033[031m{ball:0>2d}\033[0m', end=' ')
# Randomly choose 1 blue ball from the blue-ball list.
blue_ball = random.choice(blue_balls)
# Print the blue ball.
print(f'\033[034m{blue_ball:0>2d}\033[0m')
```

Se quisermos gerar apostas `N` aleatoriamente, só precisamos colocar o código acima em um loop que executa vezes `N`, conforme mostrado abaixo.

```python
"""
Random number generator for Double Color Ball

Author: Luo Hao
Version: 1.2
"""
import random

n = int(input('How many tickets to generate: '))
red_balls = [i for i in range(1, 34)]
blue_balls = [i for i in range(1, 17)]
for _ in range(n):
    # Randomly choose 6 red balls from the red-ball list (sampling without replacement).
    selected_balls = random.sample(red_balls, 6)
    # Sort the selected red balls.
    selected_balls.sort()
    # Print the red balls.
    for ball in selected_balls:
        print(f'\033[031m{ball:0>2d}\033[0m', end=' ')
    # Randomly choose 1 blue ball from the blue-ball list.
    blue_ball = random.choice(blue_balls)
    # Print the blue ball.
    print(f'\033[034m{blue_ball:0>2d}\033[0m')
```

Conforme mostrado na imagem acima, após a instalação bem-sucedida do `rich`, podemos usar o código a seguir para controlar a saída.

```python
"""
Random number generator for Double Color Ball

Author: Luo Hao
Version: 1.3
"""
import random

from rich.console import Console
from rich.table import Table

# Create the console.
console = Console()

n = int(input('How many tickets to generate: '))
red_balls = [i for i in range(1, 34)]
blue_balls = [i for i in range(1, 17)]

# Create the table and add the header.
table = Table(show_header=True)
for col_name in ('No.', 'Red Balls', 'Blue Ball'):
    table.add_column(col_name, justify='center')

for i in range(n):
    selected_balls = random.sample(red_balls, 6)
    selected_balls.sort()
    blue_ball = random.choice(blue_balls)
    # Add a row to the table: serial number, red balls, blue ball.
    table.add_row(
        str(i + 1),
        f'[red]{" ".join([f"{ball:0>2d}" for ball in selected_balls])}[/red]',
        f'[blue]{blue_ball:0>2d}[/blue]'
    )

# Output the table through the console.
console.print(table)
```

> **Nota**: Na linha 31 do código acima, uma compreensão de lista é usada para transformar os números da bola vermelha em strings e armazená-los em uma lista. `" ".join([...])` une várias strings na lista em uma string completa com espaços. Se você ainda não entendeu isso, pode deixar por enquanto. O `[red]...[/red]` na string é usado para definir a cor de saída como vermelho e `[blue]...[/blue]` na próxima linha é usado para definir a cor de saída como azul. Para obter mais informações sobre a biblioteca `rich`, você pode ler a [documentação oficial](https://github.com/textualize/rich/blob/master/README.cn.md).

### Resumo

Em Python, a estrutura subjacente de uma lista é um array que pode crescer dinamicamente. Os elementos da lista são armazenados continuamente na memória do computador, para que o acesso aleatório possa ser obtido, o que significa que podemos obter o elemento correspondente através de um índice válido, e o tempo de operação não depende de quantos elementos estão na lista. Podemos deixar esses detalhes de armazenamento de lado por enquanto, e ainda não há necessidade de que todos entendam a complexidade de tempo assintótica de cada método de lista, ou seja, a relação entre o tempo usado por um método e o número de elementos da lista. Acho que é mais importante que todos aprendam primeiro como usar listas para resolver problemas no trabalho real.
