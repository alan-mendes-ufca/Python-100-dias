## Visualização de dados - 3

Nas lições anteriores, já temos uma primeira compreensão da ferramenta de visualização de dados matplotlib. Você também deve ter notado que, embora o matplotlib forneça funções poderosas, ele possui muitos parâmetros e, se quisermos personalizar profundamente um gráfico, precisamos modificar uma longa lista deles. Isso não é muito amigável para iniciantes. Por outro lado, os gráficos estatísticos customizados com matplotlib são gráficos estáticos, que podem não ser adequados em algumas situações que requerem interação. Para resolver esses dois problemas, apresentamos aqui duas novas ferramentas de visualização: `seaborn` e `pyecharts`.

### Seaborn

Seaborn é uma ferramenta de visualização de dados construída sobre o matplotlib. Pode ser entendido como um wrapper de nível superior sobre o matplotlib. Seaborn também se integra muito bem com pandas, para que possamos construir gráficos estatísticos melhores com menos código e usá-los para explorar e compreender dados.

Seaborn inclui, mas não está limitado aos seguintes recursos:

1. Uma API orientada a conjuntos de dados para examinar relacionamentos entre múltiplas variáveis.
2. Suporte para uso de variáveis categóricas para mostrar observações ou estatísticas resumidas.
3. Visualização de distribuições univariadas ou bivariadas e comparação entre subconjuntos de dados.
4. Estimativa automática e plotagem de modelos de regressão linear.
5. Paletas e temas integrados para personalizar facilmente o efeito visual de gráficos estatísticos.

### Gráficos

ECharts era originalmente uma biblioteca de gráficos frontend desenvolvida pela Baidu. Em 2018, o ECharts entrou na Apache Incubator e agora já é um projeto de alto nível da Apache Software Foundation. Devido à sua boa interatividade e design gráfico refinado, o ECharts foi reconhecido por muitos desenvolvedores. `pyecharts` é um wrapper Python para ECharts, então os desenvolvedores Python também podem usar ECharts para desenhar gráficos estatísticos bonitos e altamente interativos.

Podemos instalar pyecharts com `pip`.

```bash
pip install pyecharts
```

No JupyterLab, podemos usar diretamente um comando mágico:

```bash
%pip install pyecharts
```

Se quisermos usar pyecharts no JupyterLab, ainda precisamos fazer um pequeno trabalho de preparação, principalmente modificando a configuração dos pyecharts.

```python
from pyecharts.globals import CurrentConfig, NotebookType

CurrentConfig.NOTEBOOK_TYPE = NotebookType.JUPYTER_LAB
```

A seguir, vejamos um exemplo simples adaptado do tutorial oficial para iniciantes do pyecharts.

```python
from pyecharts.charts import Bar
from pyecharts import options as opts

bar_chart = Bar(init_opts=opts.InitOpts(width='600px', height='450px'))
bar_chart.add_xaxis(["Shirt", "Sweater", "Chiffon Shirt", "Pants", "High Heels", "Socks"])
bar_chart.add_yaxis("Merchant A", [25, 20, 36, 10, 75, 90])
bar_chart.add_yaxis("Merchant B", [15, 12, 30, 20, 45, 60])
bar_chart.add_yaxis("Merchant C", [12, 32, 40, 52, 35, 26])
bar_chart.set_global_opts(
    xaxis_opts=opts.AxisOpts(
        axislabel_opts=opts.LabelOpts(color='navy')
    ),
    yaxis_opts=opts.AxisOpts(
        axislabel_opts=opts.LabelOpts(color='navy'),
        min_=0,
        max_=100,
        interval=10
    ),
    title_opts=opts.TitleOpts(
        title='2022 Sales Data Display',
        pos_left='2%',
        title_textstyle_opts=opts.TextStyleOpts(
            color='navy',
            font_size=16,
            font_family='PingFang SC',
            font_weight='bold'
        )
    ),
    toolbox_opts=opts.ToolboxOpts(
        orient='vertical',
        pos_left='right'
    )
)
bar_chart.load_javascript()
```

Após executar o código acima, podemos renderizar o gráfico chamando um método do objeto `bar_chart`. Se usarmos diretamente o método `render`, o gráfico será salvo em um arquivo HTML. Se usarmos `render_notebook`, o gráfico será renderizado na janela do navegador.

```python
bar_chart.render_notebook()
```

A seguir, vejamos como desenhar um gráfico de pizza, também adaptado de um exemplo oficial.

```python
import pyecharts.options as opts
from pyecharts.charts import Pie

x_data = ["Direct Access", "Email Marketing", "Affiliate Ads", "Video Ads", "Search Engine"]
y_data = [335, 310, 234, 135, 1548]
data = [(x, y) for x, y in zip(x_data, y_data)]

pie_chart = Pie(init_opts=opts.InitOpts(width="800px", height="400px"))
pie_chart.add(
    '',
    data_pair=data,
    radius=["50%", "75%"],
    label_opts=opts.LabelOpts(is_show=False),
)
pie_chart.set_global_opts(
    legend_opts=opts.LegendOpts(
        pos_left="left",
        orient="vertical"
    )
)
pie_chart.set_series_opts(
    tooltip_opts=opts.TooltipOpts(is_show=False),
    label_opts=opts.LabelOpts(formatter="{b}({c}): {d}%")
)
pie_chart.load_javascript()
```

```python
pie_chart.render_notebook()
```
