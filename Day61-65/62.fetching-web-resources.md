## Buscando dados de rede com Python

Coletar dados de rede é uma área na qual Python é muito bom. Na lição anterior, falamos sobre como os programas que coletam dados em rede são geralmente chamados de web crawlers ou programas spider. Mesmo na era do big data, os dados ainda são um ponto fraco e uma deficiência para as pequenas e médias empresas. Alguns dados podem ser obtidos por meio de APIs de dados abertos ou pagos, enquanto outros dados do setor e de concorrentes devem ser obtidos por meio da coleta de dados na web. Não importa o caminho usado para obter recursos de dados de rede, Python é uma escolha muito boa, porque a biblioteca padrão do Python e as bibliotecas de terceiros fornecem um bom suporte para coleta de dados da web.

### A Biblioteca `requests`

Para usar Python para obter dados de rede, recomendamos a biblioteca de terceiros chamada `requests`. Na verdade, usamos essa biblioteca nas lições anteriores. De acordo com seu site oficial, `requests` é um wrapper construído na biblioteca padrão do Python e simplifica a operação de acesso a recursos de rede através de HTTP ou HTTPS. Mencionamos em aula que HTTP é um protocolo de solicitação-resposta. Quando inserimos o URL correto em um navegador, geralmente também chamado de endereço de site, e pressionamos Enter, enviamos uma solicitação HTTP a um servidor Web na rede. Depois que o servidor recebe a solicitação, ele nos fornece uma resposta HTTP. No Chrome, abra as Ferramentas do desenvolvedor e mude para a guia Rede, então você poderá ver como realmente são as solicitações e respostas HTTP, conforme mostrado abaixo.

<img src="res/20210822093434.png" style="zoom:40%;">

Por meio da biblioteca `requests`, podemos permitir que um programa Python envie solicitações a um servidor Web, assim como um navegador, e receba a resposta retornada pelo servidor. Da resposta, podemos extrair os dados que desejamos. As páginas da web que o navegador nos mostra são escritas em HTML, e o navegador é como um ambiente interpretador de HTML. O conteúdo que vemos em uma página da web está todo contido em tags HTML. Após obter o código HTML, podemos extrair o conteúdo dos atributos das tags ou do texto dentro das tags. O exemplo abaixo mostra como obter o código HTML de uma página web. Usamos a função `get` da biblioteca `requests` para obter o código da página inicial do Sohu.

```python
import requests

resp = requests.get('https://www.sohu.com/')
if resp.status_code == 200:
    print(resp.text)
```

> **Observação**: A variável `resp` no código acima é um objeto `Response`, um tipo encapsulado pela biblioteca `requests`. Através do atributo `status_code` deste objeto, podemos obter o código de status da resposta, e através do atributo `text` deste objeto, podemos obter o código HTML da página.

Como o `text` do objeto `Response` é uma string, podemos usar o conhecimento de expressões regulares mencionado anteriormente para extrair títulos de notícias e links do código HTML da página. O código é mostrado abaixo.

```python
import re

import requests

pattern = re.compile(r'<a.*?href="(.*?)".*?title="(.*?)".*?>')
resp = requests.get('https://www.sohu.com/')
if resp.status_code == 200:
    all_matches = pattern.findall(resp.text)
    for href, title in all_matches:
        print(href)
        print(title)
```

Além do conteúdo de texto, também podemos usar a biblioteca `requests` para obter recursos binários através de uma URL. O exemplo abaixo mostra como obter o logotipo do Baidu e salvá-lo em um arquivo local chamado `baidu.png`. Você pode clicar com o botão direito do mouse no logotipo do Baidu na página inicial do Baidu e usar o item de menu "Copiar endereço da imagem" para obter o URL da imagem.

```python
import requests

resp = requests.get('https://www.baidu.com/img/PCtm_d9c8750bed0b3c7d089fa7d55720d6cf.png')
with open('baidu.png', 'wb') as file:
    file.write(resp.content)
```

> **Observação**: O atributo `content` do objeto `Response` pode obter os dados binários retornados pelo servidor.

A biblioteca `requests` é muito fácil de usar e suas funções também são fortes e completas. Analisaremos mais sobre isso aos poucos, durante o uso. Se você deseja saber mais sobre a biblioteca `requests`, pode ler a [documentação oficial](https://docs.python-requests.org/zh_CN/latest/).

### Escrevendo código do rastreador

A seguir, usamos "Douban Movies" como exemplo para explicar como escrever o código do rastreador. Seguindo o caminho acima, primeiro usamos `requests` para obter o código HTML da página, depois tratamos todo o código como uma longa string e, desta forma, podemos usar a captura de grupos em expressões regulares para extrair o conteúdo que precisamos da string. O código abaixo mostra como obter os nomes dos 250 melhores filmes de [Filmes de Douban](https://movie.douban.com/). A estrutura da página e o código correspondente de [Top250 do filme Douban](https://movie.douban.com/top250) são mostrados abaixo. Podemos ver que cada página mostra 25 filmes, então se quisermos obter os dados do Top250, precisamos visitar 10 páginas no total. O endereço correspondente é `https://movie.douban.com/top250?start=xxx`. Se `xxx` for `0`, será a primeira página. Se `xxx` for `100`, podemos visitar a quinta página. Para manter o código simples e fácil de ler, obtemos apenas o título e a classificação do filme.

<img src="res/20210822093447.png" style="zoom:40%;">

```python
import random
import re
import time

import requests

for page in range(1, 11):
    resp = requests.get(
        url=f'https://movie.douban.com/top250?start={(page - 1) * 25}',
        # If we do not set User-Agent in the HTTP request header, Douban will detect that it is not a browser and block the request.
        # Through the headers parameter of the get function, we set the value of User-Agent.
        # The exact value can be seen in the browser's developer tools.
        # When visiting most websites with a crawler, disguising the crawler as a browser request is a very important step.
        headers={'User-Agent': 'Mozilla/5.0 (Macintosh; Intel Mac OS X 10_14_6) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/92.0.4515.159 Safari/537.36'}
    )
    # Use a regular expression to get span tags whose class attribute is title and whose text does not start with &
    # and use a capturing group to extract the tag content
    pattern1 = re.compile(r'<span class="title">([^&]*?)</span>')
    titles = pattern1.findall(resp.text)
    # Use a regular expression to get span tags whose class attribute is rating_num
    # and use a capturing group to extract the tag content
    pattern2 = re.compile(r'<span class="rating_num".*?>(.*?)</span>')
    ranks = pattern2.findall(resp.text)
    # Use zip to combine the two lists and loop through all movie titles and ratings
    for title, rank in zip(titles, ranks):
        print(title, rank)
    # Sleep randomly for 1 to 5 seconds to avoid crawling pages too often
    time.sleep(random.random() * 4 + 1)
```

> **Observação**: Ao analisar o protocolo de robôs de Douban, descobrimos que Douban não recusa o rastreador do Baidu de obter seus dados, então também podemos disfarçar o rastreador como o rastreador do Baidu e alterar o parâmetro `headers` da função `get` para `headers={'User-Agent': 'BaiduSpider'}`.

### Usando proxies IP

Ocultar a identidade de um programa rastreador é relativamente importante ao escrever o código do rastreador. Muitos sites não gostam de rastreadores porque eles consomem grande parte da largura de banda da rede e criam muito tráfego inválido. Para ocultar a identidade, geralmente é necessário usar **proxies IP comerciais**, como Mogu Proxy, Zhima Proxy, Kuaidaili e assim por diante, para que o site rastreado não consiga obter o endereço IP real do programa rastreador e, então, não possa bloquear facilmente o rastreador apenas pelo endereço IP.

Abaixo, usamos [Eu posso proxy](http://www.moguproxy.com/) como exemplo para explicar como usar um proxy IP comercial. Primeiro, você precisa registrar uma conta no site. Após o registro, você pode [comprar](http://www.moguproxy.com/buy) o pacote correspondente para obter proxies IP comerciais. Para uso comercial, é recomendável adquirir um pacote ilimitado, para que você possa obter endereços IP proxy suficientes de acordo com suas reais necessidades. Para uso educacional, você pode comprar um pacote baseado em tempo ou decidir de acordo com suas necessidades. O Mogu Proxy fornece duas maneiras de conectar um proxy, ou seja, proxy privado de API e proxy de túnel HTTP. O primeiro obtém o endereço do servidor proxy solicitando a API do Mogu Proxy, e o último se conecta diretamente por meio de uma entrada unificada, ou seja, o nome de domínio fornecido pelo Mogu Proxy.

A seguir, usamos o proxy de túnel HTTP como exemplo para explicar como conectar um proxy IP. Você também pode consultar diretamente o código fornecido pelo site oficial do Mogu Proxy para definir um proxy para o rastreador.

```python
import requests

APP_KEY = 'Wnp******************************XFx'
PROXY_HOST = 'secondtransfer.moguproxy.com:9001'

for page in range(1, 11):
    resp = requests.get(
        url=f'https://movie.douban.com/top250?start={(page - 1) * 25}',
        # Need to set proxy authentication in the HTTP request header
        headers={
            'Proxy-Authorization': f'Basic {APP_KEY}',
            'User-Agent': 'Mozilla/5.0 (Macintosh; Intel Mac OS X 10_14_6) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/92.0.4515.159 Safari/537.36',
            'Accept-Language': 'zh-CN,zh;q=0.8,en-US;q=0.6,en;q=0.4'
        },
        # Set the proxy server
        proxies={
            'http': f'http://{PROXY_HOST}',
            'https': f'https://{PROXY_HOST}'
        },
        verify=False
    )
    pattern1 = re.compile(r'<span class="title">([^&]*?)</span>')
    titles = pattern1.findall(resp.text)
    pattern2 = re.compile(r'<span class="rating_num".*?>(.*?)</span>')
    ranks = pattern2.findall(resp.text)
    for title, rank in zip(titles, ranks):
        print(title, rank)
```

> **Observação**: O código acima exige que você altere `APP_KEY` para o valor `Appkey` de seu próprio pedido. Este valor pode ser visto na central do usuário em pedidos de usuários. O Mogu Proxy oferece testes gratuitos para proxy de API e proxy de túnel HTTP, mas a taxa de conexão dos proxies de teste não pode ser garantida. Ainda é recomendável comprar diretamente um serviço de proxy dentro de sua capacidade de pagamento e usá-lo para obter experiência.
