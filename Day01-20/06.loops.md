## Laços

Quando escrevemos programas, muito provavelmente nos depararemos com situações em que uma instrução ou algumas instruções precisarão ser executadas repetidas vezes. Por exemplo, suponha que precisamos imprimir `hello, world` na tela uma vez a cada segundo e continuar fazendo isso por uma hora. O código abaixo pode fazer isso uma vez. Se quisermos continuar imprimindo por uma hora, precisaremos escrever esse código 3.600 vezes. Você gostaria de fazer isso?

```python
import time

print('hello, world')
time.sleep(1)
```

> **Nota**: O módulo `time` integrado fornece a função `sleep`, que pode pausar o programa. O argumento `1` significa o número de segundos para dormir. Pode ser um `int` ou um `float`, então `0.05` significa `50` milissegundos. Falaremos sobre funções e módulos posteriormente neste curso.

Para resolver o problema acima, podemos usar estruturas de loop em programas Python. Uma estrutura de loop é uma estrutura que controla uma instrução ou algumas instruções a serem executadas repetidamente. Com tal estrutura, o código que acabamos de ver não precisa ser escrito 3.600 vezes. Escrevemos uma vez e depois o colocamos em uma estrutura de loop para repeti-lo 3.600 vezes. Em Python, existem duas maneiras de construir estruturas de loop: uma é o loop `for-in` e a outra é o loop `while`.

### Laços `for-in`

Se soubermos claramente quantas vezes o loop será executado, recomendamos usar um loop `for-in`. Por exemplo, para a situação de repetir 3600 vezes, podemos usar o seguinte código. Observe que o bloco de código controlado pelo loop `for-in` também é construído por meio de indentação. É o mesmo que os blocos de código são construídos em estruturas ramificadas. O bloco de código controlado pelo loop `for-in` é chamado de corpo do loop. Normalmente, as instruções no corpo do loop são executadas repetidamente de acordo com a configuração do loop.

```python
"""
Print "hello, world" once every second for one hour

Author: Luo Hao
Version: 1.0
"""
import time

for i in range(3600):
    print('hello, world')
    time.sleep(1)
```

Deve ser explicado que `range(3600)` no código acima cria um intervalo de `0` a `3599`. Quando colocamos esse intervalo em um loop `for-in`, a variável de loop `i` retira os números inteiros de `0` a `3599` um por um, e isso faz com que as instruções no corpo do loop `for-in` se repitam 3600 vezes. Claro, `range` é muito flexível. Aqui estão alguns exemplos comuns:

- `range(101)`: gera números inteiros de `0` a `100`. Observe que não inclui `101`.
- `range(1, 101)`: gera números inteiros de `1` a `100`. É como o intervalo fechado à esquerda e aberto à direita `[1, 101)`.
- `range(1, 101, 2)`: gera números ímpares de `1` a `99`. Aqui `2` é o tamanho do passo, o que significa a quantidade adicionada a cada vez. `101` não está incluído.
- `range(100, 0, -2)`: gera números pares de `100` até `2`. Aqui `-2` é o tamanho do passo, o que significa o valor subtraído de cada vez. `0` não está incluído.

Você já deve ter notado que imprimir e dormir no código acima na verdade não usa a variável de loop `i`. Para um loop `for-in` onde a variável de loop não é necessária, a convenção Python geralmente usa `_` como o nome da variável de loop. O código modificado é mostrado abaixo. O resultado não muda, mas escrever desta forma parece mais profissional.

```python
"""
Print "hello, world" once every second for one hour

Author: Luo Hao
Version: 1.1
"""
import time

for _ in range(3600):
    print('hello, world')
    time.sleep(1)
```

O código acima seria executado por uma hora. Se quiser interromper o programa mais cedo, no PyCharm você pode clicar no botão parar na janela de execução, conforme mostrado na imagem abaixo. Se você executar o código no prompt de comando ou no terminal, poderá usar `Ctrl+C` para interromper o programa.

Agora vamos usar um loop `for-in` para calcular a soma dos inteiros de 1 a 100, ou seja, $\small{\sum_{n=1}^{100}{n}}$.

```python
"""
Sum the integers from 1 to 100

Version: 1.0
Author: Luo Hao
"""
total = 0
for i in range(1, 101):
    total += i
print(total)
```

No código acima, a função da variável `total` é armazenar o resultado acumulado. Durante o loop, o valor da variável de loop `i` vai de `1` até `100`. Para cada valor de `i`, executamos `total += i`, que é igual a `total = total + i`. Esta instrução realiza a acumulação. Portanto, quando o loop termina e imprimimos o valor de `total`, é o resultado da adição de `1` a `100`, que é `5050`. Observe que a instrução `print(total)` não está indentada. Não é controlado pelo loop `for-in`, portanto não será executado repetidamente.

A seguir, vamos escrever um código para calcular a soma dos números pares de 1 a 100.

```python
"""
Sum the even numbers from 1 to 100

Version: 1.0
Author: Luo Hao
"""
total = 0
for i in range(1, 101):
    if i % 2 == 0:
        total += i
print(total)
```

> **Nota**: No loop `for-in` acima, usamos uma estrutura de ramificação para verificar se a variável de loop `i` é um número par.

Também podemos alterar os argumentos da função `range` definindo o valor inicial e o tamanho do passo como `2` e usar um código mais simples para calcular a soma dos números pares de 1 a 100.

```python
"""
Sum the even numbers from 1 to 100

Version: 1.1
Author: Luo Hao
"""
total = 0
for i in range(2, 101, 2):
    total += i
print(total)
```

Claro, uma maneira ainda mais simples é usar a função `sum` integrada do Python. Nesse caso, não precisamos de nenhum loop.

```python
"""
Sum the even numbers from 1 to 100

Version: 1.2
Author: Luo Hao
"""
print(sum(range(2, 101, 2)))
```

### Laços `while`

Se precisarmos construir uma estrutura de loop, mas não pudermos determinar antecipadamente quantas vezes ela deverá ser repetida, recomendamos usar um loop `while`. Um loop `while` usa um valor booleano ou uma expressão que pode produzir um valor booleano para controlar o loop. Quando o valor é `True`, as instruções no corpo do loop, ou seja, o bloco de código abaixo da instrução `while` com o mesmo recuo, são executadas repetidamente. Quando o valor se torna `False`, o loop termina.

Aqui está um loop `while` que calcula a soma de 1 a 100:

```python
"""
Sum the integers from 1 to 100

Version: 1.1
Author: Luo Hao
"""
total = 0
i = 1
while i <= 100:
    total += i
    i += 1
print(total)
```

Comparado com o loop `for-in`, no código acima adicionamos uma variável `i` antes do início do loop e usamos essa variável para controlar o loop, então `while` é seguido pela condição `i <= 100`. No corpo do loop, além de fazer o acúmulo, também precisamos aumentar o valor de `i`, então adicionamos a instrução `i += 1`. Desta forma, o valor de `i` passa a ser `1`, `2`, `3`, e assim sucessivamente, até `101`. Quando `i` se torna `101`, a condição do loop `while` não é mais verdadeira, o código sai do loop `while` e então imprimimos o valor de `total`, que é o resultado da soma de `1` para `100`, ou seja, `5050`.

Se quisermos calcular a soma dos números pares de 1 a 100, podemos fazer uma pequena alteração no código acima.

```python
"""
Sum the even numbers from 1 to 100

Version: 1.3
Author: Luo Hao
"""
total = 0
i = 2
while i <= 100:
    total += i
    i += 2
print(total)
```

### `break` e `continue`

O que acontece se a condição de um loop `while` for definida como `True`, de modo que a condição seja sempre verdadeira? Vejamos o código abaixo, que ainda usa `while` para construir uma estrutura de loop e calcular a soma dos números pares de `1` a `100`.

```python
"""
Sum the even numbers from 1 to 100

Version: 1.4
Author: Luo Hao
"""
total = 0
i = 2
while True:
    total += i
    i += 2
    if i > 100:
        break
print(total)
```

No código acima, `while True` constrói um loop cuja condição é sempre verdadeira. Isso significa que se não fizermos nada de especial, o loop nunca terminará. Isso é o que costumamos chamar de loop infinito. Para interromper o loop depois que `i` se tornar maior que `100`, usamos a palavra-chave `break`. Sua função é encerrar o loop. Observe que `break` só pode encerrar o loop onde ele aparece. Isso precisa de atenção ao usar loops aninhados, e em breve falaremos sobre o que são loops aninhados.

Além de `break`, existe outra palavra-chave que pode ser usada dentro de loops: `continue`. Ele pode ser usado para pular o código restante na rodada atual e deixar o loop entrar diretamente na próxima rodada, conforme mostrado abaixo.

```python
"""
Sum the even numbers from 1 to 100

Version: 1.5
Author: Luo Hao
"""
total = 0
for i in range(1, 101):
    if i % 2 != 0:
        continue
    total += i
print(total)
```

> **Nota**: No código acima, `continue` ignora o caso em que `i` é ímpar. Somente quando `i` for par o código alcançará `total += i`.

### Laços aninhados

Assim como as estruturas de ramificação, as estruturas de loop também podem ser aninhadas. Em outras palavras, podemos construir um loop dentro de outro loop. O exemplo a seguir mostra como imprimir uma tabuada.

```python
"""
Print the multiplication table

Version: 1.0
Author: Luo Hao
"""
for i in range(1, 10):
    for j in range(1, i + 1):
        print(f'{i}×{j}={i * j}', end='\t')
    print()
```

No código acima, o corpo do loop `for-in` externo usa outro loop `for-in`. O loop externo controla a saída da linha `i`, enquanto o loop interno controla a saída da coluna `j` nessa linha. Claramente, a saída do loop interno `for-in` é uma linha completa na tabela de multiplicação. Portanto, quando o loop interno termina, usamos `print()` para fazer uma quebra de linha, para que a saída a seguir comece novamente em uma nova linha. O resultado final é mostrado abaixo.

### Aplicações de laços

#### Exemplo 1: Determine se um número é primo

Tarefa: insira um número inteiro positivo maior que `1` e determine se é um número primo.

> **Dica**: um número primo é um número inteiro maior que 1 que só pode ser dividido por 1 e por ele mesmo. Para um número inteiro positivo $\small{n}$, podemos verificar se ele possui algum fator entre `2` e $\small{\sqrt{n}}`. É claro que o laço não precisa ir de `2` até $\small{n - 1}$. Para números inteiros positivos maiores que 1, os fatores aparecem em pares, então basta fazer um laço até $\small{\sqrt{n}}`.

```python
"""
Read a positive integer greater than 1 and determine whether it is prime

Version: 1.0
Author: Luo Hao
"""
num = int(input('Enter a positive integer: '))
end = int(num ** 0.5)
is_prime = True
for i in range(2, end + 1):
    if num % i == 0:
        is_prime = False
        break
if is_prime:
    print(f'{num} is a prime number')
else:
    print(f'{num} is not a prime number')
```

> **Nota**: No código acima, usamos a variável booleana `is_prime`. Primeiro atribuímos a ele o valor `True`, assumindo que `num` é um número primo. A seguir, procuramos fatores de `num` na faixa de `2` a `num ** 0.5`. Se encontrarmos um fator de `num`, então definitivamente não é um número primo. Neste momento, atribuímos `False` a `is_prime` e, ao mesmo tempo, usamos a palavra-chave `break` para encerrar o loop. Finalmente, dependendo se o valor de `is_prime` é `True` ou `False`, damos resultados diferentes.

#### Exemplo 2: Máximo Divisor Comum

Tarefa: insira dois números inteiros positivos maiores que `0` e encontre o máximo divisor comum dos dois números.

> **Dica**: O máximo divisor comum de dois números é o maior entre seus fatores comuns.

```python
"""
Input two positive integers and find their greatest common divisor

Version: 1.0
Author: Luo Hao
"""
x = int(input('x = '))
y = int(input('y = '))
for i in range(x, 0, -1):
    if x % i == 0 and y % i == 0:
        print(f'Greatest common divisor: {i}')
        break
```

> **Observação**: No código acima, os valores das variáveis ​​de loop no loop `for-in` vão de grandes a pequenos. Desta forma, o fator `i` que descobrimos que pode dividir `x` e `y` é o maior divisor comum de `x` e `y`, e então usamos `break` para parar o loop. Se `x` e `y` forem coprimos, o loop será executado até que `i` se torne `1`, porque `1` é um fator de todos os números inteiros positivos. Nesse caso, o máximo divisor comum de `x` e `y` é `1`.

Há um problema de eficiência ao usar o código acima para encontrar o máximo divisor comum. Se o valor de `x` for `999999999998` e o valor de `y` for `999999999999`, então claramente os dois números são coprimos e o máximo divisor comum é `1`. Mas se usarmos o código acima, o loop repetirá `999999999998` vezes, o que geralmente é inaceitável. Podemos usar o algoritmo euclidiano para encontrar o máximo divisor comum e isso pode nos ajudar a obter o resultado muito mais rápido. O código é mostrado abaixo.

```python
"""
Input two positive integers and find their greatest common divisor

Version: 1.1
Author: Luo Hao
"""
x = int(input('x = '))
y = int(input('y = '))
while y % x != 0:
    x, y = y % x, x
print(f'Greatest common divisor: {x}')
```

> **Observação**: O método e as etapas usadas para resolver um problema podem ser chamados de algoritmo. Para o mesmo problema, podemos projetar algoritmos diferentes. Diferentes algoritmos diferem no uso de armazenamento e na eficiência de execução, e essas diferenças mostram qual algoritmo é melhor. Você pode comparar os dois trechos de código acima e sentir por que dizemos que o algoritmo euclidiano é a melhor escolha. No código acima, a instrução `x, y = y % x, x` significa atribuir o valor de `y % x` a `x` e atribuir o valor original de `x` a `y`.

#### Exemplo 3: Adivinhe o jogo dos números

Tarefa: o computador fornece um número aleatório entre `1` e `100`. O jogador digita o número que adivinha e o computador dá a dica `A little bigger`, `A little smaller` ou `You got it right`. Se o jogador adivinhar o número corretamente, o computador informa ao usuário quantas vezes ele acertou no total e o jogo termina. Caso contrário, o jogo continua.

```python
"""
A small guess-the-number game

Version: 1.0
Author: Luo Hao
"""
import random

answer = random.randrange(1, 101)
counter = 0
while True:
    counter += 1
    num = int(input('Please enter: '))
    if num < answer:
        print('A little bigger.')
    elif num > answer:
        print('A little smaller.')
    else:
        print('You got it right.')
        break
print(f'You guessed {counter} times in total.')
```

> **Nota**: O código acima usa `import random` para importar o módulo `random` da biblioteca padrão do Python. A função `randrange` deste módulo nos ajuda a gerar um número aleatório no intervalo de `1` a `100` (não incluindo `101`). A variável `counter` é utilizada para registrar quantas vezes o loop foi executado, ou seja, quantas vezes o usuário adivinhou no total. Cada vez que o loop é executado, o valor de `counter` aumenta em `1`.

### Resumo

Depois de aprender estruturas de ramificação e estruturas de loop em Python, podemos resolver muitos problemas práticos. Através desta lição, todos já devem saber que podemos usar as palavras-chave `for` e `while` para construir estruturas de loop. **Se soubermos antecipadamente quantas vezes o loop se repete, normalmente usamos** `for` **loops**; **se o número de repetições não puder ser determinado, podemos usar** `while` **loops**. Além disso, podemos **usar** `break` **dentro de um loop para encerrá-lo**, e **também podemos usar a** `continue` **palavra-chave dentro de um loop para permitir que ele entre diretamente na próxima rodada**.
