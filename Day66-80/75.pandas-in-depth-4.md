### Data Pivoting

After the previous study, we have prepared the data and turned it into the shape we want. Next comes the most important stage, data pivoting. When we get a large amount of data, how can we quickly read valuable information from it, turn complicated data into easy-to-read statistical charts, and then generate business insights on that basis? This is exactly the core problem that data analysis needs to solve.

#### Getting descriptive statistics

First, we can obtain descriptive statistics of the data. Through descriptive statistics, we can understand the central tendency and dispersion trend of the data.

For example, suppose we have the student score table below.

```python
scores = np.random.randint(50, 101, (5, 3))
names = ('Guan Yu', 'Zhang Fei', 'Zhao Yun', 'Ma Chao', 'Huang Zhong')
courses = ('Chinese', 'Math', 'English')
df = pd.DataFrame(data=scores, columns=courses, index=names)
df
```

Output:

```
             Chinese  Math  English
Guan Yu           96    72       73
Zhang Fei         72    70       97
Zhao Yun          74    51       79
Ma Chao          100    54       54
Huang Zhong       89   100       88
```

We can use methods such as `mean`, `max`, `min`, `std`, and `var` on a `DataFrame` object to get the average score, highest score, lowest score, standard deviation, variance, and other information for each student or each course. We can also directly use `describe` to get descriptive statistics.

Compute the average score of each course:

```python
df.mean()
```

Output:

```
Chinese    86.2
Math       69.4
English    78.2
dtype: float64
```

Compute the average score of each student:

```python
df.mean(axis=1)
```

Output:

```
Guan Yu        80.333333
Zhang Fei      79.666667
Zhao Yun       68.000000
Ma Chao        69.333333
Huang Zhong    92.333333
dtype: float64
```

Compute the variance of each course:

```python
df.var()
```

> **Note**: From the variance, we can see that Math has the largest fluctuation, so its polarization may be more serious.

Output:

```
Chinese    161.2
Math       379.8
English    265.7
dtype: float64
```

Get descriptive statistics for each course:

```python
df.describe()
```

Output:

#### Sorting and taking head values

If we need to sort data, we can use the `sort_values` method of a `DataFrame`. The `by` parameter specifies which column or columns are used for sorting, and the `ascending` parameter specifies ascending or descending order. For example, the following code sorts the student table in descending order by Chinese score.

```python
df.sort_values(by='Chinese', ascending=False)
```

Output:

```
             Chinese  Math  English
Ma Chao          100    54       54
Guan Yu           96    72       73
Huang Zhong       89   100       88
Zhao Yun          74    51       79
Zhang Fei         72    70       97
```

If the amount of data in a `DataFrame` is very large, sorting can be a time-consuming operation. Sometimes we only need the top N or bottom N values, and in that case there is no need to sort the entire table. Instead, we can directly find Top-N data through a heap structure. The `nlargest` and `nsmallest` methods of `DataFrame` support Top-N operations.

Find the top 3 students in Chinese:

```python
df.nlargest(3, 'Chinese')
```

Output:

```
             Chinese  Math  English
Ma Chao          100    54       54
Guan Yu           96    72       73
Huang Zhong       89   100       88
```

Find the 3 students with the lowest Math scores:

```python
df.nsmallest(3, 'Math')
```

Output:

```
             Chinese  Math  English
Zhao Yun          74    51       79
Ma Chao          100    54       54
Zhang Fei         72    70       97
```

#### Grouped aggregation

Let us first read the 2020 sales data from the Excel file used before, and then demonstrate how to do grouped aggregation.

```python
df = pd.read_excel('data/2020-sales-data.xlsx')
df.head()
```

Output:

```
  Sales Date Sales Area Sales Channel Sales Order   Brand  Price  Sales Quantity
0 2020-01-01     Shanghai     Pinduoduo 182894-455  BPM      99              83
1 2020-01-01     Shanghai         Douyin 205635-402  BPM     219              29
2 2020-01-01     Shanghai          Tmall 205654-021  BPM     169              85
3 2020-01-01     Shanghai          Tmall 205654-519  BPM     169              14
4 2020-01-01     Shanghai          Tmall 377781-010  PPX     249              61
```

If we want to count the total sales amount of each sales area, we can first calculate the sales amount from `price` and `sales quantity`, and add a new column to the `DataFrame`.

```python
df['Sales Amount'] = df['Price'] * df['Sales Quantity']
df.head()
```

Output:

```
Head of the table with the new `Sales Amount` column added.
```

Then we group the data according to the `Sales Area` column by using the `groupby` method of a `DataFrame`. After grouping, we take the `Sales Amount` column and sum it within each group.

```python
df.groupby('Sales Area')['Sales Amount'].sum()
```

Output:

```
Total sales amount grouped by sales area.
```

If we want to count the total sales amount of each month, we can use the sales date as the grouping basis, but here we first need to turn the sales date into month values.

```python
df.groupby(df['Sales Date'].dt.month)['Sales Amount'].sum()
```

Output:

```
Total sales amount grouped by month.
```

Now let us increase the difficulty and count the total sales amount of each sales area in each month. In fact, the first argument of `groupby` can be a list, and the list can specify multiple grouping bases.

```python
df.groupby(['Sales Area', df['Sales Date'].dt.month])['Sales Amount'].sum()
```

Output:

```
Grouped totals by sales area and month in a long narrow table.
```

If we want to count the total sales amount of each area and also the maximum and minimum amount of a single order in each area, we can use the `agg` method on a `DataFrame` or `Series` and specify multiple aggregation functions.

```python
df.groupby('Sales Area')['Sales Amount'].agg(['sum', 'max', 'min'])
```

Output:

```
Grouped sum, max, and min of sales amount for each sales area.
```

If we want to customize the names of the aggregated columns, we can do it like this:

```python
df.groupby('Sales Area')['Sales Amount'].agg(Total_Sales='sum', Highest_Single_Order='max', Lowest_Single_Order='min')
```

Output:

```
Grouped result with custom column names.
```

If we need to use different aggregation functions on multiple columns, for example "count the sum of sales amount and the maximum and minimum of sales quantity for each sales area", we can write:

```python
df.groupby('Sales Area')[['Sales Amount', 'Sales Quantity']].agg({
    'Sales Amount': 'sum', 'Sales Quantity': ['max', 'min']
})
```

Output:

```
Grouped aggregation result using different functions on multiple columns.
```

#### Pivot tables and cross tables

In the example above, "count the total sales amount of each sales area in each month" produces a very long result. In actual work, we usually call a table with many rows and few columns a "narrow table". If we do not want such a narrow table, we can use the `pivot_table` method of a `DataFrame` or the `pivot_table` function to generate a pivot table. The essence of a pivot table is grouped aggregation. It means **count column B according to column A**. If you have used Excel, this concept should be familiar.

For example, if we want to "count the total sales amount of each sales area", then `Sales Area` is column A, and `Sales Amount` is column B. In the `pivot_table` function, they correspond to the `index` and `values` parameters.

```python
pd.pivot_table(df, index='Sales Area', values='Sales Amount', aggfunc='sum')
```

Output:

```
Pivot table of total sales amount by sales area.
```

> **Notice**: The result above is a little different from the result of using `groupby`. After `groupby`, if we aggregate a single column, the result is a `Series`, while the result above is a `DataFrame`.

If we want to count the total sales amount of each sales area in each month, we can also use `pivot_table`.

```python
df['Month'] = df['Sales Date'].dt.month
pd.pivot_table(df, index=['Sales Area', 'Month'], values='Sales Amount', aggfunc='sum')
```

Output:

```
Long pivot table by sales area and month.
```

The result above is still a long narrow table. If we want a "wide table" with fewer rows and more columns, we can move the column in `index` into the `columns` parameter.

```python
pd.pivot_table(df, index='Sales Area', columns='Month', values='Sales Amount', aggfunc='sum', fill_value=0)
```

Output:

```
Wide pivot table by sales area and month, with null values filled as 0.
```

> **Note**: `fill_value=0` in `pivot_table` will treat null values as `0`.

When using `pivot_table`, we can also add `margins` and `margins_name` to generate summary totals for the grouped result.

```python
pd.pivot_table(df, index='Sales Area', columns='Month', values='Sales Amount', aggfunc='sum', fill_value=0, margins=True, margins_name='Total')
```

Output:

```
Wide pivot table with totals added.
```

A cross table is a special kind of pivot table. It does not need us to first build a `DataFrame` object. Instead, it directly uses arrays or `Series` objects to specify two or more factors and calculates a statistical result. For example, if we want to count the total sales amount of each sales area, we can also do it as follows.

First prepare three groups of data:

```python
sales_area, sales_month, sales_amount = df['Sales Area'], df['Month'], df['Sales Amount']
```

Use the `crosstab` function to generate a cross table:

```python
pd.crosstab(index=sales_area, columns=sales_month, values=sales_amount, aggfunc='sum').fillna(0).astype('i8')
```

> **Note**: In the code above, `fillna` is used to treat null values as 0, and `astype` is used to convert the data type into integers.

Output:

```
Cross table of sales amount by sales area and sales month.
```

### Data Presentation

A picture is worth a thousand words. The result of our data pivoting should finally be presented in charts, because charts have very strong expressive power and let us quickly understand the value hidden in the data. Like `Series`, `DataFrame` also provides a `plot` method to support plotting. Under the hood, chart rendering is still done through matplotlib.

For example, if we want to use a bar chart to compare the total sales amount of each sales area, we can directly use the `plot` method on the pivot table. First import `matplotlib.pyplot` and modify the plotting parameters so that Chinese can be displayed properly if needed.

```python
import matplotlib.pyplot as plt

plt.rcParams['font.sans-serif'] = 'FZJKai-Z03S'
```

Use a magic command to configure vector graphics output:

```python
%config InlineBackend.figure_format = 'svg'
```

Draw a bar chart of the total sales amount of each sales area:

```python
temp = pd.pivot_table(df, index='Sales Area', values='Sales Amount', aggfunc='sum')
temp.plot(figsize=(8, 4), kind='bar')
plt.xticks(rotation=0)
plt.show()
```

If we want to draw a pie chart, we can change the `kind` parameter of `plot` to `pie`, and then use pie-chart parameters to customize the chart.

```python
temp.sort_values(by='Sales Amount', ascending=False).plot(
    figsize=(6, 6),
    kind='pie',
    y='Sales Amount',
    ylabel='',
    autopct='%.2f%%',
    pctdistance=0.8,
    wedgeprops=dict(linewidth=1, width=0.35),
    legend=False
)
plt.show()
```
