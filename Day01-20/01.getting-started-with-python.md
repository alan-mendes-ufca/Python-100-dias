## Introdução ao Python

### Introdução ao Python

Python (pronúncia britânica: `/ˈpaɪθən/`; pronúncia americana: `/ˈpaɪθɑːn/`) é uma linguagem de programação inventada pelo holandês Guido van Rossum. É hoje a linguagem de programação mais popular do mundo e também possui o maior número de usuários. Python enfatiza a legibilidade do código e a sintaxe simples. Comparado com C, C++, Java e outras linguagens de programação que também são muito influentes, Python permite que os usuários usem menos código para expressar suas ideias. Abaixo estão as classificações do Python fornecidas por várias classificações confiáveis ​​de linguagens de programação. A primeira imagem é do Índice TIOBE e a terceira imagem é do IEEE Spectrum. A segunda foto merece destaque especial. Ele mostra o quão populares são as linguagens de programação no GitHub, a maior plataforma de hospedagem de código do mundo. Nos últimos quatro anos, Python conquistou o primeiro lugar lá.

#### Crônica Python

Abaixo estão alguns momentos importantes no desenvolvimento do Python:

1. Dezembro de 1989: Guido van Rossum decidiu desenvolver uma nova linguagem de script e seu intérprete para passar um feriado chato de Natal. A nova linguagem seria a sucessora da linguagem ABC e seria usada principalmente para substituir o shell Unix e a linguagem C para fazer o gerenciamento do sistema. Como o próprio Guido era um fã fiel da série de TV da BBC *Monty Python's Flying Circus*, ele escolheu a palavra Python como o nome da nova linguagem.
2. Fevereiro de 1991: Guido van Rossum lançou o primeiro código do interpretador Python no grupo de notícias `alt.sources`, marcado como versão `0.9.0`.
3. Janeiro de 1994: Python `1.0` foi lançado. Este foi o lugar onde o sonho começou.
4. Outubro de 2000: Python `2.0` foi lançado. Todo o processo de desenvolvimento do Python tornou-se mais aberto e o ecossistema começou a se formar lentamente.
5. Dezembro de 2008: Python `3.0` foi lançado. Ele introduziu muitos recursos novos de linguagens de programação modernas, mas não era totalmente compatível com versões anteriores.
6. Abril de 2011: `pip` foi lançado pela primeira vez e Python tinha sua própria ferramenta de gerenciamento de pacotes.
7. Julho de 2018: Guido van Rossum anunciou que estava tirando "férias permanentes" do cargo de "Ditador Benevolente Vitalício" (a pessoa que tem a decisão final quando há uma disputa em uma comunidade de projeto de código aberto).
8. Janeiro de 2020: Depois que Python 2 e Python 3 viveram juntos por 11 anos, a equipe oficial parou de atualizar e manter o Python 2 e esperava que os usuários mudassem para o Python 3 o mais rápido possível.
9. Atualmente: Python tem sido amplamente utilizado em grandes modelos (GPT-3, GPT-4, BERT e assim por diante), visão computacional (reconhecimento de imagem, detecção de objetos, geração de imagens e assim por diante), recomendação inteligente (YouTube, Netflix, ByteDance e assim por diante), direção autônoma (Waymo, Apollo e assim por diante), reconhecimento de fala, ciência de dados, negociação quantitativa, testes automatizados, operações automatizadas e outros campos. O ecossistema do Python também é bastante próspero.

> **Observação**: O número da versão da maioria dos softwares geralmente é dividido em três partes, no formato `A.B.C`. Aqui, `A` significa o número da versão principal. `A` aumenta apenas quando o software é reescrito e atualizado como um todo ou quando aparece uma alteração incompatível com versões anteriores. `B` significa uma atualização de recurso. `B` aumenta quando um novo recurso aparece. `C` significa uma pequena alteração (por exemplo, correção de um bug). Enquanto houver uma mudança, `C` aumenta.

#### Vantagens e desvantagens do Python

Python tem muitas vantagens. Aqui vou simplesmente listar alguns para todos.

1. É **simples e elegante**. Comparado com muitas outras linguagens de programação, Python é **mais fácil de começar**.
2. Ele pode usar menos código para fazer mais coisas e **melhorar a eficiência do desenvolvimento**.
3. É de código aberto e possui uma **comunidade e ecossistema poderosos**.
4. Ele **pode fazer muitas coisas** e tem uma adaptabilidade muito forte.
5. É uma **linguagem de cola** e pode integrar componentes desenvolvidos em outras linguagens.
6. É uma linguagem interpretada, por isso é mais fácil **trabalhar entre plataformas** e pode ser executada em vários sistemas operacionais.

A maior desvantagem do Python é a **baixa eficiência de execução** (este é um problema comum em linguagens interpretadas). Se você se preocupa mais com a eficiência de execução de código, C, C++ ou Go podem ser uma escolha melhor para você.

### Instalando o ambiente Python

Se um trabalhador quiser fazer bem o seu trabalho, ele deve primeiro afiar as suas ferramentas. Se você deseja iniciar sua jornada de programação Python, primeiro você precisa instalar o ambiente Python em seu computador. Simplificando, isso significa instalar o interpretador Python necessário para executar programas Python. Recomendamos que todos instalem o interpretador oficial do Python 3. Está escrito em C e também costumamos chamá-lo de CPython. Pode ser sua melhor escolha neste momento. Primeiro, precisamos encontrar o link de download na [página de downloads oficial](https://www.python.org/downloads/). Após clicar no botão “Download” e entrar na página de download, você precisa escolher um instalador Python 3 adequado de acordo com o seu sistema operacional, conforme mostrado na imagem abaixo.

<img src="res/day01/python_download_page_1.png" style="zoom:40%;">

Depois de entrar na página de download, algumas versões do Python não fornecem instaladores para Windows e macOS. Elas fornecem apenas arquivos de código-fonte compactados. Para quem está familiarizado com Linux, é possível compilar e instalar a partir do código-fonte. Para quem usa Windows ou macOS, ainda **recomendamos** usar o instalador. Por exemplo, se você deseja instalar o Python 3.10, escolher Python `3.10.10` ou `3.10.11` permitirá que você encontre o pacote de instalação para Windows ou macOS. #### Ambiente Windows

Abaixo usamos o Windows 11 como exemplo para explicar como instalar o ambiente Python no sistema operacional Windows. Clique duas vezes e execute o instalador baixado do site oficial, e um assistente de instalação será aberto, conforme mostrado na imagem abaixo.

<img src="res/day01/install_python_1.png" style="zoom:50%;">

Primeiro, você deve se lembrar de verificar `Add python.exe to PATH`. Isso nos ajudará a adicionar o interpretador Python à variável de ambiente `PATH` do sistema Windows. Se você não entendeu, tudo bem, basta verificar. Em segundo lugar, `Use admin privileges when installing py.exe` serve para obter permissão de administrador durante a instalação e é recomendável verificá-lo. Em seguida, escolhemos `Customize Installation` e usamos o modo de instalação personalizado. Essa é a escolha dos profissionais, e você é esse profissional, ou pelo menos pode fingir ser esse profissional. Não recomendamos usar `Install Now` (a instalação padrão).

A seguir, o assistente de instalação solicitará que você verifique o `Optional Features` necessário. Aqui podemos verificar todos eles diretamente. Uma coisa que vale a pena mencionar é o segundo item. É a ferramenta de gerenciamento de pacotes do Python `pip` e pode nos ajudar a instalar bibliotecas e ferramentas de terceiros, portanto, lembre-se de verificá-la. Em seguida, clique em `Next` para entrar na próxima etapa.

<img src="res/day01/install_python_2.png" style="zoom:50%;">

O próximo passo é escolher `Advanced Options`. Aqui recomendamos que todos verifiquem apenas `Add Python to environment variables` e `Precompile standard library`. O primeiro nos ajudará a definir bem as variáveis ​​​​de ambiente automaticamente. O segundo irá pré-compilar a biblioteca padrão (gerar arquivos `.pyc`), portanto não precisará de compilação temporária quando a usarmos posteriormente. A mesma frase de novo: se você não entendeu, tudo bem, basta verificar. Para `Customize install location` abaixo, **recomendamos fortemente** alterá-lo para um caminho personalizado. Este caminho não deve conter caracteres chineses, espaços ou outros caracteres especiais. Prestar atenção a este ponto reduzirá muitos problemas desnecessários para você no futuro. Após a configuração ser concluída, clique em `Install` para iniciar a instalação.

<img src="res/day01/install_python_3.png" style="zoom:50%;">

Se a instalação for bem-sucedida, a tela mostrada abaixo aparecerá. A palavra-chave para uma instalação bem-sucedida é `successful`. Se a instalação falhar, a palavra aqui se tornará `failed`.

<img src="res/day01/install_python_4.png" style="zoom:50%;">

Após a conclusão da instalação, você pode abrir o Prompt de Comando do Windows ou PowerShell e inserir `python --version` ou `python -V` para verificar se a instalação foi bem-sucedida. Este comando é usado para visualizar o número da versão do interpretador Python. Se você vir a tela mostrada abaixo, parabéns, o ambiente Python foi instalado com sucesso. Aqui também recomendamos verificar se a ferramenta de gerenciamento de pacotes do Python `pip` está disponível. O comando correspondente é `pip --version` ou `pip -V`.

<img src="res/day01/install_python_5.png" style="zoom:50%;">

Depois de clicar duas vezes e executá-lo, você pode continuar clicando em `Continue` e a instalação será bem-sucedida. Você quase não precisa definir ou marcar nada, como mostra a imagem abaixo.

#### Outros métodos de instalação

Algumas pessoas podem recomendar que os iniciantes instalem diretamente [Anaconda](https://www.anaconda.com/download/success), porque o Anaconda nos ajudará a instalar o interpretador Python e algumas bibliotecas de terceiros comumente usadas. Além disso, também oferece algumas ferramentas convenientes e é especialmente adequado para iniciantes. Eu pessoalmente não recomendo esse método, porque ao instalar o Anaconda, você de alguma forma instalará uma grande pilha de bibliotecas de terceiros, sejam elas úteis ou não, e elas ocupam bastante espaço em disco. Então seu terminal ou prompt de comando será alterado pelo Anaconda (ele ativa automaticamente o ambiente virtual toda vez que é iniciado). Essas coisas não correspondem ao **princípio do menor espanto** no design de software. Não direi mais aqui sobre outros pequenos problemas do Anaconda. Se você realmente deseja usar o Anaconda, recomendo o Miniconda. Está na mesma página de download do Anaconda.

Iniciantes completos também costumam ouvir ou dizer: "Quero escrever programas Python. Não é suficiente instalar o PyCharm?" Aqui, deixe-me simplesmente explicar. PyCharm é apenas uma ferramenta que ajuda a escrever código Python. Ele próprio não tem a capacidade de executar código Python. O que executa o código Python é o interpretador Python que instalamos acima. Claro, algumas versões do PyCharm também solicitarão que você baixe o interpretador Python da Internet quando você cria um projeto Python e ele não consegue detectar um ambiente Python em seu computador. Colocaremos a instalação e uso do PyCharm na próxima lição.

### Resumo

Vamos resumir o que aprendemos:

1. Python é uma linguagem muito poderosa. Ele pode fazer muitas coisas, então vale a pena aprender.
2. Para usar Python, você deve primeiro instalar o ambiente Python, ou seja, o interpretador Python necessário para executar programas Python.
3. No Windows, você pode inserir `python --version` no prompt de comando ou PowerShell para verificar se o ambiente Python foi instalado com sucesso. No macOS, você pode inserir `python3 --version` no Terminal para fazer a verificação.
