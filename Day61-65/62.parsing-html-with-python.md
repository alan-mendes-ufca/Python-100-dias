## Analisando páginas HTML com Python

Nas lições anteriores, falamos sobre o uso da biblioteca de terceiros `requests` para obter recursos de rede e também apresentamos alguns conhecimentos básicos de front-end. A seguir, continuamos a explorar como analisar o código HTML e extrair informações úteis das páginas. Antes, tentávamos usar a captura de grupos em expressões regulares para extrair o conteúdo da página, mas escrever uma expressão regular correta também é algo que dá dor de cabeça às pessoas. Para resolver esse problema, precisamos primeiro entender mais profundamente a estrutura de uma página HTML e, com base nisso, estudar outras maneiras de analisar uma página.

### Estrutura de uma página HTML

Abra qualquer site no navegador e escolha "Exibir código-fonte da página" no menu do botão direito e você poderá ver o código HTML correspondente à página.

<img src="res/20210822094218.png" style="zoom:40%;">

A linha `1` do código é a declaração do tipo de documento. A tag `<html>` na linha `2` é a tag inicial da tag raiz de toda a página, e a última linha é a tag final da tag raiz, `</html>`. Na tag `<html>` existem duas tags filhas, `<head>` e `<body>`. O conteúdo colocado sob a tag `<body>` é mostrado na janela do navegador e esta parte é o corpo principal da página da web. O conteúdo colocado sob a tag `<head>` não é mostrado na janela do navegador, mas contém metadados importantes da página, geralmente chamados de cabeçalho da página. A estrutura aproximada do código de uma página HTML é mostrada abaixo.

```html
<!doctype html>
<html>
    <head>
        <!-- Page metadata, such as character encoding, title, keywords, media queries, and so on -->
    </head>
    <body>
        <!-- The main body of the page, the content shown in the browser window -->
    </body>
</html>
```

Tags, folhas de estilo em cascata, CSS e JavaScript são os três elementos principais que constituem uma página HTML. As tags carregam o conteúdo a ser mostrado na página, o CSS é responsável pela renderização da página e o JavaScript é usado para controlar o comportamento interativo da página. Para analisar uma página HTML, podemos usar a sintaxe XPath. XPath era originalmente uma sintaxe de consulta para XML e pode extrair o conteúdo em tags ou atributos de tags de acordo com a hierarquia de tags HTML. Além disso, podemos usar seletores CSS para localizar elementos da página, assim como usar CSS para renderizar elementos da página.

### Análise XPath

XPath é uma sintaxe para localizar informações em documentos XML. XML, que significa eXtensible Markup Language, é semelhante ao HTML porque ambos são linguagens de marcação que transportam dados com tags. A diferença é que as tags XML são extensíveis e podem ser personalizadas, e o XML tem requisitos de sintaxe mais rígidos. XPath usa expressões de caminho para selecionar nós ou conjuntos de nós em um documento XML. Os nós aqui incluem elementos, atributos, texto, namespaces, instruções de processamento, comentários, nós raiz e assim por diante. Abaixo, usamos um exemplo para explicar como usar XPath para analisar uma página.

```xml
<?xml version="1.0" encoding="UTF-8"?>
<bookstore>
    <book>
      <title lang="eng">Harry Potter</title>
      <price>29.99</price>
    </book>
    <book>
      <title lang="zh">Learning XML</title>
      <price>39.95</price>
    </book>
</bookstore>
```

Para o arquivo XML acima, podemos usar a sintaxe XPath abaixo para obter nós no documento.

| Expressão de caminho | Resultado |
| --------------- | ------ |
| `/bookstore` | Selecione a livraria do elemento raiz. **Aviso**: se um caminho começar com uma barra `/`, esse caminho sempre representará o caminho absoluto para um elemento. |
| `//book` | Selecione todos os elementos filhos `book`, independentemente de onde estejam no documento. |
| `//@lang` | Selecione todos os atributos denominados `lang`. |
| `/bookstore/book[1]` | Selecione o primeiro elemento `book` que é um elemento filho de `bookstore`. |
| `/bookstore/book[last()]` | Selecione o último elemento `book` que é um elemento filho de `bookstore`. |
| `/bookstore/book[last()-1]` | Selecione o penúltimo elemento `book` que é um elemento filho de `bookstore`. |
| `/bookstore/book[position()<3]` | Selecione os dois primeiros elementos `book` que são elementos filhos de `bookstore`. |
| `//title[@lang]` | Selecione todos os elementos `title` que possuem um atributo denominado `lang`. |
| `//title[@lang='eng']` | Selecione todos os elementos `title` cujo valor do atributo `lang` seja `eng`. |
| `/bookstore/book[price>35.00]` | Selecione todos os elementos `book` em `bookstore` cujo valor do elemento `price` é maior que `35.00`. |
| `/bookstore/book[price>35.00]/title` | Selecione todos os elementos `title` dos elementos `book` em `bookstore`, onde o valor do elemento `price` é maior que `35.00`. |

XPath também oferece suporte ao uso de curingas, conforme mostrado abaixo.

| Expressão de caminho | Resultado |
| --------------- | ------ |
| `/bookstore/*` | Selecione todos os elementos filhos do elemento `bookstore`. |
| `//*` | Selecione todos os elementos do documento. |
| `//title[@*]` | Selecione todos os elementos `title` que possuem atributos. |

Se quiser selecionar vários nós, você pode usar as formas mostradas abaixo.

| Expressão de caminho | Resultado |
| --------------- | ------ |
| `//livro/título \ | //reserva/preço` | Selecione todos os elementos `title` e `price` dos elementos `book`. |
| `//título \ | //preço` | Selecione todos os elementos `title` e `price` no documento. |
| `/livraria/livro/título\ | //preço` | Selecione todos os elementos `title` dos elementos `book` em `bookstore` e todos os elementos `price` no documento. |

Para implementar a análise XPath, é necessária a biblioteca de terceiros `lxml`. Você pode usar o comando abaixo para instalar `lxml`.

```bash
pip install lxml
```

Abaixo, reescrevemos o código anterior para obter o Douban Movie Top250 usando a análise XPath, conforme mostrado abaixo.

```python
from lxml import etree
import requests

HEADERS = {
    'User-Agent': 'Mozilla/5.0 (compatible; Python tutorial example)'
}

for page in range(1, 11):
    resp = requests.get(
        url=f'https://movie.douban.com/top250?start={(page - 1) * 25}',
        headers=HEADERS,
        timeout=10,
    )
    resp.raise_for_status()
    tree = etree.HTML(resp.text)
    # Extract movie titles from the page through XPath syntax
    title_spans = tree.xpath('//*[@id="content"]/div/div[1]/ol/li/div/div[2]/div[1]/a/span[1]')
    # Extract movie ratings from the page through XPath syntax
    rank_spans = tree.xpath('//*[@id="content"]/div/div[1]/ol/li[1]/div/div[2]/div[2]/div/span[2]')
    for title_span, rank_span in zip(title_spans, rank_spans):
        print(title_span.text, rank_span.text)
```

### Análise do seletor CSS

Para desenvolvedores familiarizados com seletores CSS e JavaScript, obter elementos de página por meio de seletores CSS pode ser uma escolha mais simples, porque o JavaScript executado no próprio navegador pode usar os métodos `querySelector()` e `querySelectorAll()` do objeto `document` para obter elementos de página com base em seletores CSS. Em Python, podemos usar a biblioteca de terceiros `beautifulsoup4` ou `pyquery` para fazer a mesma coisa. Beautiful Soup pode ser usado para analisar documentos HTML e XML, reparar documentos que contêm erros, como tags não fechadas, e criar uma estrutura de árvore na memória para a página a ser analisada, de modo que envolva a operação de extração de dados da página. Você pode usar o comando abaixo para instalar o Beautiful Soup.

```python
pip install beautifulsoup4
```

Abaixo está o código reescrito com `bs4` para obter os nomes dos filmes Douban Movie Top250.

```python
import bs4
import requests

HEADERS = {
    'User-Agent': 'Mozilla/5.0 (compatible; Python tutorial example)'
}

for page in range(1, 11):
    resp = requests.get(
        url=f'https://movie.douban.com/top250?start={(page - 1) * 25}',
        headers=HEADERS,
        timeout=10,
    )
    resp.raise_for_status()
    # Create a BeautifulSoup object
    soup = bs4.BeautifulSoup(resp.text, 'lxml')
    # Extrai da página tags span que contêm títulos de filmes usando seletores CSS
    title_spans = soup.select('div.info > div.hd > a > span:nth-child(1)')
    # Extrai da página tags span que contêm avaliações de filmes usando seletores CSS
    rank_spans = soup.select('div.info > div.bd > div > span.rating_num')
    for title_span, rank_span in zip(title_spans, rank_spans):
        print(title_span.text, rank_span.text)
```

Para saber mais sobre BeautifulSoup, você pode consultar a [documentação oficial](https://www.crummy.com/software/BeautifulSoup/bs4/doc.zh/).
