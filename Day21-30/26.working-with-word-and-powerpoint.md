## Trabalhando com arquivos Word e PowerPoint em Python

No trabalho diário, muitos trabalhos simples e repetidos podem ser totalmente entregues a programas Python. Por exemplo, de acordo com um arquivo de amostra, um arquivo de modelo, podemos gerar muitos arquivos Word ou PowerPoint em lote. Word é o programa de processamento de texto desenvolvido pela Microsoft. Acredito que todos estão familiarizados com isso. Muitos documentos formais no trabalho diário de escritório são escritos e editados com o Word, e agora a extensão dos arquivos do Word é geralmente `.docx`. PowerPoint é o programa de apresentação desenvolvido pela Microsoft. É parte da família de software Office da Microsoft e é amplamente utilizado por empresários, professores, estudantes e outros grupos. Também é geralmente chamado de "slides". Em Python, podemos usar a biblioteca de terceiros `python-docx` para operar o Word e usar a biblioteca de terceiros `python-pptx` para gerar o PowerPoint.

### Trabalhando com documentos do Word

Podemos primeiro instalar `python-docx` com o seguinte comando.

```bash
pip install python-docx
```

De acordo com a [documentação oficial](https://python-docx.readthedocs.io/en/latest/), podemos usar o código abaixo para gerar um documento Word simples.

```python
from docx import Document
from docx.shared import Cm, Pt

from docx.document import Document as Doc

# Create the Doc object representing a Word document.
document = Document()  # type: Doc
# Add a top-level heading.
document.add_heading('Happy learning Python', 0)
# Add a paragraph.
p = document.add_paragraph('Pythonis a very popular programming language that')
run = p.add_run('simple')
run.bold = True
run.font.size = Pt(18)
p.add_run('and')
run = p.add_run('elegant')
run.font.size = Pt(18)
run.underline = True
p.add_run('。')

# Add a level-1 heading.
document.add_heading('Heading, level 1', level=1)
# Add a styled paragraph.
document.add_paragraph('Intense quote', style='Intense Quote')
# Add an unordered list.
document.add_paragraph(
    'first item in unordered list', style='List Bullet'
)
document.add_paragraph(
    'second item in ordered list', style='List Bullet'
)
# Add an ordered list.
document.add_paragraph(
    'first item in ordered list', style='List Number'
)
document.add_paragraph(
    'second item in ordered list', style='List Number'
)

# Add an image. Make sure the path and image exist.
document.add_picture('resources/guido.jpg', width=Cm(5.2))

# Add a section break.
document.add_section()

records = (
    ('Luo Hao', 'male', '1995-5-5'),
    ('Sun Meili', 'female', '1992-2-2')
)
# Add a table.
table = document.add_table(rows=1, cols=3)
table.style = 'Dark List'
hdr_cells = table.rows[0].cells
hdr_cells[0].text = 'name'
hdr_cells[1].text = 'gender'
hdr_cells[2].text = 'Date of birth'
# Add rows to the table.
for name, sex, birthday in records:
    row_cells = table.add_row().cells
    row_cells[0].text = name
    row_cells[1].text = sex
    row_cells[2].text = birthday

# Add a page break.
document.add_page_break()

# Save the document.
document.save('demo.docx')
```

> **Dica**: O comentário `# type: Doc` no código acima é usado para obter dicas de conclusão de código no PyCharm, porque se o tipo de dados concreto do objeto não estiver claro, o PyCharm não poderá fornecer dicas de conclusão de código para o objeto `Doc` no código posterior.

Para um arquivo Word existente, podemos iterar todos os seus parágrafos e obter o conteúdo correspondente com o código abaixo.

```python
from docx import Document
from docx.document import Document as Doc

doc = Document('resources/Termination Certificate.docx')  # type: Doc
for no, p in enumerate(doc.paragraphs):
    print(no, p.text)
```

> **Dica**: Se precisar do arquivo Word usado no código acima, use um arquivo próprio com estrutura semelhante; o link de armazenamento da fonte original foi removido por ser pouco acessível fora da China.

Podemos transformar o certificado de demissão acima em um arquivo de modelo e substituir informações como nome, número de identificação, data de início e data de término por espaços reservados. Então, substituindo os espaços reservados, podemos escrever as informações reais correspondentes de acordo com as necessidades reais. Desta forma, podemos gerar documentos Word em lote.

Seguindo a ideia acima, primeiro editamos um arquivo modelo de certificado de demissão, conforme mostrado abaixo.

```python
from docx import Document
from docx.document import Document as Doc

# Store the real data as dictionaries inside a list.
employees = [
    {
        'name': 'Luo Hao',
        'id': '100200198011280001',
        'sdate': '2008March 1,',
        'edate': '201229 February',
        'department': 'Product R&D',
        'position': 'Architect',
        'company': 'Chengdu Huawei Technology Co., Ltd.'
    },
    {
        'name': 'Wang Dahuang',
        'id': '510210199012125566',
        'sdate': '2019January 1,',
        'edate': '2021April 30,',
        'department': 'Product R&D',
        'position': 'PythonDevelopment Engineer',
        'company': 'Chengdu Gudao Technology Co., Ltd.'
    },
    {
        'name': 'Li Yuanfang',
        'id': '2102101995103221599',
        'sdate': '2020May 10,',
        'edate': '2021March 5,',
        'department': 'Product R&D',
        'position': 'JavaDevelopment Engineer',
        'company': 'Tongcheng Enterprise Management Group Co., Ltd.'
    },
]
# Iterate through the list and generate Word documents in batches.
for emp_dict in employees:
    # Read the resignation-certificate template.
    doc = Document('resources/Termination Proof Template.docx')  # type: Doc
    # Iterate over all paragraphs and look for placeholders.
    for p in doc.paragraphs:
        if '{' not in p.text:
            continue
        # We cannot modify the entire paragraph directly, otherwise styling is lost.
        # So we iterate over runs inside the paragraph and replace the text there.
        for run in p.runs:
            if '{' not in run.text:
                continue
            # Replace the placeholder with actual content.
            start, end = run.text.find('{'), run.text.find('}')
            key, place_holder = run.text[start + 1:end], run.text[start:end + 1]
            run.text = run.text.replace(place_holder, emp_dict[key])
    # Save one Word document for each person.
    doc.save(f'{emp_dict["name"]}Termination Certificate.docx')
```

### Gerando arquivos PowerPoint

Primeiro precisamos instalar a biblioteca de terceiros `python-pptx` com o comando abaixo.

```bash
pip install python-pptx
```

Usar Python para operar conteúdo do PowerPoint não é uma situação real muito comum, então não pretendo falar muito aqui. Os leitores interessados podem ler a [documentação oficial](https://python-pptx.readthedocs.io/en/latest/) do `python-pptx` por conta própria. Abaixo mostro apenas um trecho de código da documentação oficial.

```python
from pptx import Presentation

# Create a presentation object.
pres = Presentation()

# Choose a layout and add a slide.
title_slide_layout = pres.slide_layouts[0]
slide = pres.slides.add_slide(title_slide_layout)
# Get the title box and subtitle box.
title = slide.shapes.title
subtitle = slide.placeholders[1]
# Edit the title and subtitle.
title.text = "Welcome to Python"
subtitle.text = "Life is short, I use Python"

# Choose a layout and add another slide.
bullet_slide_layout = pres.slide_layouts[1]
slide = pres.slides.add_slide(bullet_slide_layout)
# Get all shapes on the slide.
shapes = slide.shapes
# Get the title and body.
title_shape = shapes.title
body_shape = shapes.placeholders[1]
# Edit the title.
title_shape.text = 'Introduction'
# Edit the body text.
tf = body_shape.text_frame
tf.text = 'History of Python'
# Add a first-level paragraph.
p = tf.add_paragraph()
p.text = 'X\'mas 1989'
p.level = 1
# Add a second-level paragraph.
p = tf.add_paragraph()
p.text = 'Guido began to write interpreter for Python.'
p.level = 2

# Save the presentation.
pres.save('test.pptx')
```
