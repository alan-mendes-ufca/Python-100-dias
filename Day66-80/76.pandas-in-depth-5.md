### Cálculo do crescimento mês a mês e ano a ano

Anteriormente usamos um exemplo de contagem de vendas mensais. Poderíamos fazer agregação agrupada com `groupby` ou gerar uma tabela dinâmica com `pivot_table`, conforme mostrado abaixo.

```python
sales_df = pd.read_excel('data/2020-sales-data.xlsx')
sales_df['Month'] = sales_df['Sales Date'].dt.month
sales_df['Sales Amount'] = sales_df['Price'] * sales_df['Sales Quantity']
result_df = sales_df.pivot_table(index='Month', values='Sales Amount', aggfunc='sum')
result_df.rename(columns={'Sales Amount': 'Current Month Sales'}, inplace=True)

```

Depois de obter as vendas mensais, se precisarmos calcular o crescimento mês a mês, há duas maneiras aqui. A primeira maneira é usar o método `shift` para mover os dados, alinhar os dados do mês passado com os dados deste mês e, em seguida, calcular o crescimento mês a mês com `(current month sales - last month sales) / last month sales`.

```python
result_df['Previous Month Sales'] = result_df['Current Month Sales'].shift(1)
result_df
```

No exemplo acima, o parâmetro de `shift` é `1`, o que significa mover os dados uma célula para baixo. Claro, podemos usar `-1` para mover os dados uma célula para cima. Você provavelmente também pode pensar no seguinte: se tivéssemos dados de mais anos, poderíamos definir o parâmetro como `12` e, então, poderíamos calcular a comparação ano a ano entre cada mês deste ano e o mesmo mês do ano passado.

```python
result_df['MoM'] = (
    result_df['Current Month Sales'] - result_df['Previous Month Sales']
) / result_df['Previous Month Sales']
result_df.style.format(
    formatter={'Previous Month Sales': '{:.0f}', 'MoM': '{:.2%}'},
    na_rep='--------'
)
```

Uma segunda maneira mais simples é usar diretamente o método `pct_change` para calcular a variação percentual. Primeiro, vamos remover as colunas de vendas do mês anterior e mês a mês.

```python
result_df.drop(columns=['Previous Month Sales', 'MoM'], inplace=True)
```

A seguir, usamos o método `pct_change` do objeto `DataFrame` para calcular o valor mês a mês. Vale ressaltar que `pct_change` possui um parâmetro denominado `periods`, e seu valor padrão é `1`, que calcula a variação percentual entre itens adjacentes. Esse é exatamente o resultado mês a mês que desejamos. Se tivéssemos muitos anos de dados e alterássemos este parâmetro para `12`, poderíamos obter a mudança ano a ano entre anos adjacentes.

```python
result_df['MoM'] = result_df.pct_change()
result_df
```

Saída:

```
Result table with `MoM` calculated directly by `pct_change`.
```

### Cálculos de janela

O método `rolling` de um `DataFrame` nos permite colocar dados em uma janela e, em seguida, realizar cálculos nos dados dentro da janela. Por exemplo, se obtivermos dados recentes de ações de uma empresa e quisermos criar uma média móvel de 5 dias e uma média móvel de 10 dias, precisamos primeiro definir uma janela e depois calcular dentro dela. Vamos primeiro ler os dados de ações de 2022 do Baidu.

```python
baidu_df = pd.read_excel('data/2022-stock-data.xlsx', sheet_name='BIDU', index_col='Date')
baidu_df.sort_index(inplace=True)
baidu_df
```

Saída:

<img src="res/baidu_stock.png" style="zoom:50%;">

O `DataFrame` acima tem cinco colunas, `Open`, `High`, `Low`, `Close` e `Volume`, que representam o preço de abertura, preço mais alto, preço mais baixo, preço de fechamento e volume de negociação do estoque. A seguir, fazemos cálculos de janela nos dados de ações do Baidu.

```python
baidu_df.rolling(5).mean()
```

Saída:

<img src="res/baidu_stock_ma5.png" style="zoom:50%;">

Também podemos usar `rolling` em um `Series` e fazer cálculos dentro da janela. Por exemplo, podemos calcular a média móvel de 5 dias e a média móvel de 10 dias para o preço de fechamento (coluna `Close`), depois usar a função `merge` para montá-los em um `DataFrame` e, finalmente, desenhar um gráfico de média móvel dupla.

```python
close_ma5 = baidu_df.Close.rolling(5).mean()
close_ma10 = baidu_df.Close.rolling(10).mean()
result_df = pd.merge(close_ma5, close_ma10, left_index=True, right_index=True)
result_df.rename(columns={'Close_x': 'MA5', 'Close_y': 'MA10'}, inplace=True)
result_df.plot(kind='line', figsize=(10, 6))
plt.show()
```

Saída:

<img src="res/baidu_double_MA.png" style="zoom:50%;">

### Julgando Correlação

Nas estatísticas, geralmente usamos a covariância para medir o grau de mudança conjunta entre duas variáveis ​​aleatórias. Se valores grandes da variável $\small{X}$ correspondem principalmente a valores grandes de outra variável $\small{Y}$, e valores pequenos também correspondem a valores pequenos, então as duas variáveis ​​tendem a apresentar comportamento semelhante, e a covariância é positiva. Se valores grandes de uma variável correspondem principalmente a valores pequenos da outra variável, então as duas variáveis ​​tendem a apresentar comportamento oposto e a covariância é negativa. Simplificando, o sinal de covariância mostra a correlação entre duas variáveis. A variância é um caso especial de covariância, ou seja, a covariância de uma variável consigo mesma.

$$ cov(X,Y) = E((X - \mu)(Y - \upsilon)) = E(X \cdot Y) - \mu\upsilon $$

Se $\small{X}$ e $\small{Y}$ são estatisticamente independentes, então sua covariância é `0`, porque nesse caso:

$$ E(X \cdot Y) = E(X) \cdot E(Y) = \mu\upsilon $$

O tamanho da covariância depende do tamanho das variáveis, por isso geralmente não é fácil de explicar diretamente. Mas a forma normalizada de covariância pode mostrar a força da relação linear entre as variáveis. Nas estatísticas, o coeficiente de correlação produto-momento de Pearson é a forma normalizada de covariância. É utilizado para medir o grau de correlação linear entre duas variáveis ​​$\small{X}$ e $\small{Y}$, e seu valor está entre `-1` e `1`.

$$ \frac {cov(X, Y)} {\sigma_{X}\sigma_{Y}} $$

Ao estimar a covariância e o desvio padrão da amostra, obtemos o coeficiente de Pearson da amostra, geralmente representado pela letra grega $\small{\rho}$.

$$ \rho = \frac {\sum_{i=1}^{n}(X_i - \bar{X})(Y_i - \bar{Y})} {\sqrt{\sum_{i=1}^{n}(X_i - \bar{X})^2} \sqrt{\sum_{i=1}^{n}(Y_i - \bar{Y})^2}} $$

Quando usamos $\small{\rho}$ para julgar a correlação dos indicadores, geralmente seguimos duas etapas.

1. Julgue se os indicadores estão positivamente correlacionados, negativamente correlacionados ou não correlacionados.
   - Quando $\small{\rho \gt 0}$, pensamos que as variáveis ​​estão positivamente correlacionadas, o que significa que têm a mesma tendência.
   - Quando $\small{\rho \lt 0}$, pensamos que as variáveis ​​estão correlacionadas negativamente, o que significa que suas tendências são opostas.
   - Quando $\small{\rho \approx 0}$, pensamos que as variáveis ​​não estão correlacionadas, mas isso não significa que sejam estatisticamente independentes.
2. Julgue a força da correlação.
   - Quando o valor absoluto de $\small{\rho}$ está em $\small{[0.6,1]}$, pensamos que as variáveis ​​estão fortemente correlacionadas.
   - Quando o valor absoluto de $\small{\rho}$ está em $\small{[0.1,0.6)}$, pensamos que as variáveis ​​são fracamente correlacionadas.
   - Quando o valor absoluto de $\small{\rho}$ está em $\small{[0,0.1)}$, pensamos que não há correlação.

A correlação de Pearson é adequada quando:

1. As duas variáveis ​​​​têm uma relação linear e são dados contínuos.
2. As populações das duas variáveis ​​são normalmente distribuídas, ou próximas de uma distribuição unimodal normal.
3. As observações das duas variáveis ​​são pareadas e cada par é independente.

Aqui, mencionemos também o que fazer se os dois grupos de variáveis ​​não forem valores contínuos de uma população normal. Para dados em escala ordinal, podemos usar o coeficiente de correlação de postos de Spearman, cuja fórmula é:

$$ r_{s}=1-{\frac {6\sum d_{i}^{2}}{n(n^{2}-1)}} $$

onde $\small{d_{i}=\operatorname {R} (X_{i})-\operatorname {R} (Y_{i})}$ é a diferença entre as classificações das duas variáveis ​​em cada grupo de observação, e $\small{n}$ é o tamanho da amostra.

Para dados em escala nominal, podemos usar o teste do qui-quadrado para avaliar se eles estão relacionados. Na verdade, muitas vezes os valores contínuos também podem ser agrupados em classificações ou categorias discretas e, então, podemos usar o coeficiente de classificação de Spearman ou o teste do qui-quadrado para julgar a correlação.

O método `cov` de um `DataFrame` é usado para calcular a covariância, e o método `corr` é usado para calcular os coeficientes de correlação. O método `corr` possui um parâmetro denominado `method`, cujo valor padrão é `pearson`, que significa correlação de Pearson. Além disso, também podemos especificar `kendall` ou `spearman` para calcular o coeficiente de Kendall ou o coeficiente de correlação de classificação de Spearman.

O coeficiente de correlação de classificação de Spearman tem requisitos menos rigorosos nas condições dos dados do que a correlação de Pearson. Contanto que as observações das duas variáveis ​​sejam dados de classificação emparelhados ou dados convertidos de variáveis ​​contínuas em dados de classificação, podemos usar a correlação de classificação de Spearman, independentemente da distribuição da população ou do tamanho da amostra. Podemos pré-processar parte dos recursos da seguinte maneira e então calcular o coeficiente de correlação de classificação de Spearman.

```python
boston_df['CRIM'] = boston_df.CRIM.apply(lambda x: x // 5 if x < 25 else 5).map(int)
boston_df['ZN'] = pd.qcut(boston_df.ZN, q=[0, 0.75, 0.8, 0.85, 0.9, 0.95, 1], labels=np.arange(6))
boston_df['AGE'] = (boston_df.AGE // 20).map(int)
boston_df['DIS'] = (boston_df.DIS // 2.05).map(int)
boston_df['B'] = (boston_df.B // 66).map(int)
boston_df['PRICE'] = pd.qcut(boston_df.PRICE, q=[0, 0.15, 0.3, 0.5, 0.7, 0.85, 1], labels=np.arange(6))
boston_df[['CRIM', 'ZN', 'AGE', 'DIS', 'B', 'PRICE']].corr(method='spearman')
```

Saída:

<img src="res/boston_spearman_correlation.png" style="zoom:50%;">

Podemos ver que o preço da habitação tem uma correlação negativa bastante óbvia com a taxa de criminalidade (`CRIM`) e a idade da habitação (`AGE`), e uma correlação positiva fraca com o tamanho do lote habitacional (`ZN`). A correlação pode nos ajudar a encontrar soluções de negócios no trabalho real, ou seja, a encontrar os fatores relacionados que podem afetar ou alterar os resultados do trabalho.
