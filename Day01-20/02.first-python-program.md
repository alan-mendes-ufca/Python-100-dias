## Seu primeiro programa Python

Na lição anterior, aprendemos sobre o passado e o presente do Python e preparamos o ambiente interpretador necessário para executar programas Python. Acredito que todos já estão ansiosos para iniciar sua própria jornada de programação em Python, mas surge uma nova questão: onde devemos escrever programas em Python e como devemos executá-los?

### Ferramentas para escrever código

Abaixo explicaremos várias ferramentas que podem ser usadas para escrever e executar código Python. Todos podem escolher uma ferramenta adequada de acordo com suas necessidades. Claro, para iniciantes, eu pessoalmente recomendo o PyCharm, porque ele não precisa de muita configuração e também é muito poderoso, por isso é amigável para iniciantes. Se você já conhece o PyCharm ou já gosta dele, pode pular a introdução às outras ferramentas abaixo e ir direto para a parte sobre o PyCharm.

#### O ambiente interativo padrão

Abra o prompt de comando do Windows ou PowerShell, digite `python` e pressione a tecla `Enter`. Este comando nos leva a um ambiente interativo. Um ambiente interativo significa que inserimos uma linha de código e pressionamos `Enter`, e o código é executado imediatamente. Se o código produzir um resultado, o resultado será mostrado na janela, conforme mostrado abaixo.

```Bash
Python 3.10.10
Type "help", "copyright", "credits" or "license" for more information.
>>> 2 * 3
6
>>> 2 + 3
5
>>>
```

> **Observação**: Os usuários do macOS precisam abrir o Terminal e digitar `python3` para entrar no ambiente interativo.

Caso queira sair do ambiente interativo, você pode inserir `quit()` no ambiente interativo, conforme mostrado abaixo.

```Bash
>>> quit()
```

#### Um ambiente interativo melhor - IPython

O ambiente interativo mencionado acima não oferece uma experiência de usuário muito boa, e você pode sentir isso depois de experimentá-lo. Podemos substituí-lo pelo IPython, porque o IPython oferece recursos de edição e interação mais poderosos. Podemos usar a ferramenta de gerenciamento de pacotes do Python `pip` no prompt de comando ou Terminal para instalar o IPython, conforme mostrado abaixo.

```bash
pip install ipython
```

> **Dica**: Antes de usar o comando acima para instalar o IPython, você pode primeiro usar `pip config set global.index-url https://pypi.doubanio.com/simple` ou `pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple/` para alterar a fonte de download para o espelho Douban ou espelho Tsinghua na China. Caso contrário, o download e a instalação poderão ser muito lentos.

A seguir, você pode usar o comando abaixo para iniciar o IPython e entrar no ambiente interativo.

```bash
ipython
```

> **Observação**: Há também uma versão web do IPython chamada Jupyter. Iremos apresentá-lo quando precisarmos usá-lo.

### Olá, mundo

De acordo com a prática da indústria, o primeiro programa que escrevemos ao aprender qualquer linguagem de programação é imprimir `hello, world`, porque esta linha de código foi o primeiro código escrito pelo grande Dennis Ritchie (o pai da linguagem C, que desenvolveu o sistema operacional Unix junto com Ken Thompson) e Brian Kernighan (o inventor da linguagem awk) em seu livro imortal *The C Programming Language*. Abaixo está a versão Python.

```python
print('hello, world')
```

> **Observação**: Os parênteses e aspas simples no código acima devem ser inseridos no modo de entrada em inglês. Se você acidentalmente usar parênteses chineses ou aspas simples chinesas, erros como `SyntaxError: invalid character '（' (U+FF08)` ou `SyntaxError: invalid character '‘' (U+2018)` aparecerão quando você executar o código.

O código acima possui apenas uma instrução. Nesta instrução, usamos uma função chamada `print` e ela nos ajuda a gerar o conteúdo especificado. O `'hello, world'` entre parênteses da função `print` é uma string e representa um trecho de texto. Em Python, podemos usar aspas simples ou duplas para representar uma string. Ao contrário de linguagens de programação como C, C++ ou Java, as instruções em Python não precisam de ponto e vírgula no final. Em outras palavras, se quisermos escrever outra instrução, basta pressionar Enter e iniciar uma nova linha, conforme mostrado abaixo. Além disso, o código Python não precisa de uma função de entrada chamada `main` para ser executado. Fornecer uma função de entrada é algo que deve ser feito ao escrever código executável C, C++ ou Java, e muitos programadores estão familiarizados com isso, mas em Python isso não é necessário.

```python
print('hello, world')
print('goodbye, world')
```

Se não usarmos um ambiente de desenvolvimento integrado como o PyCharm, também podemos chamar o interpretador Python diretamente para executar um programa Python. Podemos salvar o código acima como um arquivo chamado `example01.py`. No Windows, suponhamos que o arquivo esteja em `C:\code`. Abrimos o Prompt de Comando ou PowerShell e digitamos o comando abaixo para executá-lo.

```powershell
python C:\code\example01.py
```

No macOS, suponha que nosso arquivo esteja em `/Users/Hao`, então podemos inserir o comando abaixo no Terminal para executar o programa.

```Bash
python3 /Users/Hao/example01.py
```

> **Dica**: Se o caminho for longo e você não quiser digitá-lo manualmente, você pode arrastar o arquivo diretamente para o Prompt de Comando ou Terminal, e o caminho completo do arquivo será inserido automaticamente.

Você pode tentar alterar o código acima. Por exemplo, substitua `hello, world` entre aspas simples por outro conteúdo ou escreva mais algumas declarações como esta e veja o resultado obtido. Um lembrete é que ao escrever código Python, é melhor escrever apenas uma instrução em cada linha. Embora possamos usar `;` como separador para escrever várias instruções em uma linha, isso torna o código muito feio e prejudica a legibilidade.

### Comente seu código

Os comentários são uma parte importante das linguagens de programação. Eles são usados ​​para explicar o que o código faz, melhorando assim a legibilidade. Claro, também podemos usar comentários para desabilitar código que não precisa ser executado no momento. Então, quando precisar usar esse código novamente, você só precisará remover os marcadores de comentários. Simplificando, **os comentários facilitam a compreensão do código, mas não afetam o resultado da execução**.

Python tem duas formas de comentários:

1. Comentários de linha única: comece com `#` e um espaço, podendo comentar a linha inteira após `#`.
2. Comentários multilinhas: começam com aspas triplas (geralmente aspas duplas) e terminam com aspas triplas, e geralmente são usados ​​para conteúdo explicativo multilinhas.

```python
"""
Your first Python program - hello, world

Version: 1.0
Author: Luo Hao
"""
# print('hello, world')
print("Hello, world!")
```
