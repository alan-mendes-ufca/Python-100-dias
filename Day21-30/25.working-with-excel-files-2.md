## Lendo e gravando arquivos Excel em Python, Parte 2

### Uma introdução ao Excel

Excel é um software de planilha desenvolvido pela Microsoft para Windows e macOS. Devido à sua interface intuitiva, excelente capacidade de cálculo e ferramentas gráficas, juntamente com um marketing de sucesso, o Excel sempre foi o software de processamento de dados mais popular em computadores pessoais. É claro que o Excel também tem muitos produtos concorrentes, como Planilhas Google, LibreOffice Calc e Numbers. Basicamente, esses produtos concorrentes também podem ser compatíveis com o Excel, pelo menos podem ler e gravar arquivos Excel mais recentes. É claro que estes não são o foco da nossa discussão. Dominar a operação de arquivos Excel com programas Python pode tornar o trabalho diário de automação de escritório mais fácil e agradável. Além disso, em muitos projetos comerciais, importar e exportar arquivos Excel são funções muito comuns.

Nesta lição, continuamos com outra biblioteca de terceiros, `openpyxl`, e primeiro a instalamos.

```bash
pip install openpyxl
```

A vantagem do `openpyxl` é que depois de abrirmos um arquivo Excel, podemos fazer operações de leitura e gravação nele e, em termos de conveniência, é melhor que `xlwt` e `xlrd`. Além disso, se quisermos fazer edição de estilo e cálculo de fórmulas, usar `openpyxl` é muito mais simples do que o método apresentado na lição anterior. Além disso, `openpyxl` suporta operações como pivô de dados e inserção de gráficos, portanto suas funções são muito poderosas. Um ponto que precisa ser enfatizado novamente é que `openpyxl` não oferece suporte à operação de arquivos Excel de versões anteriores ao Office 2007.

### Lendo arquivos Excel

Por exemplo, suponha que exista um arquivo Excel denominado `Alibaba 2020 Equity Data.xlsx` na pasta atual. Se quisermos ler e exibir seu conteúdo, podemos usar o código abaixo.

```python
import datetime

import openpyxl

# Load a workbook -> Workbook
wb = openpyxl.load_workbook('Alibaba 2020 Equity Data.xlsx')
# Get worksheet names
print(wb.sheetnames)
# Get a worksheet -> Worksheet
sheet = wb.worksheets[0]
# Get the cell range
print(sheet.dimensions)
# Get the row count and column count
print(sheet.max_row, sheet.max_column)

# Get the value of a specific cell
print(sheet.cell(3, 3).value)
print(sheet['C3'].value)
print(sheet['G255'].value)

# Get multiple cells, returned as nested tuples
print(sheet['A2:C5'])

# Read the data in all cells
for row_ch in range(2, sheet.max_row + 1):
    for col_ch in 'ABCDEFG':
        value = sheet[f'{col_ch}{row_ch}'].value
        if type(value) == datetime.datetime:
            print(value.strftime('%YDay %d, % mm, yyyy'), end='\t')
        elif type(value) == int:
            print(f'{value:<10d}', end='\t')
        elif type(value) == float:
            print(f'{value:.4f}', end='\t')
        else:
            print(value, end='\t')
    print()
```

> **Dica**: Para executar o exemplo, use uma planilha própria com estrutura semelhante a `Alibaba 2020 Equity Data.xlsx`; o link de armazenamento da fonte original foi removido por ser pouco acessível fora da China.

Há um ponto que precisa ser lembrado a todos. `openpyxl` possui duas maneiras de obter uma célula especificada. Uma é através do método `cell`. Deve-se observar que o índice de linha e o índice de coluna deste método começam em `1`. Isso corresponde ao hábito de quem está acostumado com o Excel. A outra é por meio da operação de índice. Ao especificar a coordenada da célula, como `C3` e `G255`, também podemos obter a célula correspondente. Então, através do atributo `value` do objeto célula, podemos obter o valor da célula. Através do código acima, acredito que todos também notaram que através de operações de slice como `sheet['A2:C5']` ou `sheet['A2':'C5']`, podemos obter múltiplas células. Esta operação retorna tuplas aninhadas, o que equivale a obter várias linhas e colunas.

### Escrevendo arquivos Excel

A seguir, vamos usar `openpyxl` para escrever um arquivo Excel.

```python
import random

import openpyxl

# Step 1: create a workbook
wb = openpyxl.Workbook()

# Step 2: add a worksheet
sheet = wb.active
sheet.title = 'End of Period Results'

titles = ('name', 'Chinese', 'Mathematics', 'English')
for col_index, title in enumerate(titles):
    sheet.cell(1, col_index + 1, title)

names = ('Guan Yu', 'Zhang Fei', 'Zhao Yun', 'Ma Chao', 'Huang Zhong')
for row_index, name in enumerate(names):
    sheet.cell(row_index + 2, 1, name)
    for col_index in range(2, 5):
        sheet.cell(row_index + 2, col_index, random.randrange(50, 101))

# Step 4: save the workbook
wb.save('Exam Score Sheet.xlsx')
```

### Ajustando estilos e cálculo de fórmulas

Ao utilizar `openpyxl` para operar o Excel, se quisermos ajustar o estilo de uma célula, podemos operar diretamente através das propriedades do objeto célula, o objeto `Cell`. As propriedades do objeto de célula incluem fonte, alinhamento, borda e assim por diante. Para obter detalhes, você pode consultar [documentação oficial de `openpyxl`](https://openpyxl.readthedocs.io/en/stable/index.html). Ao usar `openpyxl`, se precisarmos fazer o cálculo da fórmula, podemos fazê-lo totalmente da mesma forma que no Excel. O código é mostrado abaixo.

```python
import openpyxl
from openpyxl.styles import Font, Alignment, Border, Side

# Alignment settings
alignment = Alignment(horizontal='center', vertical='center')
# Border line style
side = Side(color='ff7f50', style='mediumDashed')

wb = openpyxl.load_workbook('Exam Score Sheet.xlsx')
sheet = wb.worksheets[0]

# Adjust row height and column width
sheet.row_dimensions[1].height = 30
sheet.column_dimensions['E'].width = 120

sheet['E1'] = 'average score'
# Set the font
sheet.cell(1, 5).font = Font(size=18, bold=True, color='ff1493', name='Chinese Italic')
# Set the alignment
sheet.cell(1, 5).alignment = alignment
# Set the border
sheet.cell(1, 5).border = Border(left=side, top=side, right=side, bottom=side)
for i in range(2, 7):
    # Calculate the average score for each student with a formula
    sheet[f'E{i}'] = f'=average(B{i}:D{i})'
    sheet.cell(i, 5).font = Font(size=12, color='4169e1', italic=True)
    sheet.cell(i, 5).alignment = alignment

wb.save('Exam Score Sheet.xlsx')
```

### Gerando Gráficos Estatísticos

Com a biblioteca `openpyxl` também podemos inserir gráficos diretamente no Excel. O processo geral é bastante semelhante à inserção de gráficos no próprio Excel. Podemos criar um objeto gráfico do tipo desejado, configurá-lo através de suas propriedades e, o mais importante, vincular dados a ele, ou seja, o que o eixo horizontal representa, o que o eixo vertical representa e quais são os valores reais. Finalmente, o objeto gráfico pode ser adicionado à planilha. O código é mostrado abaixo.

```python
from openpyxl import Workbook
from openpyxl.chart import BarChart, Reference

wb = Workbook(write_only=True)
sheet = wb.create_sheet()

rows = [
    ('Category', 'Sales Group A', 'Sales Group B'),
    ('mobile phone', 40, 30),
    ('Tablet', 50, 60),
    ('Notebook', 80, 70),
    ('Peripherals', 20, 10),
]

# Add rows to the worksheet
for row in rows:
    sheet.append(row)

# Create a chart object
chart = BarChart()
chart.type = 'col'
chart.style = 10
# Set the chart title
chart.title = 'Sales Statistics Chart'
# Set the vertical-axis title
chart.y_axis.title = 'Sales volume'
# Set the horizontal-axis title
chart.x_axis.title = 'Product Categories'
# Set the data range
data = Reference(sheet, min_col=2, min_row=1, max_row=5, max_col=3)
# Set the category range
cats = Reference(sheet, min_col=1, min_row=2, max_row=5)
# Add data to the chart
chart.add_data(data, titles_from_data=True)
# Set the categories
chart.set_categories(cats)
chart.shape = 4
# Add the chart to a specified position in the worksheet
sheet.add_chart(chart, 'A10')

wb.save('demo.xlsx')
```
