```python
import matplotlib.pyplot as plt
```

### Criando uma Figura

A função `figure` em `pyplot` pode ser usada para criar uma figura. Ao criar uma figura, podemos utilizar o parâmetro `figsize` para especificar o tamanho da figura, cujo valor padrão é `[6.4, 4.8]`; podemos usar o parâmetro `dpi` para definir a resolução do desenho, pois `dpi` significa o número de pixels por polegada. Além disso, podemos utilizar o parâmetro `facecolor` para definir a cor de fundo da figura. O valor de retorno de `figure` é um objeto `Figure`, que representa a tela usada para plotagem. Com base na figura, podemos criar eixos para desenho.

```python
plt.figure(figsize=(8, 4), dpi=120, facecolor='darkgray')
```

### Criando Eixos

Podemos usar diretamente a função `subplot` de `pyplot` para criar eixos. Esta função retorna um objeto `Axes`. Os primeiros três parâmetros de `subplot` especificam em quantas linhas e colunas toda a figura é dividida e o índice dos eixos atuais. Seus valores padrão são todos `1`. Se não criarmos eixos explicitamente, a plotagem usará o padrão e apenas os eixos da figura. Se precisarmos de vários eixos em uma figura, podemos usar esta função. Claro, também podemos criar eixos através dos métodos `add_subplot` ou `add_axes` do objeto `Figure`. O primeiro funciona como `subplot`, enquanto o último pode produzir eixos aninhados.

```python
plt.subplot(2, 2, 1)
```

### Desenhando gráficos

#### Gráficos de linhas

Se não chamarmos `figure` ou `subplot` primeiro, a plotagem usará a figura e os eixos padrão. Para desenhar um gráfico de linhas, podemos usar a função `plot` de `pyplot` e especificar dados para os eixos horizontal e vertical. Os gráficos de linhas são mais adequados para observar tendências, especialmente quando o eixo horizontal representa o tempo. Podemos usar o parâmetro `color` para personalizar a cor da linha, `marker` para personalizar marcadores de ponto, `linestyle` para personalizar o estilo da linha e `linewidth` para personalizar a espessura da linha.

O código abaixo desenha uma curva senoidal. Aqui `marker='*'` define o marcador de ponto em forma de estrela e `color='red'` desenha a linha em vermelho.

```python
import numpy as np

x = np.linspace(-2 * np.pi, 2 * np.pi, 120)
y = np.sin(x)

plt.figure(figsize=(8, 4), dpi=120)
plt.plot(x, y, linewidth=2, marker='*', color='red')
plt.show()
```

Em seguida, tente o código a seguir e veja como funciona.

```python
fig = plt.figure(figsize=(10, 4), dpi=120)

ax = fig.add_axes((0.595, 0.6, 0.3, 0.25))
```

#### Gráficos de dispersão

Um gráfico de dispersão pode nos ajudar a compreender a relação entre duas variáveis. Se quisermos entender a relação entre três variáveis, podemos transformar o gráfico de dispersão em um gráfico de bolhas. No código abaixo, as matrizes `x` e `y` representam a renda mensal e as despesas mensais com compras online. Se quisermos saber se existe uma correlação entre eles, podemos desenhar um gráfico de dispersão como este.

```python
x = np.array([5550, 7500, 10500, 15000, 20000, 25000, 30000, 40000])
y = np.array([800, 1800, 1250, 2000, 1800, 2100, 2500, 3500])

plt.figure(figsize=(6, 4), dpi=120)
```

#### Gráficos de barras

Ao comparar diferenças nos dados, os gráficos de barras são uma escolha muito boa. Podemos usar a função `bar` em `pyplot` para gerar um gráfico de barras e a função `barh` para gerar um gráfico de barras horizontais. Primeiro vamos preparar alguns dados.

```python
x = np.arange(4)
y1 = np.random.randint(20, 50, 4)
y2 = np.random.randint(10, 60, 4)
```

O código para desenhar um gráfico de barras é:

```python
plt.figure(figsize=(6, 4), dpi=120)
plt.bar(x - 0.1, y1, width=0.2, label='Sales Group A')
plt.bar(x + 0.1, y2, width=0.2, label='Sales Group B')
plt.xticks(x, labels=['Q1', 'Q2', 'Q3', 'Q4'])
plt.legend()
plt.show()
```

```python
labels = ['Q1', 'Q2', 'Q3', 'Q4']
```

#### Gráficos de pizza

Um gráfico de pizza é um gráfico estatístico que divide os dados em várias áreas em forma de leque. É usado principalmente para descrever relações relativas entre quantidades e frequências. O tamanho de cada setor em um gráfico de pizza representa a proporção da quantidade que representa. Quando precisamos mostrar a composição, gráficos de pizza, mapas de árvore e gráficos em cascata não são escolhas ruins. Podemos usar a função `pie` em `pyplot` para desenhar um gráfico de pizza.

```python
data = np.random.randint(100, 500, 7)
labels = ['Apple', 'Banana', 'Peach', 'Lychee', 'Pomegranate', 'Mangosteen', 'Durian']

plt.figure(figsize=(5, 5), dpi=120)
plt.pie(
    data,
    autopct='%.1f%%',
    radius=1,
    pctdistance=0.8,
    colors=np.random.rand(7, 3),
    textprops=dict(fontsize=8, color='black'),
    wedgeprops=dict(linewidth=1, width=0.35),
    labels=labels
)
plt.title('Share of Fruit Sales')
plt.show()
```

#### Histogramas

Nas estatísticas, um histograma é um gráfico usado para mostrar a distribuição dos dados. Os dados abaixo representam a altura de 100 alunos do sexo masculino em uma escola. Se quisermos saber a distribuição, podemos usar um histograma.

```python
heights = np.array([
    170, 163, 174, 164, 159, 168, 165, 171, 171, 167,
    165, 161, 175, 170, 174, 170, 174, 170, 173, 173,
    167, 169, 173, 153, 165, 169, 158, 166, 164, 173,
    162, 171, 173, 171, 165, 152, 163, 170, 171, 163,
    165, 166, 155, 155, 171, 161, 167, 172, 164, 155,
    168, 171, 173, 169, 165, 162, 168, 177, 174, 178,
    161, 180, 155, 155, 166, 175, 159, 169, 165, 174,
    175, 160, 152, 168, 164, 175, 168, 183, 166, 166,
    182, 174, 167, 168, 176, 170, 169, 173, 177, 168,
    172, 159, 173, 185, 161, 170, 170, 184, 171, 172
])
```

Podemos usar a função `hist` de `pyplot` para desenhar o histograma. O parâmetro `bins` representa o esquema de binning que usamos.

```python
plt.figure(figsize=(6, 4), dpi=120)
plt.hist(heights, bins=np.arange(145, 196, 5), color='darkcyan')
plt.xlabel('Height')
plt.ylabel('Probability Density')
plt.show()
```

Saída:

<img src="res/hist_count.png" style="zoom:50%;">

Ao desenhar um histograma, se alterarmos o parâmetro `density` de `hist` para `True` e também definirmos `cumulative=True`, o eixo vertical exibirá a densidade de probabilidade e o gráfico mostrará a distribuição cumulativa de probabilidade.

```python
plt.figure(figsize=(6, 4), dpi=120)
plt.hist(heights, bins=np.arange(145, 196, 5), color='darkcyan', density=True, cumulative=True)
plt.xlabel('Height')
plt.ylabel('Probability')
```

#### Gráficos de caixa

Um box plot é um gráfico estatístico usado para mostrar a distribuição de um conjunto de dados. A borda superior da caixa é o quartil superior, a borda inferior é o quartil inferior, a linha no meio é a mediana e a altura da caixa é o intervalo interquartil. Os bigodes se estendem até os valores máximo e mínimo, e os pontos fora dos bigodes são discrepantes.

Podemos usar a função `boxplot` de `pyplot` para desenhar um box plot.

```python
data = np.random.randint(0, 100, 47)
data = np.append(data, 160)
data = np.append(data, 200)
data = np.append(data, -50)

plt.figure(figsize=(6, 4), dpi=120)
plt.boxplot(data, whis=1.5, showmeans=True, notch=True)
plt.ylim([-100, 250])
plt.xticks([1], labels=['data'])
plt.show()
```

### Exibindo e salvando gráficos

Podemos usar a função `show` de `pyplot` para exibir um gráfico, o que já fizemos acima. Se quisermos salvar um gráfico, podemos usar a função `savefig`. Uma coisa a observar é que se quisermos salvar e exibir um gráfico, devemos chamar primeiro `savefig` e depois `show`, porque depois de `show` a figura já foi divulgada.

```python
plt.savefig('chart.png')
plt.show()
```
