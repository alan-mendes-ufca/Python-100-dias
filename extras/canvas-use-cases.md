## Aprendendo programação com Canvas

Esta nota explica como um espaço de trabalho de codificação interativo no estilo canvas pode ajudar na prática de programação.

### Casos de uso comuns

1. Edição e refinamento de código em tempo real
2. Aprendendo sintaxe e conceitos de programação a partir de exemplos
3. Compreendendo algoritmos passo a passo
4. Experimentando implementações alternativas
5. Combinando código e explicação em um só lugar

A ideia central é que uma superfície de edição interativa torne mais fácil:

- iterar no código rapidamente
- faça perguntas focadas
- comparar versões
- entenda por que uma mudança ajuda

Para aprendizagem, o fluxo de trabalho mais útil geralmente é:

1. escreva um pequeno exemplo
2. peça uma explicação
3. refinar o código
4. repita até que a lógica esteja clara
