## Visão geral do front-end da web

> **Nota**: Algumas ilustrações usadas nesta lição vêm de *[HTML e CSS: projetar e construir sites](https://www.amazon.com/dp/1118008189)* de Jon Duckett. Este é um livro muito bom para começar com o desenvolvimento frontend. Os leitores interessados podem procurar este livro na Amazon ou em outras livrarias online.

HTML é uma linguagem usada para descrever páginas da web. Seu nome completo é Hyper-Text Markup Language. O texto, botões, imagens, vídeos e outros elementos que vemos quando navegamos em uma página web são todos escritos em HTML e mostrados pelo navegador.

### Breve história do HTML

1. Outubro de 1991: um documento informal do CERN tornou públicas pela primeira vez 18 tags HTML. O autor deste documento foi o físico Tim Berners-Lee, portanto ele é o inventor da World Wide Web e também o presidente do W3C.
2. Novembro de 1995: HTML 2.0 foi lançado como RFC 1866.
3. Janeiro de 1997: HTML 3.2 foi lançado como recomendação do W3C.
4. Dezembro de 1997: HTML 4.0 foi lançado como recomendação do W3C.
5. Dezembro de 1999: HTML 4.01 foi lançado como recomendação do W3C.
6. Janeiro de 2008: HTML5 foi publicado pelo W3C como um rascunho de trabalho.
7. Maio de 2011: o W3C avançou o HTML5 para o estágio de Última Chamada.
8. Dezembro de 2012: o W3C designou o HTML5 como uma recomendação candidata.
9. Outubro de 2014: HTML5 foi lançado como uma recomendação estável do W3C. Isso significa que a padronização do HTML5 foi concluída.

#### Novos recursos do HTML5

1. Introduzido suporte nativo de multimídia (tags `audio` e `video`)
2. Conteúdo programável introduzido (tag `canvas`)
3. Introduzido suporte da web semântica (`article`, `aside`, `details`, `figure`, `footer`, `header`, `nav`, `section`, `summary` e outras tags)
4. Introduzidos novos controles de formulário (calendário, e-mail, pesquisa, controle deslizante e assim por diante)
5. Introduzido melhor suporte para armazenamento offline (`localStorage` e `sessionStorage`)
6. Introduzido suporte para localização, arrastar e soltar, WebSocket, tarefas em segundo plano e assim por diante

#### Estrutura Progressiva - [Vue.js](https://cn.vuejs.org/)

Esta é uma estrutura obrigatória para desenvolvimento separado frontend-backend (renderização frontend).

##### Início rápido

1. Importe o arquivo JavaScript do Vue. Ainda recomendamos carregá-lo de um servidor CDN.

   ```html
   <script src="https://cdn.jsdelivr.net/npm/vue"></script>
   ```

2. Vinculação de dados (renderização declarativa).

   ```html
   <div id="app">
       <h1>{{ product }} inventory information</h1>
   </div>

   <script src="https://cdn.jsdelivr.net/npm/vue"></script>
   <script>
       const app = new Vue({
           el: '#app',
           data: {
               product: 'iPhone X'
           }
       });
   </script>
   ```

3. Condições e loops.

   ```html
   <div id="app">
       <h1>Inventory information</h1>
       <hr>
       <ul>
           <li v-for="product in products">
               {{ product.name }} - {{ product.quantity }}
               <span v-if="product.quantity === 0">
                   Sold out
               </span>
           </li>
       </ul>
   </div>
   
   <script src="https://cdn.jsdelivr.net/npm/vue"></script>
   <script>
       const app = new Vue({
           el: '#app',
           data: {
               products: [
                   {"id": 1, "name": "iPhone X", "quantity": 20},
                   {"id": 2, "name": "Huawei Mate20", "quantity": 0},
                   {"id": 3, "name": "Xiaomi Mix3", "quantity": 50}

           }
       });
   </script>
   ```

4. Propriedades computadas.

   ```html

       <hr>
       <ul>
           <li v-for="product in products">
               {{ product.name }} - {{ product.quantity }}

                   Sold out
               </span>
           </li>
       </ul>
       <h2>Total inventory: {{ totalQuantity }} units</h2>
   </div>

   <script src="https://cdn.jsdelivr.net/npm/vue"></script>
   <script>
       const app = new Vue({
           el: '#app',
           data: {
               products: [
                   {"id": 1, "name": "iPhone X", "quantity": 20},
                   {"id": 2, "name": "Huawei Mate20", "quantity": 0},
                   {"id": 3, "name": "Xiaomi Mix3", "quantity": 50}
               ]
           },
           computed: {
               totalQuantity() {
                   return this.products.reduce((sum, product) => {
                       return sum + product.quantity
                   }, 0);
               }
           }
       });

   ```

