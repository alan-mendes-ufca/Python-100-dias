## Operadores em Python

Python oferece suporte a muitos tipos de operadores. A tabela abaixo lista os operadores Python de alta precedência a baixa precedência. Com variáveis e operadores podemos construir todo tipo de expressões para resolver problemas reais. Na ciência da computação, **uma expressão é uma unidade de sintaxe em um programa de computador. É composto por uma ou mais constantes, variáveis, funções e operadores, e a linguagem de programação pode interpretá-lo e calculá-lo para obter outro valor**. Não importa se você não entendeu esta frase por enquanto, mas você deve saber uma coisa: não importa qual linguagem de programação você use, construir expressões é muito importante.

| Operador | Descrição |
| --- | --- |
| `[]`, `[:]` | indexação, fatiamento |
| `**` | exponenciação |
| `~`, `+`, `-` | NOT bit a bit, unário mais, unário menos |
| `*`, `/`, `%`, `//` | multiplicação, divisão, módulo, divisão de piso |
| `+`, `-` | adição, subtração |
| `>>`, `<<` | deslocamento para a direita, deslocamento para a esquerda |
| `&` | E bit a bit |
| `^`, ` | ` | XOR bit a bit, OU bit a bit |
| `<=`, `<`, `>`, `>=` | menor ou igual a, menor que, maior que, maior que ou igual a |
| `==`, `!=` | igual a, não igual a |
| `is`, `is not` | operadores de identidade |
| `in`, `not in` | operadores de adesão |
| `not`, `or`, `and` | operadores lógicos |
| `=`, `+=`, `-=`, `*=`, `/=`, `%=`, `//=`, `**=`, `&=`, ` | =`, `^=`, `>>=`, `<<=` | operadores de atribuição |

> **Observação**: Precedência significa a ordem de execução quando uma expressão contém vários operadores. Ao escrever código, se não tiver certeza sobre a precedência dos operadores em uma expressão, você pode usar parênteses para ter certeza de que a ordem é a desejada.

### Operadores Aritméticos

Python possui muitos operadores aritméticos. Além da adição, subtração, multiplicação e divisão familiares, há também o operador de divisão mínima, o operador módulo (resto) e o operador de exponenciação. O exemplo abaixo mostra como usar operadores aritméticos.

```python
"""
Arithmetic operators

Version: 1.0
Author: Luo Hao
"""
print(321 + 12)     # addition -> 333
print(321 - 12)     # subtraction -> 309
print(321 * 12)     # multiplication -> 3852
print(321 / 12)     # division -> 26.75
print(321 // 12)    # floor division -> 26
print(321 % 12)     # modulo -> 9
print(321 ** 12)    # exponentiation -> 1196906950228928915420617322241
```

Nas operações aritméticas, a multiplicação e a divisão são feitas antes da adição e da subtração. Isto é o mesmo que na matemática. Em outras palavras, a multiplicação e a divisão têm maior precedência do que a adição e a subtração. Se também houver exponenciação, a exponenciação terá maior precedência do que a multiplicação e a divisão. Se quiser alterar a ordem de execução das operações aritméticas, você pode usar parênteses no modo de entrada em inglês. A expressão entre parênteses será executada primeiro, conforme mostrado abaixo.

```python
"""
Operator precedence in arithmetic

Version: 1.0
Author: Luo Hao
"""
print(2 + 3 * 5)           # 17
print((2 + 3) * 5)         # 25
print((2 + 3) * 5 ** 2)    # 125
print(((2 + 3) * 5) ** 2)  # 625
```

### Operadores de Atribuição

O operador de atribuição é provavelmente o operador mais comum. Sua função é atribuir o valor do lado direito à variável do lado esquerdo. O operador de atribuição também pode ser combinado com os operadores aritméticos acima para formar operadores de atribuição compostos. Por exemplo, `a += b` é igual a `a = a + b` e `a *= a + 2` é igual a `a = a * (a + 2)`. O exemplo abaixo mostra o uso de operadores de atribuição e operadores de atribuição compostos.

```python
"""
Assignment and compound assignment operators

Version: 1.0
Author: Luo Hao
"""
a = 10
b = 3
a += b        # equivalent to a = a + b
a *= a + 2    # equivalent to a = a * (a + 2)
print(a)      # try to work out the result yourself
```

Uma expressão construída com o operador de atribuição não produz nenhum valor por si só. Em outras palavras, se você colocar uma expressão de atribuição na função `print` e tentar imprimir o valor dessa expressão, ocorrerá um erro de sintaxe. Para resolver este problema, Python 3.8 introduziu um novo operador de atribuição, `:=`. Chamamos-lhe operador de morsa. Você pode adivinhar por que tem esse nome. O operador morsa também atribui o valor do lado direito do operador à variável do lado esquerdo. A diferença é que o valor do lado direito também é o valor de toda a expressão. Veja o código abaixo e ficará claro.

```python
"""
The walrus operator

Version: 1.0
Author: Luo Hao
"""
# SyntaxError: invalid syntax
# print((a = 10))
print((a := 10))  # 10
print(a)          # 10
```

> **Dica**: Se você remover o comentário de `print((a = 10))`, o Python gerará `SyntaxError: invalid syntax`. Se você escrever acidentalmente `print(a = 10)`, obterá `TypeError: 'a' is an invalid keyword argument for print()`. Quando falarmos sobre funções mais tarde, esse erro fará mais sentido.

### Comparação e operadores lógicos

Os operadores de comparação também são chamados de operadores relacionais. Eles incluem `==`, `!=`, `<`, `>`, `<=` e `>=`, e acredito que todos podem entendê-los rapidamente. Uma coisa a lembrar é que a verificação da igualdade usa `==`. Observe que há dois sinais de igual aqui, porque `=` é o operador de atribuição de que acabamos de falar. A verificação da desigualdade usa `!=`, que é diferente do símbolo $\small{\neq}$ usado em livros didáticos de matemática. Python 2 já usou `<>` para significar diferente. No Python 3, usar `<>` gera `SyntaxError`. Os operadores de comparação produzem valores booleanos, `True` ou `False`.

Existem três operadores lógicos: `and`, `or` e `not`.

- O significado literal de `and` é "e", portanto, o operador `and` conecta dois valores booleanos ou expressões que produzem valores booleanos. Se os valores booleanos em ambos os lados forem `True`, o resultado será `True`. Se um deles for `False`, o resultado final será `False`.
- Se o valor booleano no lado esquerdo do operador `and` for `False`, não importa qual seja o lado direito, o resultado final será `False`. Neste momento, o lado direito é ignorado. O nome profissional para isso é processamento de curto-circuito. Se o lado direito de `and` for uma expressão, essa expressão não será executada.
- O significado literal de `or` é "ou", portanto, o operador `or` também conecta dois valores booleanos ou expressões que produzem valores booleanos. Se um dos lados for `True`, o resultado final será `True`.
- O operador `or` também possui comportamento de curto-circuito. Quando o valor do lado esquerdo for `True`, o lado direito será ignorado. Se o lado direito de `or` for uma expressão, essa expressão não será executada.
- O operador `not` pode ser seguido por um valor booleano. Se o valor ou expressão booleana após `not` for `True`, o resultado será `False`. Se o valor ou expressão booleana após `not` for `False`, o resultado será `True`.

```python
"""
Comparison and logical operators

Version: 1.0
Author: Luo Hao
"""
flag0 = 1 == 1
flag1 = 3 > 2
flag2 = 2 < 1
flag3 = flag1 and flag2
flag4 = flag1 or flag2
flag5 = not flag0
print('flag0 =', flag0)     # flag0 = True
print('flag1 =', flag1)     # flag1 = True
print('flag2 =', flag2)     # flag2 = False
print('flag3 =', flag3)     # flag3 = False
print('flag4 =', flag4)     # flag4 = True
print('flag5 =', flag5)     # flag5 = False
print(flag1 and not flag2)  # True
print(1 > 2 or 2 == 3)      # False
```

> **Observação**: Os operadores de comparação têm precedência mais alta que os operadores de atribuição, portanto `flag0 = 1 == 1` primeiro avalia `1 == 1` e produz o valor booleano `True` e, em seguida, atribui esse valor à variável `flag0`. A função `print` pode gerar vários valores. Os valores podem ser separados por `,` e são separados por espaços na saída por padrão.

### Aplicando Operadores e Expressões

#### Exemplo 1: Converter Fahrenheit em Celsius

> **Tarefa**: Insira uma temperatura Fahrenheit e converta-a para Celsius. A fórmula de conversão é $\small{C = (F - 32) / 1,8}$.

```python
"""
Convert Fahrenheit to Celsius

Version: 1.0
Author: Luo Hao
"""
f = float(input('Enter Fahrenheit: '))
c = (f - 32) / 1.8
print('%.1fF = %.1fC' % (f, c))
```

> **Nota**: A função `input` é usada para receber entradas do usuário do teclado. Como a entrada é sempre uma string, se você quiser tratá-la como um número de ponto flutuante em cálculos posteriores, você pode usar o método de conversão de tipo que aprendemos na lição anterior e usar a função `float` para converter o valor `str` em `float`.

No código acima, formatamos a saída do conteúdo pela função `print`. A sequência de saída possui dois espaços reservados `%.1f`. Esses dois espaços reservados serão substituídos pelos dois valores `float` em `(f, c)` após `%`, e um dígito será mantido após o ponto decimal. Se uma string contiver um espaço reservado `%d`, usaremos um valor do tipo `int` para substituí-lo. Se contiver um espaço reservado `%s`, ele será substituído por um valor do tipo `str`.

Além do método de formatação acima, Python também pode usar o método abaixo para saída formatada. Fornecemos uma string com espaços reservados, e `f` antes da string significa que a string precisa de formatação. Por enquanto, você pode primeiro tratar `{f:.1f}` e `{c:.1f}` como `{f}` e `{c}`. Isso significa que os valores das variáveis `f` e `c` substituirão esses dois espaços reservados quando a string for impressa. O `:.1f` significa que este é um número de ponto flutuante e um dígito é mantido após o ponto decimal.

```python
"""
Convert Fahrenheit to Celsius

Version: 1.1
Author: Luo Hao
"""
f = float(input('Enter Fahrenheit: '))
c = (f - 32) / 1.8
print(f'{f:.1f}F = {c:.1f}C')
```

#### Exemplo 2: Calcule a circunferência e a área de um círculo

> **Tarefa**: Insira o raio $\small{r}$ de um círculo e calcule sua circunferência $\small{2 \pi r}$ e sua área $\small{\pi r^{2}}$.

```python
"""
Calculate the circumference and area of a circle

Version: 1.0
Author: Luo Hao
"""
radius = float(input('Enter the radius: '))
perimeter = 2 * 3.1416 * radius
area = 3.1416 * radius * radius
print('Perimeter: %.2f' % perimeter)
print('Area: %.2f' % area)
```

Python possui um módulo integrado chamado `math`, e esse módulo define uma variável chamada `pi`. Seu valor é a constante matemática pi. Se quisermos usar o `pi` integrado do Python, podemos fazer uma pequena alteração no código acima.

```python
"""
Calculate the circumference and area of a circle

Version: 1.1
Author: Luo Hao
"""
import math

radius = float(input('Enter the radius: '))
perimeter = 2 * math.pi * radius
area = math.pi * radius ** 2
print(f'Perimeter: {perimeter:.2f}')
print(f'Area: {area:.2f}')
```

> **Observação**: `import math` significa importar o módulo `math`. Somente após importar esse módulo podemos usar `math.pi` para obter o valor de pi.

Na verdade, existe outro método de formatação aqui. É um novo recurso adicionado no Python 3.8. Basta olhar o código abaixo e ficará claro.

```python
"""
Calculate the circumference and area of a circle

Version: 1.2
Author: Luo Hao
"""
import math

radius = float(input('Enter the radius: '))  # input: 5.5
perimeter = 2 * math.pi * radius
area = math.pi * radius ** 2
print(f'{perimeter = :.2f}')  # output: perimeter = 34.56
print(f'{area = :.2f}')       # output: area = 95.03
```

> **Observação**: Se o valor da variável `a` for `9.87`, então o valor da string `f'{a = }'` será `a = 9.87`; e o valor da sequência `f'{a = :.1f}'` é `a = 9.9`. Este método de formatação imprime o nome da variável e o valor da variável ao mesmo tempo.

#### Exemplo 3: Determine se um ano é bissexto

Tarefa: insira um ano após 1582 e determine se esse ano é bissexto.

```python
"""
Read a year and output True for a leap year, False otherwise

Version: 1.0
Author: Luo Hao
"""
year = int(input('Enter a year: '))
is_leap = year % 4 == 0 and year % 100 != 0 or year % 400 == 0
print(f'{is_leap = }')
```

> **Nota**: No calendário gregoriano, que é o calendário civil que usamos hoje, a regra para determinar um ano bissexto é: 1. Um ano que não é divisível por 4 é um ano comum. 2. Um ano divisível por 4, mas não divisível por 100, é um ano bissexto. 3. Um ano divisível por 400 é um ano bissexto. O calendário gregoriano foi introduzido pelo Papa Gregório XIII em outubro de 1582 como uma revisão e substituição do calendário juliano, por isso tenha isso em mente ao inserir um ano. O código acima usa `%` para determinar se `year` é um múltiplo de `4`, `100` e `400` e, em seguida, usa `and` e Operadores `or` combinam as três condições: as duas primeiras condições devem ser satisfeitas, enquanto a terceira condição precisa apenas de um lado da condição combinada para ser satisfeita.
