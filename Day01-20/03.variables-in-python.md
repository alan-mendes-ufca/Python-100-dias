## Variáveis ​​em Python

Para iniciantes que desejam aprender programação, há duas questões que eles podem realmente querer saber. Uma delas é: "O que é um programa de computador?" A outra é: "O que a escrita de programas de computador pode fazer?" Deixe-me primeiro falar sobre minha compreensão dessas duas questões: **um programa é uma coleção ordenada de dados e instruções**, e **escrever programas significa usar dados e instruções para controlar o computador e fazê-lo fazer o que queremos**. Hoje, por que tantas pessoas escolhem Python para escrever programas? Porque Python é bastante simples e poderoso. Comparado com linguagens de programação como C, C++ e Java, Python é mais amigável para iniciantes e não profissionais, e muitos problemas podem ser resolvidos de maneira simples e elegante em Python. A seguir, começaremos com os elementos mais básicos da linguagem e conheceremos o Python e aprenderemos como usá-lo.

### Alguns princípios básicos

Antes de começarmos a aprender programação Python de forma sistemática, vamos primeiro falar sobre alguns conhecimentos básicos de informática. Um sistema de hardware de computador geralmente é composto de cinco partes principais: **unidade aritmética**, **unidade de controle**, **memória**, **dispositivos de entrada** e **dispositivos de saída**. Entre elas, a unidade aritmética e a unidade de controle juntas são o que costumamos chamar de **unidade central de processamento** (CPU). Sua função é executar todos os tipos de instruções aritméticas e de controle. Como acabamos de dizer, um programa é uma coleção de instruções. Escrever um programa significa organizar de alguma forma uma série de instruções e depois usar essas instruções para controlar o computador e fazê-lo fazer o que queremos.

A memória pode ser dividida em **memória interna** e **memória externa**. O primeiro é o que costumamos chamar de RAM. É o espaço de armazenamento que a CPU pode endereçar diretamente. Enquanto um programa está em execução, os dados e instruções relacionados precisam ser carregados na memória. Dispositivos de entrada e dispositivos de saída costumam ser chamados juntos de dispositivos de E/S. Teclados, mouses, microfones e câmeras são dispositivos de entrada típicos, enquanto monitores, impressoras, alto-falantes e assim por diante são dispositivos de saída típicos. Atualmente, a maioria dos computadores que usamos segue a **arquitetura von Neumann**. Este tipo de computador tem dois pontos principais: primeiro, **a memória é separada da CPU**; segundo, **os dados são codificados em formato binário**.

Binário é um método de contagem que “carrega um para cada dois”. Em essência, é o mesmo método de contagem decimal que as pessoas usam, que “carrega um para cada dez”. Como os humanos têm dez dedos, usamos a contagem decimal. Depois que todos os dez dedos forem usados, só podemos usar o transporte para representar números maiores. Claro, existem exceções. Os maias também podem ter usado os dedos dos pés porque muitas vezes andavam descalços, por isso usaram um sistema de contagem de base 20. Devido a esta forma de contagem, o calendário maia era diferente do calendário que usamos na vida diária. De acordo com o calendário maia, 2012 foi o último ano da chamada "idade solar" anterior e 2013 foi o início de uma nova "idade solar". Mais tarde, isso foi passado de boca em boca e se transformou na afirmação absurda de que “2012 foi o fim do mundo previsto pelos maias”. Hoje, muitas pessoas acham que o lento desenvolvimento da civilização maia teve algo a ver com o uso da base 20.

Para computadores, o binário é a coisa mais fácil de implementar em dispositivos físicos, porque a alta tensão pode representar `1` e a baixa tensão pode representar `0`. Nem todo mundo que escreve programas precisa estar familiarizado com binário ou com conversão entre decimal, binário, octal e hexadecimal. Na maioria das vezes, ainda podemos escrever programas mesmo que não compreendamos esse conhecimento. No entanto, devemos saber que os computadores contam em binário e, não importa que tipo de dado seja, uma vez que entra na memória do computador, ele existe na forma binária.

> **Observação**: Em relação à contagem binária e como ela é convertida de e para outros sistemas numéricos, você pode consultar livros como *Introdução aos Computadores* ou *Cultura da Computação*. Você pode encontrar o conhecimento relacionado lá, portanto não será explicado em detalhes aqui. Os leitores que não têm clareza sobre isso podem estudá-lo por conta própria.

### Variáveis ​​e Tipos

Se quisermos armazenar dados na memória do computador, primeiro precisamos falar sobre o conceito de variáveis. Nas linguagens de programação, **variáveis ​​são portadoras de dados**. Simplificando, uma variável é um espaço de memória usado para salvar dados e **o valor de uma variável pode ser lido e alterado**. Esta é a base de toda computação e controle. Os computadores podem processar muitos tipos de dados. O tipo mais comum são os números, mas além dos números também existem texto, imagens, áudio, vídeo e muitos outros tipos de dados. Embora todos os dados existam em formato binário dentro de um computador, podemos usar variáveis ​​de diferentes tipos para mostrar as diferenças entre os tipos de dados. Python tem muitos tipos de dados integrados e também nos permite definir novos tipos de dados. Falaremos sobre isso mais tarde. Vamos primeiro entender vários dos tipos de dados mais comumente usados ​​em Python.

1. Inteiro (`int`): Python pode lidar com números inteiros de qualquer tamanho e suporta binário (por exemplo, `0b100`, que é `4` em decimal), octal (por exemplo, `0o100`, que é `64` em decimal), decimal (`100`) e notação hexadecimal (`0x100`, que é `256` em decimal). Execute o código abaixo e veja o que ele gera.

    ```python
    print(0b100)  # binary integer
    print(0o100)  # octal integer
    print(100)    # decimal integer
    print(0x100)  # hexadecimal integer
    ```

2. Ponto flutuante (`float`): Os números de ponto flutuante são números decimais. Eles são chamados de números de ponto flutuante porque na notação científica a posição do ponto decimal pode mudar. Além da notação normal, como `123.456`, os flutuadores também suportam notação científica, como `1.23456e2`, que significa $\small{1,23456 \times 10^{2}}$. Execute o código abaixo e veja o que ele gera.

    ```python
    print(123.456)    # standard decimal notation
    print(1.23456e2)  # scientific notation
    ```

3. String (`str`): Uma string é qualquer pedaço de texto entre aspas simples ou duplas, como `'hello'` ou `"hello"`.

4. Booleano (`bool`): O tipo Booleano possui apenas dois valores, `True` e `False`. Pode ser usado para representar "sim" e "não" no mundo real, o "verdadeiro" e o "falso" das proposições, estados "bons" e "ruins", níveis "altos" e "baixos", e assim por diante. Se uma variável tiver apenas dois estados possíveis, podemos usar o tipo booleano.

### Nomeando Variáveis

Para cada variável precisamos dar-lhe um nome, assim como cada um de nós tem o seu próprio nome. Em Python, os nomes das variáveis ​​precisam seguir as regras e convenções abaixo.

- Regras:
  - Regra 1: Os nomes das variáveis ​​são compostos de **letras**, **dígitos** e **sublinhados** e não podem começar com um dígito. Deve ser explicado que as “letras” aqui significam caracteres Unicode. Unicode é um conjunto de caracteres universal e inclui a maioria dos sistemas de escrita do mundo. Isso significa que letras chinesas, japonesas, gregas e assim por diante podem aparecer em nomes de variáveis. Mas caracteres especiais como `！`, `@` e `#` não podem aparecer em nomes de variáveis. Recomendamos fortemente que você entenda "letras" aqui como **usando apenas letras em inglês, tanto quanto possível**.
  - Regra 2: Python é uma linguagem de programação **que diferencia maiúsculas de minúsculas**. Simplificando, `A` maiúsculo e `a` minúsculo são duas variáveis ​​diferentes. Isso não é realmente uma regra, mas é algo que todos precisam prestar atenção.
  - Regra 3: Um nome de variável **não deve ter o mesmo nome de uma palavra-chave Python** e deve **evitar palavras reservadas do Python tanto quanto possível**. Palavras-chave são palavras com significado especial em programas Python, como `is`, `if`, `else`, `for`, `while`, `True` e `False`. Palavras reservadas significam principalmente os nomes de funções integradas, módulos integrados e assim por diante em Python, como `int`, `print`, `input`, `str`, `math` e `os`.
- Convenções:
  - Convenção 1: Use **letras minúsculas** e junte várias palavras com sublinhados.
  - Convenção 2: Variáveis ​​protegidas geralmente começam com um único sublinhado.
  - Convenção 3: Variáveis ​​privadas geralmente começam com dois sublinhados.

Você não precisa se preocupar com as convenções 2 e 3 por enquanto. Você os compreenderá naturalmente mais tarde. Claro que, como programador profissional, também é muito importante escolher nomes que mostrem claramente o seu significado. Isso mostra a qualidade profissional de um programador, e muitas entrevistas para cargos de desenvolvimento também se preocupam muito com esse ponto.

### Usando variáveis

Os exemplos abaixo mostram tipos de variáveis ​​e como usá-las.

```python
"""
Use variables to store data and perform arithmetic

Version: 1.0
Author: Luo Hao
"""
a = 45        # define variable a and assign 45
b = 12        # define variable b and assign 12
print(a, b)   # 45 12
print(a + b)  # 57
print(a - b)  # 33
print(a * b)  # 540
print(a / b)  # 3.75
```

Em Python, podemos usar a função `type` para verificar o tipo de uma variável. Na programação, o conceito de função é muito semelhante ao conceito de função em matemática. Acredito que todos estão familiarizados com funções matemáticas. Eles incluem um nome de função, variáveis ​​independentes e variáveis ​​dependentes. Se você não entende o conceito de função no momento, tudo bem. Explicaremos a definição e o uso de funções posteriormente.

```python
"""
Use the type function to inspect variable types

Version: 1.0
Author: Luo Hao
"""
a = 100
b = 123.45
c = 'hello, world'
d = True
print(type(a))  # <class 'int'>
print(type(b))  # <class 'float'>
print(type(c))  # <class 'str'>
print(type(d))  # <class 'bool'>
```

Podemos alterar o tipo de uma variável através das funções integradas do Python. Abaixo estão algumas funções comuns relacionadas a tipos de variáveis.

- `int()`: converte um número ou string em um número inteiro e também pode especificar a base.
- `float()`: converta uma string em um número de ponto flutuante quando possível.
- `str()`: converte o objeto especificado em formato de string e um método de codificação também pode ser especificado.
- `chr()`: converte um código de caractere inteiro em uma sequência de um caractere.
- `ord()`: converte uma string de um caractere em seu código de caractere inteiro.

O exemplo abaixo mostra a conversão de tipo em Python.

```python
"""
Type conversion for variables

Version: 1.0
Author: Luo Hao
"""
a = 100
b = 123.45
c = '123'
d = '100'
e = '123.45'
f = 'hello, world'
g = True
print(float(a))         # convert int 100 to float -> 100.0
print(int(b))           # convert float 123.45 to int -> 123
print(int(c))           # convert string '123' to int -> 123
print(int(c, base=16))  # treat string '123' as hexadecimal -> 291
print(int(d, base=2))   # treat string '100' as binary -> 4
print(float(e))         # convert string '123.45' to float -> 123.45
print(bool(f))          # non-empty string converts to True
print(int(g))           # True converts to 1
print(chr(a))           # integer 100 converts to 'd'
print(ord('d'))         # character 'd' converts to 100
```

> **Observação**: Ao converter de `str` para `int`, você pode usar o parâmetro `base` para especificar o sistema numérico e converter a string como um número inteiro nessa base. Ao converter de `str` para `bool`, desde que a string tenha conteúdo e não seja `''` ou `""`, o valor booleano resultante será `True`. Ao converter de `bool` para `int`, `True` se torna `1` e `False` se torna `0`. Tanto no conjunto de caracteres ASCII quanto no conjunto de caracteres Unicode, o caractere `'d'` corresponde ao código `100`.
