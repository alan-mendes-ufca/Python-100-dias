## Lendo e gravando arquivos Excel em Python, Parte 1

### Uma introdução ao Excel

Excel é um software de planilha desenvolvido pela Microsoft para Windows e macOS. Devido à sua interface intuitiva, excelente capacidade de cálculo e ferramentas gráficas, juntamente com um marketing de sucesso, o Excel sempre foi o software de processamento de dados mais popular em computadores pessoais. É claro que o Excel também tem muitos produtos concorrentes, como Planilhas Google, LibreOffice Calc e Numbers. Basicamente, esses produtos concorrentes também podem ser compatíveis com o Excel, pelo menos podem ler e gravar arquivos Excel mais recentes. É claro que estes não são o foco da nossa discussão. Dominar a operação de arquivos Excel com programas Python pode tornar o trabalho diário de automação de escritório mais fácil e agradável. Além disso, em muitos projetos comerciais, importar e exportar arquivos Excel são funções muito comuns.

Python precisa de suporte de biblioteca de terceiros para funcionar com Excel. Se você deseja compatibilidade com versões do Excel anteriores a 2007, ou seja, arquivos `.xls`, você pode usar as bibliotecas de terceiros `xlrd` e `xlwt`, onde a primeira lê arquivos do Excel e a segunda os grava. Se estiver usando arquivos Excel mais recentes, ou seja, `.xlsx`, você pode usar a biblioteca `openpyxl`. É claro que essa biblioteca pode fazer mais do que apenas lidar com o Excel, pois também pode operar em outros arquivos de planilhas baseados no Office Open XML.

Nesta lição, abordamos primeiro as operações de arquivo Excel com base em `xlwt` e `xlrd`. Você pode instalar essas duas bibliotecas junto com o módulo auxiliar `xlutils` usando o comando abaixo.

```bash
pip install xlwt xlrd xlutils
```

### Lendo arquivos Excel

Por exemplo, se houver um arquivo Excel chamado `Alibaba 2020 Equity Data.xls` na pasta atual, se quisermos ler e exibir o conteúdo deste arquivo, podemos fazê-lo com o código mostrado abaixo.

```python
import xlrd

# Use xlrd.open_workbook to open the specified Excel file and get a Book object.
wb = xlrd.open_workbook('Alibaba 2020 Equity Data.xls')
# Get the names of all worksheets.
sheetnames = wb.sheet_names()
print(sheetnames)
# Get a Sheet object by worksheet name.
sheet = wb.sheet_by_name(sheetnames[0])
# Get the row count and column count of the worksheet.
print(sheet.nrows, sheet.ncols)
for row in range(sheet.nrows):
    for col in range(sheet.ncols):
        # Get the specified Cell object with sheet.cell.
        # Then get the value stored in the cell.
        value = sheet.cell(row, col).value
        # Format all rows other than the header row.
        if row > 0:
            # Convert the xldate value in the first column to a tuple,
            # then format it as year-month-day.
            if col == 0:
                # The second argument to xldate_as_tuple can only be 0 or 1.
                # 0 means dates are based on 1900-01-01, and 1 means 1904-01-01.
                value = xlrd.xldate_as_tuple(value, 0)
                value = f'{value[0]}Year{value[1]:>02d}moon{value[2]:>02d}day'
            # Format number values in the remaining columns with two decimal places.
            else:
                value = f'{value:.2f}'
        print(value, end='\t')
    print()
# Get the data type of the last cell.
# 0 - empty, 1 - string, 2 - number, 3 - date, 4 - boolean, 5 - error
last_cell_type = sheet.cell_type(sheet.nrows - 1, sheet.ncols - 1)
print(last_cell_type)
# Get the values in the first row as a list.
print(sheet.row_values(0))
# Get a range of columns from a specified row as a list.
# The first argument is the row index, the second and third are the start
# and end column indices, with the end index excluded.
print(sheet.row_slice(3, 0, 5))
```

> **Dica**: Para executar o exemplo, use uma planilha própria com estrutura semelhante a `Alibaba 2020 Equity Data.xls`; o link de armazenamento da fonte original foi removido por ser pouco acessível fora da China.

Acredito que pelo código acima todos já entenderam como ler um arquivo Excel. Se você quiser saber mais sobre o módulo `xlrd`, pode ler a [documentação oficial](https://xlrd.readthedocs.io/en/latest/).

### Escrevendo arquivos Excel

Para escrever um arquivo Excel, podemos criar um objeto de pasta de trabalho por meio da classe `Workbook` do módulo `xlwt`, adicionar uma planilha por meio do método `add_sheet` do objeto de pasta de trabalho, gravar dados na célula especificada por meio do método `write` do objeto de planilha e, finalmente, gravar a pasta de trabalho no arquivo ou memória especificado por meio de o método `save` do objeto pasta de trabalho. O código abaixo completa a gravação das notas dos exames de 5 alunos em 3 cursos em um arquivo Excel.

```python
import random

import xlwt

student_names = ['Guan Yu', 'Zhang Fei', 'Zhao Yun', 'Ma Chao', 'Huang Zhong']
scores = [[random.randrange(50, 101) for _ in range(3)] for _ in range(5)]
# Create a Workbook object.
wb = xlwt.Workbook()
# Create a Worksheet object.
sheet = wb.add_sheet('Grade 1 Class 2')
# Add header cells.
titles = ('name', 'Chinese', 'Mathematics', 'English')
for index, title in enumerate(titles):
    sheet.write(0, index, title)
# Write student names and scores into the worksheet.
for row in range(len(scores)):
    sheet.write(row + 1, 0, student_names[row])
    for col in range(len(scores[row])):
        sheet.write(row + 1, col + 1, scores[row][col])
# Save the Excel workbook.
wb.save('Exam Score Sheet.xls')
```

### Ajustando estilos de células

Ao escrever um arquivo Excel, também podemos definir estilos para células, incluindo principalmente fonte, alinhamento, borda e plano de fundo. `xlwt` fornece classes correspondentes para todos eles. Para definir um estilo de célula, primeiro criamos um objeto `XFStyle` e, em seguida, configuramos sua fonte, alinhamento, borda e outras propriedades. Por exemplo, se quisermos alterar as células do cabeçalho para um fundo amarelo, podemos fazer assim.

```python
header_style = xlwt.XFStyle()
pattern = xlwt.Pattern()
pattern.pattern = xlwt.Pattern.SOLID_PATTERN
# 0 - black, 1 - white, 2 - red, 3 - green, 4 - blue, 5 - yellow, 6 - pink, 7 - cyan
pattern.pattern_fore_colour = 5
header_style.pattern = pattern
titles = ('name', 'Chinese', 'Mathematics', 'English')
for index, title in enumerate(titles):
    sheet.write(0, index, title, header_style)
```

Se quisermos atribuir uma fonte específica ao cabeçalho, podemos usar a classe `Font` e adicionar o código abaixo.

```python
font = xlwt.Font()
# Font name
font.name = 'Chinese Italic'
# Font size (20 is the base unit, 18 means 18px)
font.height = 20 * 18
# Use bold or not
font.bold = True
# Use italic or not
font.italic = False
# Font color
font.colour_index = 1
header_style.font = font
```

> **Observação**: O nome da fonte atribuído acima por meio de `font.name` deve ser uma fonte disponível em seu sistema local.

Se quisermos que o cabeçalho seja centralizado vertical e horizontalmente, podemos usar o código a seguir.

```python
align = xlwt.Alignment()
# Vertical alignment
align.vert = xlwt.Alignment.VERT_CENTER
# Horizontal alignment
align.horz = xlwt.Alignment.HORZ_CENTER
header_style.alignment = align
```

Se quisermos que o cabeçalho tenha uma borda tracejada amarela, podemos configurá-lo assim.

```python
borders = xlwt.Borders()
props = (
    ('top', 'top_colour'), ('right', 'right_colour'),
    ('bottom', 'bottom_colour'), ('left', 'left_colour')
)
# Use a loop to configure style and color for borders on all four sides.
for position, color in props:
    # Use setattr to assign attribute values dynamically.
    setattr(borders, position, xlwt.Borders.DASHED)
    setattr(borders, color, 5)
header_style.borders = borders
```

Se quisermos ajustar a largura da coluna e a altura da linha do cabeçalho, podemos fazer isso conforme mostrado abaixo.

```python
# Set the row height to 40px.
sheet.row(0).set_style(xlwt.easyxf(f'font:height {20 * 40}'))
titles = ('name', 'Chinese', 'Mathematics', 'English')
for index, title in enumerate(titles):
    # Set the column width to 200px.
    sheet.col(index).width = 20 * 200
    # Set the cell value and style.
    sheet.write(0, index, title, header_style)
```

### Cálculo de Fórmula

Para o arquivo `Alibaba 2020 Equity Data.xls` que abrimos anteriormente, se quisermos calcular a média anual do preço de fechamento, o campo `Close`, e o total anual do volume de negociação, o campo `Volume`, podemos fazê-lo com fórmulas do Excel. Podemos primeiro usar `xlrd` para ler o arquivo Excel e, em seguida, usar a função `copy` fornecida por `xlutils` para converter a pasta de trabalho em um objeto `Workbook` gravável e, ao chamar `write`, podemos escrever um `xlwt.Formula` em uma célula.

O código para cálculo da fórmula é mostrado abaixo.

```python
import xlrd
import xlwt
from xlutils.copy import copy

wb_for_read = xlrd.open_workbook('Alibaba 2020 Equity Data.xls')
sheet1 = wb_for_read.sheet_by_index(0)
nrows, ncols = sheet1.nrows, sheet1.ncols
wb_for_write = copy(wb_for_read)
sheet2 = wb_for_write.get_sheet(0)
sheet2.write(nrows, 4, xlwt.Formula(f'average(E2:E{nrows})'))
sheet2.write(nrows, 6, xlwt.Formula(f'sum(G2:G{nrows})'))
wb_for_write.save('Alibaba 2020 Stock Data Summary.xls')
```
